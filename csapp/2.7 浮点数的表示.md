# 听课笔记：浮点数的表示

## 1. 二进制小数

- 类比十进制小数，二进制小数以小数点为界：
  - 左边：权重为 $2^i$（$i \ge 0$）
  - 右边：权重为 $2^{-j}$（$j \ge 1$）
- 示例：二进制 $101.11_2 = 1\cdot2^2 + 0\cdot2^1 + 1\cdot2^0 + 1\cdot2^{-1} + 1\cdot2^{-2} = 4 + 0 + 1 + 0.5 + 0.25 = 5.75$
- **局限性**：定点表示无法有效表示非常大或非常小的数。

---

## 2. IEEE浮点数标准

- 用公式 $V = (-1)^s \times M \times 2^E$ 表示，其中：
  - $s$：符号位（0 正，1 负）
  - $M$：尾数（有效数字）
  - $E$：阶码（指数）

### 2.1 单精度（float，32位）

| 位域          | 位数 | 说明             |
| ------------- | ---- | ---------------- |
| 符号位 s      | 1    | 最高位（第31位） |
| 阶码字段 exp  | 8    | 第23~30位        |
| 小数字段 frac | 23   | 第0~22位         |

### 2.2 双精度（double，64位）

| 位域          | 位数 | 说明             |
| ------------- | ---- | ---------------- |
| 符号位 s      | 1    | 最高位（第63位） |
| 阶码字段 exp  | 11   | 第52~62位        |
| 小数字段 frac | 52   | 第0~51位         |

---

## 3. 浮点数的分类

由阶码字段 `exp` 决定：

| exp 值       | 类型           | 用途/含义             |
| ------------ | -------------- | --------------------- |
| 不全0且不全1 | **规格化值**   | 常规数值              |
| 全0          | **非规格化值** | 表示 0 和接近 0 的数  |
| 全1          | **特殊值**     | 无穷大 或 NaN（非数） |

### 3.1 规格化值

- **阶码**：`exp` 不为全0也不为全1，其值为 $e = \text{exp}$（无符号整数）。
  - **实际指数** $E = e - \text{bias}$
  - **偏置 bias**：
    - 单精度（8位阶码）：$bias = 127 = 2^{8-1}-1$
    - 双精度（11位阶码）：$bias = 1023 = 2^{11-1}-1$
  - 因此单精度 $E$ 范围：$-126 \sim 127$
- **尾数**：小数字段 `frac` 表示 $f$，实际尾数 $M = 1 + f$（隐含开头的 1，因为 $1 \le M < 2$）。
- **数值**：$V = (-1)^s \times (1 + f) \times 2^{e - bias}$

### 3.2 非规格化值

- **阶码**：`exp` 全0，此时 $E = 1 - \text{bias}$（不是 $0 - bias$，这是为了平滑过渡）
- **尾数**：$M = f$（没有隐含的 1），$f$ 为小数字段的值。
- **用途**：
  1. **表示 0**：当 `frac` 全0，且 `s=0` 得 +0，`s=1` 得 -0。
  2. **表示接近 0 的数**：逐渐下溢，提供从 0 到最小规格化数之间的数值。
- **数值**：$V = (-1)^s \times f \times 2^{1 - bias}$

### 3.3 特殊值

- **无穷大**：`exp` 全1，`frac` 全0。
  - $s=0$：$+\infty$
  - $s=1$：$-\infty$
- **NaN（Not a Number）**：`exp` 全1，`frac` 非全0。
  - 用于表示无法确定的运算结果（如 $\sqrt{-1}$、$\infty - \infty$ 等）。

## 4. 示例：8位浮点数（简化理解）

假设格式：符号位 s（1位）、阶码 exp（4位）、尾数 frac（3位）。

- **bias** = $2^{4-1}-1 = 7$
- 非规格化数：$E = 1 - 7 = -6$，$M = f$（3位小数）
- 规格化数：$E = e - 7$，$M = 1 + f$

| 类别       | exp (二进制) | frac  | e 值 | E   | f   | M    | 公式                 | 十进制值 |
| ---------- | ------------ | ----- | ---- | --- | --- | ---- | -------------------- | -------- |
| 零         | 0000         | 000   | 0    | -6  | 0   | 0    | $(-1)^s \times 0$    | ±0       |
| 非规格化   | 0000         | 001   | 0    | -6  | 1/8 | 1/8  | $1/8 \times 2^{-6}$  | 0.001953 |
| 非规格化   | 0000         | 111   | 0    | -6  | 7/8 | 7/8  | $7/8 \times 2^{-6}$  | 0.013672 |
| 最小规格化 | 0001         | 000   | 1    | -6  | 0   | 1    | $1 \times 2^{-6}$    | 0.015625 |
| 规格化     | 0110         | 101   | 6    | -1  | 5/8 | 13/8 | $13/8 \times 2^{-1}$ | 0.8125   |
| 最大规格化 | 1110         | 111   | 14   | 7   | 7/8 | 15/8 | $15/8 \times 2^{7}$  | 240      |

## 5. 关键点总结

- **二进制小数**：权重为 2 的负幂。
- **IEEE 754**：用符号、阶码、尾数编码浮点数。
- **三种类型**：
  - **规格化**：阶码非全0/1，隐含 1 开头。
  - **非规格化**：阶码全0，表示 0 和渐近下溢的数。
  - **特殊值**：阶码全1，表示无穷大或 NaN。
- **偏置**：使指数可正可负，且让非规格化到规格化平滑过渡。
- 理解 8 位示例有助于掌握概念。
