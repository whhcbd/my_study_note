## 寄存器

### 通用寄存器（64位）

| 64位 | 32位 | 16位 | 8位 | 常见用途 |
|------|------|------|-----|----------|
| `%rax` | `%eax` | `%ax` | `%al` | 返回值 |
| `%rbx` | `%ebx` | `%bx` | `%bl` | 被调用者保存 |
| `%rcx` | `%ecx` | `%cx` | `%cl` | 第4个参数 |
| `%rdx` | `%edx` | `%dx` | `%dl` | 第3个参数 |
| `%rsi` | `%esi` | `%si` | `%sil` | 第2个参数 |
| `%rdi` | `%edi` | `%di` | `%dil` | 第1个参数 |
| `%rbp` | `%ebp` | `%bp` | `%bpl` | 被调用者保存 |
| `%rsp` | `%esp` | `%sp` | `%spl` | 栈指针 |
| `%r8` | `%r8d` | `%r8w` | `%r8b` | 第5个参数 |
| `%r9` | `%r9d` | `%r9w` | `%r9b` | 第6个参数 |
| `%r10`-`%r15` | `%r10d`-`%r15d` | `%r10w`-`%r15w` | `%r10b`-`%r15b` | 临时寄存器 |

### 条件码寄存器

| 标志 | 名称 | 含义 |
|------|------|------|
| **CF** | Carry Flag | 最高位产生进位（无符号溢出） |
| **ZF** | Zero Flag | 结果为零 |
| **SF** | Sign Flag | 结果为负（最高位为1） |
| **OF** | Overflow Flag | 有符号数溢出 |

## 数据传送指令

### MOV 指令家族

| 指令 | 操作数大小 | 说明 | 示例 |
|------|-----------|------|------|
| `movb` | 1字节 | 字节传送 | `movb $0xFF, %al` |
| `movw` | 2字节 | 字传送 | `movw %ax, (%rdi)` |
| `movl` | 4字节 | 双字传送 | `movl $-1, %eax` |
| `movq` | 8字节 | 四字传送 | `movq %rax, %rbx` |
| `movabsq` | 8字节 | 64位立即数传送 | `movabsq $0x123456789ABCDEF, %rax` |

**重要规则**：
- 源和目的操作数**不能同时为内存地址**
- `movl` 操作会将目标寄存器的**高32位清零**
- `movq` 的立即数只能是32位（符号扩展），需要64位立即数时用 `movabsq`

### 零扩展传送（MOVZ）

| 指令 | 源→目的 | 说明 |
|------|---------|------|
| `movzbw` | 字节→字 | 零扩展到16位 |
| `movzbl` | 字节→双字 | 零扩展到32位 |
| `movzwl` | 字→双字 | 零扩展到32位 |
| `movzbq` | 字节→四字 | 零扩展到64位 |
| `movzwq` | 字→四字 | 零扩展到64位 |

### 符号扩展传送（MOVS）

| 指令 | 源→目的 | 说明 |
|------|---------|------|
| `movsbw` | 字节→字 | 符号扩展到16位 |
| `movsbl` | 字节→双字 | 符号扩展到32位 |
| `movswl` | 字→双字 | 符号扩展到32位 |
| `movsbq` | 字节→四字 | 符号扩展到64位 |
| `movswq` | 字→四字 | 符号扩展到64位 |
| `movslq` | 双字→四字 | 符号扩展到64位 |
| `cltq` | `%eax`→`%rax` | 等价于 `movslq %eax, %rax` |

## 算术运算指令

### 加载有效地址

| 指令 | 格式 | 说明 |
|------|------|------|
| `leaq` | `leaq Src, Dst` | 计算地址表达式，结果存入目标寄存器（不访问内存） |

**示例**：
```asm
leaq 7(%rdx, %rdx, 4), %rax    ; %rax = 7 + %rdx + 4*%rdx = 5*%rdx + 7
leaq (%rdi, %rsi, 4), %rax     ; %rax = %rdi + 4*%rsi
```

### 一元运算指令

| 指令 | 效果 | 说明 |
|------|------|------|
| `inc D` | D = D + 1 | 加1（设置OF和ZF，不改变CF） |
| `dec D` | D = D - 1 | 减1（设置OF和ZF，不改变CF） |
| `neg D` | D = -D | 取负 |
| `not D` | D = ~D | 按位取反 |

### 二元运算指令

| 指令 | 效果 | 说明 |
|------|------|------|
| `add S, D` | D = D + S | 加法 |
| `sub S, D` | D = D - S | 减法 |
| `imul S, D` | D = D * S | 有符号乘法（截断） |
| `imul S` | `%rdx:%rax` = `%rax` * S | 有符号乘法（全宽度） |
| `mul S` | `%rdx:%rax` = `%rax` * S | 无符号乘法（全宽度） |
| `idiv S` | `%rax` = `%rdx:%rax` / S<br>`%rdx` = 余数 | 有符号除法 |
| `div S` | `%rax` = `%rdx:%rax` / S<br>`%rdx` = 余数 | 无符号除法 |

**注意**：
- 第二个操作数既是源也是目的
- 第二个操作数**不能是立即数**

## 逻辑运算指令

| 指令 | 效果 | 说明 |
|------|------|------|
| `xor S, D` | D = D ^ S | 异或（常用 `xor %rax, %rax` 清零） |
| `or S, D` | D = D \| S | 或 |
| `and S, D` | D = D & S | 与 |

**条件码设置**：
- CF 和 OF 置为 0
- SF 和 ZF 根据结果设置

## 移位指令

| 指令 | 说明 | 填充方式 |
|------|------|----------|
| `sal k, D` | 左移 | 右边填0 |
| `shl k, D` | 左移（同sal） | 右边填0 |
| `sar k, D` | 算术右移 | 左边填符号位 |
| `shr k, D` | 逻辑右移 | 左边填0 |

**移位量 k 可以是**：
- 立即数：`sal $3, %rax`
- 寄存器 `%cl`：`sal %cl, %rax`（只能用 `%cl`）

**有效位数**：

| 操作数大小 | 有效位数 |
|-----------|---------|
| 字节（b） | `%cl` 的低3位 |
| 字（w） | `%cl` 的低4位 |
| 双字（l） | `%cl` 的低5位 |
| 四字（q） | `%cl` 的低6位 |

## 比较与测试指令

| 指令 | 效果 | 说明 |
|------|------|------|
| `cmp S2, S1` | 计算 S1 - S2 | 设置条件码，不保存结果 |
| `test S2, S1` | 计算 S1 & S2 | 设置条件码，不保存结果 |

**常见用法**：
```asm
cmp %rsi, %rdi      ; 比较 %rdi 和 %rsi
test %rax, %rax     ; 检查 %rax 是否为0
```

## 条件设置指令

根据条件码的组合，将目标字节设置为 0 或 1。

### 有符号数比较

| 指令 | 条件 | 说明 |
|------|------|------|
| `sete D` | ZF | 相等 / Equal |
| `setne D` | ~ZF | 不相等 / Not Equal |
| `setl D` | SF ^ OF | 小于 / Less |
| `setle D` | (SF ^ OF) \| ZF | 小于等于 / Less or Equal |
| `setg D` | ~(SF ^ OF) & ~ZF | 大于 / Greater |
| `setge D` | ~(SF ^ OF) | 大于等于 / Greater or Equal |

### 无符号数比较

| 指令 | 条件 | 说明 |
|------|------|------|
| `sete D` | ZF | 相等 |
| `setne D` | ~ZF | 不相等 |
| `setb D` | CF | 低于 / Below |
| `setbe D` | CF \| ZF | 低于等于 / Below or Equal |
| `seta D` | ~CF & ~ZF | 高于 / Above |
| `setae D` | ~CF | 高于等于 / Above or Equal |

**示例**：
```asm
cmp %rsi, %rdi      ; 比较 a 和 b
sete %al            ; 若 a == b，则 %al = 1
movzbl %al, %eax    ; 零扩展到32位
```

## 跳转指令

### 无条件跳转

| 指令 | 说明 |
|------|------|
| `jmp Label` | 直接跳转到标号 |
| `jmp *Operand` | 间接跳转（操作数是地址） |

### 条件跳转（有符号）

| 指令 | 条件 | 说明 |
|------|------|------|
| `je Label` | ZF | 相等时跳转 |
| `jne Label` | ~ZF | 不相等时跳转 |
| `jl Label` | SF ^ OF | 小于时跳转 |
| `jle Label` | (SF ^ OF) \| ZF | 小于等于时跳转 |
| `jg Label` | ~(SF ^ OF) & ~ZF | 大于时跳转 |
| `jge Label` | ~(SF ^ OF) | 大于等于时跳转 |

### 条件跳转（无符号）

| 指令 | 条件 | 说明 |
|------|------|------|
| `je Label` | ZF | 相等时跳转 |
| `jne Label` | ~ZF | 不相等时跳转 |
| `jb Label` | CF | 低于时跳转 |
| `jbe Label` | CF \| ZF | 低于等于时跳转 |
| `ja Label` | ~CF & ~ZF | 高于时跳转 |
| `jae Label` | ~CF | 高于等于时跳转 |

**典型用法**：
```asm
cmp %rsi, %rdi      ; 比较
jl .L4              ; 若 %rdi < %rsi，跳转到 .L4
```

## 条件传送指令

根据条件码选择性地传送数据，**避免分支预测失败的代价**。

### 有符号数条件传送

| 指令 | 条件 | 说明 |
|------|------|------|
| `cmove S, D` | ZF | 相等时传送 |
| `cmovne S, D` | ~ZF | 不相等时传送 |
| `cmovl S, D` | SF ^ OF | 小于时传送 |
| `cmovle S, D` | (SF ^ OF) \| ZF | 小于等于时传送 |
| `cmovg S, D` | ~(SF ^ OF) & ~ZF | 大于时传送 |
| `cmovge S, D` | ~(SF ^ OF) | 大于等于时传送 |

### 无符号数条件传送

| 指令 | 条件 | 说明 |
|------|------|------|
| `cmove S, D` | ZF | 相等时传送 |
| `cmovne S, D` | ~ZF | 不相等时传送 |
| `cmovb S, D` | CF | 低于时传送 |
| `cmovbe S, D` | CF \| ZF | 低于等于时传送 |
| `cmova S, D` | ~CF & ~ZF | 高于时传送 |
| `cmovae S, D` | ~CF | 高于等于时传送 |

**优势**：
- 无需跳转，保持流水线满载
- 避免分支预测失败的性能损失

**示例**：
```asm
mov %rdi, %rax      ; 假设计算 x-y
sub %rsi, %rax
mov %rsi, %rdx      ; 假设计算 y-x
sub %rdi, %rdx
cmp %rsi, %rdi      ; 比较 x 和 y
cmovge %rdx, %rax   ; 若 x >= y，选择 y-x
```

## 栈操作指令

栈从**高地址向低地址**增长，`%rsp` 指向栈顶。

| 指令 | 效果 | 等价操作 |
|------|------|----------|
| `pushq S` | 压栈 | `subq $8, %rsp`<br>`movq S, (%rsp)` |
| `popq D` | 弹栈 | `movq (%rsp), D`<br>`addq $8, %rsp` |

**示例**：
```asm
pushq %rax          ; 将 %rax 压入栈
popq %rbx           ; 从栈顶弹出到 %rbx
```

## 过程调用指令

| 指令 | 效果 | 说明 |
|------|------|------|
| `call Label` | 调用函数 | 1. 将返回地址压栈<br>2. 跳转到 Label |
| `call *Operand` | 间接调用 | 操作数是函数地址 |
| `ret` | 返回 | 1. 从栈弹出返回地址<br>2. 跳转到该地址 |

**等价操作**：
```asm
; call Label 等价于：
pushq %rip + 指令长度    ; 压入返回地址
jmp Label

; ret 等价于：
popq %rip                ; 弹出返回地址并跳转
```

## 内存寻址模式

### 通用格式

```
Imm(rb, ri, s)
```

**有效地址** = `Imm + Reg[rb] + Reg[ri] * s`

| 组成 | 说明 | 限制 |
|------|------|------|
| `Imm` | 立即数偏移 | 可选 |
| `rb` | 基址寄存器 | 可选 |
| `ri` | 变址寄存器 | 可选 |
| `s` | 比例因子 | 必须是 1, 2, 4, 8 |

### 常见形式

| 格式 | 有效地址 | 示例 |
|------|----------|------|
| `Imm` | Imm | `0x8048` |
| `(rb)` | Reg[rb] | `(%rax)` |
| `Imm(rb)` | Imm + Reg[rb] | `8(%rsp)` |
| `(rb, ri)` | Reg[rb] + Reg[ri] | `(%rax, %rdx)` |
| `Imm(rb, ri)` | Imm + Reg[rb] + Reg[ri] | `8(%rax, %rdx)` |
| `(, ri, s)` | Reg[ri] * s | `(, %rdx, 4)` |
| `Imm(, ri, s)` | Imm + Reg[ri] * s | `8(, %rdx, 4)` |
| `(rb, ri, s)` | Reg[rb] + Reg[ri] * s | `(%rax, %rdx, 4)` |
| `Imm(rb, ri, s)` | Imm + Reg[rb] + Reg[ri] * s | `8(%rax, %rdx, 4)` |

## 快速记忆技巧

### 指令后缀

| 后缀 | 大小 | 字节数 |
|------|------|--------|
| `b` | byte | 1 |
| `w` | word | 2 |
| `l` | long (double word) | 4 |
| `q` | quad word | 8 |

### 条件码助记

- **有符号**：L (Less), G (Greater), E (Equal)
- **无符号**：B (Below), A (Above), E (Equal)
- **取反**：加 N 前缀，如 NE (Not Equal)

### 常见模式

1. **清零寄存器**：`xor %rax, %rax`（比 `mov $0, %rax` 更高效）
2. **测试零**：`test %rax, %rax`
3. **比较相等**：`cmp %rax, %rbx` + `je`
4. **函数返回值**：存放在 `%rax`
5. **栈帧建立**：`pushq %rbp` + `movq %rsp, %rbp`

---

## 性能优化要点

1. **LEAQ 用于算术**：比 `imul` 更快，可实现 `x + k*y` 形式的计算
2. **移位代替乘除**：`sal $3, %rax` 比 `imul $8, %rax` 快
3. **条件传送优于跳转**：在简单条件下，`cmov` 避免分支预测失败
4. **跳转表优化 switch**：O(1) 时间复杂度，适合 case 较多的情况
