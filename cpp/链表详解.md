## 链表基础概念

### 1. 什么是链表

链表是一种线性数据结构，其中的元素以节点形式存储，每个节点包含：

- **数据域**：存储节点数据
- **指针域**：指向下一个节点（或上一个节点）

```
链表基本结构：

[数据|指针] -> [数据|指针] -> [数据|指针] -> [数据|NULL]
 节点1       节点2       节点3       节点4
```

### 2. 链表类型对比

| 链表类型 | 指针数量 | 是否可逆 | 尾节点指向 | 应用场景 |
|----------|----------|----------|------------|----------|
| 单向链表 | 1个（next） | 否 | NULL | 基础应用，栈实现 |
| 双向链表 | 2个（next, prev） | 是 | NULL | 需要双向遍历的场景 |
| 循环单向链表 | 1个（next） | 否 | 头节点 | 轮询，约瑟夫问题 |
| 循环双向链表 | 2个（next, prev） | 是 | 头节点 | 缓存淘汰算法 |

### 3. 节点结构定义

```cpp
// 单向链表节点
struct ListNode {
    int val;
    ListNode* next;
    ListNode(int x) : val(x), next(nullptr) {}
};

// 双向链表节点
struct DoublyListNode {
    int val;
    DoublyListNode* prev;
    DoublyListNode* next;
    DoublyListNode(int x) : val(x), prev(nullptr), next(nullptr) {}
};
```

## 单向链表

### 1. 结构示意图

```
头指针
  ↓
┌─────┬───┐   ┌─────┬───┐   ┌─────┬───┐   ┌─────┬────┐
│ 10  │ ● ├─>│ 20  │ ● ├─>│ 30  │ ● ├─>│ 40  │NULL│
└─────┴───┘   └─────┴───┘   └─────┴───┘   └─────┴────┘
 节点1         节点2         节点3         节点4
```

### 2. 特点

- **单向遍历**：只能从头到尾访问
- **操作简单**：插入删除只需修改指针
- **空间效率**：每个节点只存储一个指针

### 3. 基本操作

#### 3.1 遍历链表

```cpp
void traverse(ListNode* head) {
    ListNode* current = head;
    while (current != nullptr) {
        std::cout << current->val << " -> ";
        current = current->next;
    }
    std::cout << "NULL" << std::endl;
}
```

#### 3.2 查找节点

```cpp
ListNode* find(ListNode* head, int target) {
    ListNode* current = head;
    while (current != nullptr) {
        if (current->val == target) {
            return current;
        }
        current = current->next;
    }
    return nullptr;
}
```

## 双向链表

### 1. 结构示意图

```
      NULL                                            NULL
        ↑                                              ↑
┌──────┬────────┬───┐   ┌──────┬────────┬───┐   ┌──────┬────────┬───┐
│      │   10   │ ● ├─>│      │   20   │ ● ├─>│      │   30   │ ● │
│ NULL ├────────┤   │<─┼──●───┼────────┤   │<─┼──●───┼────────┤   │
└──────┴────────┴───┘   └──────┴────────┴───┘   └──────┴────────┴───┘
      节点1                 节点2                 节点3
```

### 2. 特点

- **双向遍历**：可以向前或向后遍历
- **操作灵活**：便于在任意位置插入删除
- **空间开销**：每个节点多一个prev指针

### 3. 基本操作

#### 3.1 前向遍历

```cpp
void forwardTraverse(DoublyListNode* head) {
    DoublyListNode* current = head;
    while (current != nullptr) {
        std::cout << current->val << " -> ";
        current = current->next;
    }
    std::cout << "NULL" << std::endl;
}
```

#### 3.2 反向遍历

```cpp
void backwardTraverse(DoublyListNode* tail) {
    DoublyListNode* current = tail;
    while (current != nullptr) {
        std::cout << current->val << " -> ";
        current = current->prev;
    }
    std::cout << "NULL" << std::endl;
}
```

## 循环链表

### 1. 结构示意图

```
单向循环链表：
        ┌─────────────────────────────┐
        ↓                             │
┌─────┬───┐   ┌─────┬───┐   ┌─────┬───┐
│ 10  │ ● ├─>│ 20  │ ● ├─>│ 30  │ ● │
└─────┴───┘   └─────┴───┘   └─────┴─┼─┘
                           ↑           │
                           └───────────┘
```

```
双向循环链表：
        ┌─────────────────────────────────────┐
        ↓                                     │
┌──────┬────────┬───┐   ┌──────┬────────┬───┐
│      │   10   │ ● ├─>│      │   20   │ ● │
│ ●───┼────────┤   │<─┼──●───┼────────┤   │
└──────┴────────┴───┘   └──────┴────────┴─┼─┘
      ↑                              │      │
      └──────────────────────────────┴──────┘
```

### 2. 特点

- **无尾NULL**：尾节点指向头节点
- **循环访问**：可以从任意节点开始遍历整个链表
- **适合轮询**：常用于需要循环访问的场景

### 3. 判断循环结束

```cpp
void circularTraverse(ListNode* head) {
    if (head == nullptr) return;
    
    ListNode* current = head;
    do {
        std::cout << current->val << " -> ";
        current = current->next;
    } while (current != head);
    std::cout << "(back to head)" << std::endl;
}
```

## 链表基本操作

### 1. 创建链表

#### 1.1 创建单个节点

```cpp
ListNode* createNode(int val) {
    return new ListNode(val);
}
```

#### 1.2 从数组创建链表

```cpp
ListNode* createFromArray(std::vector<int>& arr) {
    if (arr.empty()) return nullptr;
    
    ListNode* head = new ListNode(arr[0]);
    ListNode* current = head;
    
    for (size_t i = 1; i < arr.size(); i++) {
        current->next = new ListNode(arr[i]);
        current = current->next;
    }
    
    return head;
}
```

### 2. 插入操作

#### 2.1 头部插入

```
插入前：  [头] -> [节点1] -> [节点2] -> NULL

插入后：  [新节点] -> [头] -> [节点1] -> [节点2] -> NULL
             ↓
            新头指针
```

```cpp
ListNode* insertAtHead(ListNode* head, int val) {
    ListNode* newNode = new ListNode(val);
    newNode->next = head;
    return newNode;
}
```

#### 2.2 尾部插入

```
插入前：  [节点1] -> [节点2] -> NULL

插入后：  [节点1] -> [节点2] -> [新节点] -> NULL
```

```cpp
void insertAtTail(ListNode*& head, int val) {
    ListNode* newNode = new ListNode(val);
    
    if (head == nullptr) {
        head = newNode;
        return;
    }
    
    ListNode* current = head;
    while (current->next != nullptr) {
        current = current->next;
    }
    current->next = newNode;
}
```

#### 2.3 中间插入（在指定节点后）

```
插入前：  [节点1] -> [节点2] -> [节点3]
                    ↑
                  current

插入后：  [节点1] -> [节点2] -> [新节点] -> [节点3]
                    ↑
                  current
```

```cpp
void insertAfter(ListNode* current, int val) {
    if (current == nullptr) return;
    
    ListNode* newNode = new ListNode(val);
    newNode->next = current->next;
    current->next = newNode;
}
```

#### 2.4 按位置插入

```cpp
ListNode* insertAtPosition(ListNode* head, int val, int position) {
    if (position <= 0) {
        return insertAtHead(head, val);
    }
    
    ListNode* newNode = new ListNode(val);
    ListNode* current = head;
    
    for (int i = 0; current != nullptr && i < position - 1; i++) {
        current = current->next;
    }
    
    if (current == nullptr) {
        delete newNode;
        return head;
    }
    
    newNode->next = current->next;
    current->next = newNode;
    
    return head;
}
```

### 3. 删除操作

#### 3.1 删除头节点

```
删除前：  [头] -> [节点1] -> [节点2] -> NULL

删除后：        [节点1] -> [节点2] -> NULL
                      ↓
                   新头指针
```

```cpp
ListNode* deleteAtHead(ListNode* head) {
    if (head == nullptr) return nullptr;
    
    ListNode* temp = head;
    head = head->next;
    delete temp;
    
    return head;
}
```

#### 3.2 删除尾节点

```
删除前：  [节点1] -> [节点2] -> [节点3] -> NULL

删除后：  [节点1] -> [节点2] -> NULL
```

```cpp
void deleteAtTail(ListNode*& head) {
    if (head == nullptr) return;
    
    if (head->next == nullptr) {
        delete head;
        head = nullptr;
        return;
    }
    
    ListNode* current = head;
    while (current->next->next != nullptr) {
        current = current->next;
    }
    
    delete current->next;
    current->next = nullptr;
}
```

#### 3.3 删除指定节点（已知前驱节点）

```cpp
void deleteAfter(ListNode* current) {
    if (current == nullptr || current->next == nullptr) return;
    
    ListNode* toDelete = current->next;
    current->next = toDelete->next;
    delete toDelete;
}
```

#### 3.4 删除指定值的节点

```cpp
ListNode* deleteByValue(ListNode* head, int val) {
    if (head == nullptr) return nullptr;
    
    if (head->val == val) {
        ListNode* temp = head;
        head = head->next;
        delete temp;
        return head;
    }
    
    ListNode* current = head;
    while (current->next != nullptr && current->next->val != val) {
        current = current->next;
    }
    
    if (current->next != nullptr) {
        ListNode* toDelete = current->next;
        current->next = toDelete->next;
        delete toDelete;
    }
    
    return head;
}
```

### 4. 查找操作

#### 4.1 按值查找

```cpp
ListNode* find(ListNode* head, int val) {
    ListNode* current = head;
    while (current != nullptr) {
        if (current->val == val) {
            return current;
        }
        current = current->next;

```

#### 4.2 按位置查找

```cpp
ListNode* getAt(ListNode* head, int index) {
    int pos = 0;
    ListNode* current = head;
    
    while (current != nullptr && pos < index) {
        current = current->next;
        pos++;
    }
    
    return current;
}
```

### 5. 更新操作

```cpp
bool update(ListNode* head, int oldValue, int newValue) {
    ListNode* node = find(head, oldValue);
    if (node != nullptr) {
        node->val = newValue;
        return true;
    }
    return false;
}
```

## 链表高级应用

### 1. 反转链表

#### 1.1 迭代法

```
反转过程：

原链表：  1 -> 2 -> 3 -> NULL

步骤1：
prev: NULL, current: 1, next: 2
1 -> NULL

步骤2：
prev: 1, current: 2, next: 3
2 -> 1 -> NULL

步骤3：
prev: 2, current: 3, next: NULL
3 -> 2 -> 1 -> NULL
```

```cpp
ListNode* reverseList(ListNode* head) {
    ListNode* prev = nullptr;
    ListNode* current = head;
    
    while (current != nullptr) {
        ListNode* next = current->next;
        current->next = prev;
        prev = current;
        current = next;
    }
    
    return prev;
}
```

#### 1.2 递归法

```cpp
ListNode* reverseListRecursive(ListNode* head) {
    if (head == nullptr || head->next == nullptr) {
        return head;
    }
    
    ListNode* newHead = reverseListRecursive(head->next);
    head->next->next = head;
    head->next = nullptr;
    
    return newHead;
}
```

### 2. 检测环

#### 2.1 快慢指针法

```
有环链表：
      ┌─────────────────────┐
      ↓                     │
1 -> 2 -> 3 -> 4 -> 5 -> 6
           ↑              │
           └──────────────┘

快指针：每次走2步
慢指针：每次走1步

如果有环，快慢指针最终会相遇
```

```cpp
bool hasCycle(ListNode* head) {
    if (head == nullptr || head->next == nullptr) {
        return false;
    }
    
    ListNode* slow = head;
    ListNode* fast = head->next;
    
    while (slow != fast) {
        if (fast == nullptr || fast->next == nullptr) {
            return false;
        }
        slow = slow->next;
        fast = fast->next->next;
    }
    
    return true;
}
```

#### 2.2 找到环的起点

```cpp
ListNode* detectCycle(ListNode* head) {
    if (head == nullptr || head->next == nullptr) {
        return nullptr;
    }
    
    ListNode* slow = head;
    ListNode* fast = head;
    
    while (fast != nullptr && fast->next != nullptr) {
        slow = slow->next;
        fast = fast->next->next;
        
        if (slow == fast) {
            slow = head;
            while (slow != fast) {
                slow = slow->next;
                fast = fast->next;
            }
            return slow;
        }
    }
    
    return nullptr;
}
```

### 3. 寻找中间节点

#### 3.1 快慢指针法

```
链表：1 -> 2 -> 3 -> 4 -> 5 -> NULL

慢指针：1 -> 2 -> 3
快指针：1 -> 3 -> 5 -> NULL

结果：返回节点3
```

```cpp
ListNode* findMiddle(ListNode* head) {
    ListNode* slow = head;
    ListNode* fast = head;
    
    while (fast != nullptr && fast->next != nullptr) {
        slow = slow->next;
        fast = fast->next->next;
    }
    
    return slow;
}
```

### 4. 合并有序链表

#### 4.1 合并两个有序链表

```cpp
ListNode* mergeTwoLists(ListNode* l1, ListNode* l2) {
    ListNode dummy(0);
    ListNode* current = &dummy;
    
    while (l1 != nullptr && l2 != nullptr) {
        if (l1->val <= l2->val) {
            current->next = l1;
            l1 = l1->next;
        } else {
            current->next = l2;
            l2 = l2->next;
        }
        current = current->next;
    }
    
    current->next = (l1 != nullptr) ? l1 : l2;
    
    return dummy.next;
}
```

#### 4.2 合并K个有序链表（使用优先队列）

```cpp
#include <queue>

struct Compare {
    bool operator()(ListNode* a, ListNode* b) {
        return a->val > b->val;
    }
};

ListNode* mergeKLists(std::vector<ListNode*>& lists) {
    std::priority_queue<ListNode*, std::vector<ListNode*>, Compare> minHeap;
    
    for (ListNode* node : lists) {
        if (node != nullptr) {
            minHeap.push(node);
        }
    }
    
    ListNode dummy(0);
    ListNode* current = &dummy;
    
    while (!minHeap.empty()) {
        ListNode* node = minHeap.top();
        minHeap.pop();
        
        current->next = node;
        current = current->next;
        
        if (node->next != nullptr) {
            minHeap.push(node->next);
        }
    }
    
    return dummy.next;
}
```

### 5. 删除链表的倒数第N个节点

```cpp
ListNode* removeNthFromEnd(ListNode* head, int n) {
    ListNode dummy(0);
    dummy.next = head;
    
    ListNode* fast = &dummy;
    ListNode* slow = &dummy;
    
    for (int i = 0; i <= n; i++) {
        if (fast == nullptr) return head;
        fast = fast->next;
    }
    
    while (fast != nullptr) {
        fast = fast->next;
        slow = slow->next;
    }
    
    ListNode* toDelete = slow->next;
    slow->next = toDelete->next;
    delete toDelete;
    
    return dummy.next;
}
```

### 6. 旋转链表

```cpp
ListNode* rotateRight(ListNode* head, int k) {
    if (head == nullptr || head->next == nullptr) return head;
    
    int length = 1;
    ListNode* tail = head;
    while (tail->next != nullptr) {
        tail = tail->next;
        length++;
    }
    
    k = k % length;
    if (k == 0) return head;
    
    int stepsToNewHead = length - k;
    ListNode* newTail = head;
    for (int i = 1; i < stepsToNewHead; i++) {
        newTail = newTail->next;
    }
    
    ListNode* newHead = newTail->next;
    newTail->next = nullptr;
    tail->next = head;
    
    return newHead;
}
```

### 7. 排序链表（归并排序）

```cpp
ListNode* sortList(ListNode* head) {
    if (head == nullptr || head->next == nullptr) {
        return head;
    }
    
    ListNode* mid = getMid(head);
    ListNode* right = sortList(mid->next);
    mid->next = nullptr;
    ListNode* left = sortList(head);
    
    return mergeTwoLists(left, right);
}

ListNode* getMid(ListNode* head) {
    ListNode* slow = head;
    ListNode* fast = head->next;
    
    while (fast != nullptr && fast->next != nullptr) {
        slow = slow->next;
        fast = fast->next->next;
    }
    
    return slow;
}
```

## 链表vs数组

### 1. 复杂度对比

| 操作 | 数组 | 链表 |
|------|------|------|
| 按索引访问 | O(1) | O(n) |
| 按值查找 | O(n) | O(n) |
| 头部插入 | O(n) | O(1) |
| 尾部插入 | O(1)* | O(n)** |
| 中间插入 | O(n) | O(n)* |
| 头部删除 | O(n) | O(1) |
| 尾部删除 | O(1) | O(n) |
| 中间删除 | O(n) | O(n)* |
| 内存空间 | 连续 | 分散 |

* 假设已知插入位置的前驱节点
** 如果有尾指针则为O(1)

### 2. 空间占用

#### 数组
```
┌───┬───┬───┬───┬───┬───┬───┐
│ 1 │ 2 │ 3 │ 4 │ 5 │ ? │ ? │
└───┴───┴───┴───┴───┴───┴───┘
  连续内存，无额外开销
```

#### 链表
```
┌────┬───┐   ┌────┬───┐   ┌────┬───┐
│ 1  │ ● ├─>│ 2  │ ● ├─>│ 3  │ ● │
└────┴───┘   └────┴───┘   └────┴───┘
    数据       指针
每个节点有额外的指针开销
```

### 3. 使用场景

#### 选择数组的情况
- 需要频繁按索引访问
- 数据量固定，预先知道大小
- 内存空间有限，需要节省空间
- 需要更好的缓存局部性

#### 选择链表的情况
- 频繁在头部插入删除
- 数据量动态变化很大
- 不需要按索引访问
- 内存分散分配

## 常见问题与技巧

### 1. 哨兵节点（Dummy Node）

```cpp
ListNode* removeElements(ListNode* head, int val) {
    ListNode dummy(0);
    dummy.next = head;
    
    ListNode* current = &dummy;
    while (current->next != nullptr) {
        if (current->next->val == val) {
            ListNode* toDelete = current->next;
            current->next = toDelete->next;
            delete toDelete;
        } else {
            current = current->next;
        }
    }
    
    return dummy.next;
}
```

**优势**：统一处理头部和非头部节点，减少边界判断

### 2. 快慢指针技巧

```cpp
// 应用1：找中间节点
ListNode* findMiddle(ListNode* head) {
    ListNode* slow = head;
    ListNode* fast = head;
    
    while (fast != nullptr && fast->next != nullptr) {
        slow = slow->next;
        fast = fast->next->next;
    }
    return slow;
}

// 应用2：检测环
bool hasCycle(ListNode* head) {
    ListNode* slow = head;
    ListNode* fast = head;
    
    while (fast != nullptr && fast->next != nullptr) {
        slow = slow->next;
        fast = fast->next->next;
        if (slow == fast) return true;
    }
    return false;
}
```

#### 3.1 删除整个链表

```cpp
void deleteList(ListNode* head) {

        head = head->next;
        delete temp;
```

#### 3.2 使用智能指针

```cpp
#include <memory>

struct SmartListNode {
    int val;
    std::unique_ptr<SmartListNode> next;
    
    SmartListNode(int x) : val(x), next(nullptr) {}
};

// 自动内存管理，无需手动delete
void createSmartList() {
    auto head = std::make_unique<SmartListNode>(1);
```

```cpp
    ListNode* fast = head;

        slow = slow->next;
```

```cpp
#include <unordered_set>

ListNode* deleteDuplicatesUnsorted(ListNode* head) {
    if (head == nullptr) return nullptr;
    
    std::unordered_set<int> seen;
    ListNode dummy(0);
    dummy.next = head;
    ListNode* prev = &dummy;
    ListNode* current = head;
    
    while (current != nullptr) {
        if (seen.find(current->val) != seen.end()) {
            prev->next = current->next;
            delete current;
            current = prev->next;
        } else {
            seen.insert(current->val);
            prev = current;
            current = current->next;
        }
    }
    
    return dummy.next;
}
```

### 1. 单向链表完整类实现

```cpp
#include <iostream>
#include <vector>
#include <stack>

class LinkedList {
private:
    struct Node {
        int val;
        Node* next;
        Node(int x) : val(x), next(nullptr) {}
    };
    
    Node* head;
    int size;
    
public:
    LinkedList() : head(nullptr), size(0) {}
    
    ~LinkedList() {
        clear();
    }
    
    void append(int val) {
        Node* newNode = new Node(val);
        if (head == nullptr) {
            head = newNode;
        } else {
            Node* current = head;
            while (current->next != nullptr) {
                current = current->next;
            }
            current->next = newNode;
        }
        size++;
    }
    
    void prepend(int val) {
        Node* newNode = new Node(val);
        newNode->next = head;
        head = newNode;
        size++;
    }
    
    void insert(int index, int val) {
        if (index < 0 || index > size) return;
        
        if (index == 0) {
            prepend(val);
            return;
        }
        
        Node* newNode = new Node(val);
        Node* current = head;
        for (int i = 0; i < index - 1; i++) {
            current = current->next;
        }
        
        newNode->next = current->next;
        current->next = newNode;
        size++;
    }
    
    void remove(int index) {
        if (index < 0 || index >= size) return;
        
        Node* toDelete;
        if (index == 0) {
            toDelete = head;
            head = head->next;
        } else {
            Node* current = head;
            for (int i = 0; i < index - 1; i++) {
                current = current->next;
            }
            toDelete = current->next;
            current->next = toDelete->next;
        }
        
        delete toDelete;
        size--;
    }
    
    int get(int index) {
        if (index < 0 || index >= size) throw std::out_of_range("Index out of range");
        
        Node* current = head;
        for (int i = 0; i < index; i++) {
            current = current->next;
        }
        return current->val;
    }
    
    void set(int index, int val) {
        if (index < 0 || index >= size) return;
        
        Node* current = head;
        for (int i = 0; i < index; i++) {
            current = current->next;
        }
        current->val = val;
    }
    
    int find(int val) {
        Node* current = head;
        int index = 0;
        
        while (current != nullptr) {
            if (current->val == val) {
                return index;
            }
            current = current->next;
            index++;
        }
        
        return -1;
    }
    
    void reverse() {
        Node* prev = nullptr;
        Node* current = head;
        
        while (current != nullptr) {
            Node* next = current->next;
            current->next = prev;
            prev = current;
            current = next;
        }
        
        head = prev;
    }
    
    void print() {
        Node* current = head;
        while (current != nullptr) {
            std::cout << current->val;
            if (current->next != nullptr) {
                std::cout << " -> ";
            }
            current = current->next;
        }
        std::cout << std::endl;
    }
    
    int getSize() {
        return size;
    }
    
    bool isEmpty() {
        return size == 0;
    }
    
    void clear() {
        while (head != nullptr) {
            Node* temp = head;
            head = head->next;
            delete temp;
        }
        size = 0;
    }
    
    std::vector<int> toVector() {
        std::vector<int> result;
        Node* current = head;
        while (current != nullptr) {
            result.push_back(current->val);
            current = current->next;
        }
        return result;
    }
};
```

### 2. 双向链表完整类实现

```cpp
#include <iostream>
#include <vector>

class DoublyLinkedList {
private:
    struct Node {
        int val;
        Node* prev;
        Node* next;
        Node(int x) : val(x), prev(nullptr), next(nullptr) {}
    };
    
    Node* head;
    Node* tail;
    int size;
    
public:
    DoublyLinkedList() : head(nullptr), tail(nullptr), size(0) {}
    
    ~DoublyLinkedList() {
        clear();
    }
    
    void append(int val) {
        Node* newNode = new Node(val);
        if (tail == nullptr) {
            head = tail = newNode;
        } else {
            tail->next = newNode;
            newNode->prev = tail;
            tail = newNode;
        }
        size++;
    }
    
    void prepend(int val) {
        Node* newNode = new Node(val);
        if (head == nullptr) {
            head = tail = newNode;
        } else {
            head->prev = newNode;
            newNode->next = head;
            head = newNode;
        }
        size++;
    }
    
    void remove(int val) {
        Node* current = head;
        while (current != nullptr) {
            if (current->val == val) {
                if (current == head) {
                    head = current->next;
                    if (head != nullptr) {
                        head->prev = nullptr;
                    } else {
                        tail = nullptr;
                    }
                } else if (current == tail) {
                    tail = current->prev;
                    tail->next = nullptr;
                } else {
                    current->prev->next = current->next;
                    current->next->prev = current->prev;
                }
                
                Node* toDelete = current;
                current = current->next;
                delete toDelete;
                size--;
            } else {
                current = current->next;
            }
        }
    }
    
    void printForward() {
        Node* current = head;
        while (current != nullptr) {
            std::cout << current->val;
            if (current->next != nullptr) {
                std::cout << " <-> ";
            }
            current = current->next;
        }
        std::cout << std::endl;
    }
    
    void printBackward() {
        Node* current = tail;
        while (current != nullptr) {
            std::cout << current->val;
            if (current->prev != nullptr) {
                std::cout << " <-> ";
            }
            current = current->prev;
        }
        std::cout << std::endl;
    }
    
    int getSize() {
        return size;
    }
    
    bool isEmpty() {
        return size == 0;
    }
    
    void clear() {
        while (head != nullptr) {
            Node* temp = head;
            head = head->next;
            delete temp;
        }
        tail = nullptr;
        size = 0;
    }
};
```
