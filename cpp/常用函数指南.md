## 字符串处理函数

### std::string 常用函数

#### 1. length() / size() - 获取字符串长度

**函数签名：**

```cpp
size_t length() const;
size_t size() const;
```

**参数说明：** 无参数

**返回值：** `size_t` 类型，表示字符串的字符个数（不包括结尾的 `\0`）

**示例：**

```cpp
#include <string>

string str = "Hello";
size_t len = str.length();  // 返回 5
size_t size = str.size();   // 返回 5
```

**注意事项：**

- `length()` 和 `size()` 完全等价，推荐使用 `size()`
- 返回类型是 `size_t`（无符号长整型），比较时注意类型
- 空字符串返回 0

---

#### 2. empty() - 判断是否为空

**函数签名：**

```cpp
bool empty() const;
```

**参数说明：** 无参数

**返回值：** `bool` 类型

- `true`：字符串为空
- `false`：字符串不为空

**示例：**

```cpp
string str1 = "";
string str2 = "Hello";

bool empty1 = str1.empty();  // true
bool empty2 = str2.empty();  // false

if (str.empty()) {
    cout << "字符串为空" << endl;
}
```

**注意事项：**

- 判断字符串是否为空的首选方法
- 比 `str.size() == 0` 更清晰

---

#### 3. clear() - 清空字符串

**函数签名：**

```cpp
void clear();
```

**参数说明：** 无参数

**返回值：** 无（void）

**示例：**

```cpp
string str = "Hello";
cout << "清空前: " << str << endl;  // Hello
str.clear();
cout << "清空后: " << str << endl;  // (空)
cout << "长度: " << str.length() << endl;  // 0
```

**注意事项：**

- 清空后字符串长度变为 0
- 不释放内存，只是清空内容

---

#### 4. append() - 追加字符串

**函数签名：**

```cpp
string& append(const string& str);
string& append(const char* s);
string& append(size_t n, char c);
```

**参数说明：**

- `str`：要追加的字符串
- `s`：要追加的 C 风格字符串
- `n`：要追加的字符个数
- `c`：要追加的字符

**返回值：** 返回字符串的引用（支持链式调用）

**示例：**

```cpp
string str = "Hello";
str.append(" World");   // "Hello World"
str.append(3, '!');     // "Hello World!!!"
cout << str << endl;
```

**注意事项：**

- 会修改原字符串
- 支持链式调用：`str.append("a").append("b")`

---

#### 5. push_back() - 追加单个字符

**函数签名：**

```cpp
void push_back(char c);
```

**参数说明：**

- `c`：要追加的字符

**返回值：** 无（void）

**示例：**

```cpp
string str = "Hello";
str.push_back(' ');    // "Hello "
str.push_back('W');     // "Hello W"
cout << str << endl;
```

**注意事项：**

- 只能追加单个字符
- 比 `append()` 更高效（用于单个字符）

---

#### 6. insert() - 插入字符或字符串

**函数签名：**

```cpp
string& insert(size_t pos, const string& str);
string& insert(size_t pos, const char* s);
string& insert(size_t pos, size_t n, char c);
```

**参数说明：**

- `pos`：插入位置（从0开始）
- `str`：要插入的字符串
- `s`：要插入的 C 风格字符串
- `n`：要插入的字符个数
- `c`：要插入的字符

**返回值：** 返回字符串的引用

**示例：**

```cpp
string str = "Hello World";
str.insert(5, ",");      // "Hello, World"  在位置5插入逗号
str.insert(12, "!!!");   // "Hello, World!!!"
cout << str << endl;
```

**注意事项：**

- `pos` 不能超过字符串长度
- 插入位置从0开始计数
- 如果 `pos` 越界会抛出异常

---

#### 7. erase() - 删除字符

**函数签名：**

```cpp
string& erase(size_t pos = 0, size_t len = npos);
string& erase(iterator p);
```

**参数说明：**

- `pos`：删除起始位置（默认为0）
- `len`：删除的字符个数（默认为 npos，表示到结尾）
- `p`：迭代器指向要删除的位置

**返回值：** 返回字符串的引用

**示例：**

```cpp
string str = "Hello World";
str.erase(5, 6);  // "Hello"  从位置5开始删除6个字符
cout << str << endl;

str.erase(2, 3);  // "He"  从位置2开始删除3个字符
cout << str << endl;
```

**注意事项：**

- `pos` 不能超过字符串长度
- `len` 可以超过剩余长度（会删除到结尾）
- `npos` 是 `size_t` 的最大值，表示"直到结尾"

---

#### 8. substr() - 获取子串

**函数签名：**

```cpp
string substr(size_t pos = 0, size_t len = npos) const;
```

**参数说明：**

- `pos`：子串起始位置（从0开始，默认为0）
- `len`：子串长度（默认为 npos，表示到结尾）

**返回值：** 返回一个新的 `string` 对象（子串）

**示例：**

```cpp
string str = "Hello World";
string sub1 = str.substr(0, 5);   // "Hello"  从0开始取5个字符
string sub2 = str.substr(6);     // "World"  从6开始到结尾
string sub3 = str.substr(6, 3);  // "Wor"   从6开始取3个字符

cout << sub1 << endl;
cout << sub2 << endl;
cout << sub3 << endl;
```

**注意事项：**

- 不会修改原字符串
- 如果 `pos` 超过长度，抛出异常
- 如果 `len` 超过剩余长度，只取到结尾

---

#### 9. find() - 查找子串

**函数签名：**

```cpp
size_t find(const string& str, size_t pos = 0) const;
size_t find(const char* s, size_t pos = 0) const;
size_t find(char c, size_t pos = 0) const;
```

**参数说明：**

- `str`：要查找的字符串
- `s`：要查找的 C 风格字符串
- `c`：要查找的字符
- `pos`：从哪个位置开始查找（默认为0）

**返回值：** `size_t` 类型

- 找到：返回第一次出现的位置（从0开始）
- 未找到：返回 `string::npos`

**示例：**

```cpp
string str = "Hello World";
size_t pos1 = str.find("World");  // 返回 6
size_t pos2 = str.find("o");     // 返回 4
size_t pos3 = str.find("xyz");   // 返回 npos

if (pos1 != string::npos) {
    cout << "找到 World 在位置: " << pos1 << endl;
}

if (pos3 == string::npos) {
    cout << "未找到 xyz" << endl;
}
```

**注意事项：**

- ⚠️ 返回值类型是 `size_t`，比较时不要用 `-1`
- ⚠️ 必须和 `string::npos` 比较，不能用 `pos == -1`
- `npos` 通常等于 `-1`（当转换为 `int` 时），但类型是 `size_t`
- 查找从左到右，返回第一个匹配的位置

#### 10. replace() - 替换子串

**函数签名：**

```cpp
string& replace(size_t pos, size_t len, const string& str);
```

**参数说明：**

- `pos`：替换起始位置
- `len`：要替换的字符个数
- `str`：替换的新字符串

**返回值：** 返回字符串的引用

**示例：**

```cpp
string str = "Hello World";
str.replace(6, 5, "C++");  // "Hello C++"  从位置6开始替换5个字符
cout << str << endl;
```

**注意事项：**

- 会修改原字符串
- `len` 可以超过剩余长度（会替换到结尾）

#### 11. c_str() - 转换为 C 风格字符串

**函数签名：**

```cpp
const char* c_str() const;
```

**参数说明：** 无参数

**返回值：** 返回 `const char*` 类型的指针

**示例：**

```cpp
string str = "Hello";
const char* cstr = str.c_str();  // 返回 "Hello"
printf("%s\n", cstr);
```

**注意事项：**

- 返回的指针指向的内容是临时的
- ⚠️ 不要保存这个指针，字符串修改后指针会失效
- 用于需要 `char*` 的函数（如 `printf`, `fopen` 等）

### C 风格字符串函数（<cstring>）

#### 1. strlen() - 获取字符串长度

**函数签名：**

```cpp
size_t strlen(const char* str);
```

**参数说明：**

- `str`：C 风格字符串（以 `\0` 结尾）

**返回值：** `size_t` 类型，表示字符串长度（不包括 `\0`）

**示例：**

```cpp
#include <cstring>

char str[] = "Hello";
size_t len = strlen(str);  // 返回 5
cout << "长度: " << len << endl;
```

**注意事项：**

- 必须以 `\0` 结尾
- 如果没有 `\0`，会继续读取直到遇到 `\0`（危险）

---

#### 2. strcpy() / strncpy() - 复制字符串

**函数签名：**

```cpp
char* strcpy(char* dest, const char* src);
char* strncpy(char* dest, const char* src, size_t n);
```

**参数说明：**

- `dest`：目标缓冲区
- `src`：源字符串
- `n`：最多复制的字符个数（strncpy）

**返回值：** 返回 `dest` 指针

**示例：**

```cpp
char dest[20];
char src[] = "Hello";
strcpy(dest, src);       // 复制整个字符串
cout << dest << endl;    // Hello

strncpy(dest, src, 3);   // 只复制前3个字符
cout << dest << endl;    // Hel
```

**注意事项：**

- ⚠️ `dest` 缓冲区必须足够大，否则会溢出
- `strcpy` 会复制 `\0`
- `strncpy` 如果复制了 `n` 个字符，不会自动添加 `\0`

---

#### 3. strcat() - 连接字符串

**函数签名：**

```cpp
char* strcat(char* dest, const char* src);
```

**参数说明：**

- `dest`：目标字符串（必须有足够空间）
- `src`：要连接的源字符串

**返回值：** 返回 `dest` 指针

**示例：**

```cpp
char dest[20] = "Hello ";
char src[] = "World";
strcat(dest, src);  // dest 变为 "Hello World"
cout << dest << endl;
```

**注意事项：**

- ⚠️ `dest` 必须有足够空间容纳连接后的字符串
- 会覆盖 `dest` 的 `\0`

---

#### 4. strcmp() - 比较字符串

**函数签名：**

```cpp
int strcmp(const char* str1, const char* str2);
```

**参数说明：**

- `str1`：第一个字符串
- `str2`：第二个字符串

**返回值：** `int` 类型

- `str1 < str2`：返回负数
- `str1 == str2`：返回 0
- `str1 > str2`：返回正数

**示例：**

```cpp
char str1[] = "Apple";
char str2[] = "Banana";
char str3[] = "Apple";

int result1 = strcmp(str1, str2);  // 负数
int result2 = strcmp(str1, str3);  // 0

if (result1 < 0) {
    cout << "str1 小于 str2" << endl;
}
if (result2 == 0) {
    cout << "str1 等于 str3" << endl;
}
```

**注意事项：**

- 比较是字典序（逐字符比较 ASCII 值）
- 返回值不一定是 `-1` 或 `1`，只是负数或正数

---

#### 5. strchr() / strstr() - 查找字符/子串

**函数签名：**

```cpp
char* strchr(const char* str, int c);
char* strstr(const char* str, const char* substr);
```

**参数说明：**

- `str`：要搜索的字符串
- `c`：要查找的字符
- `substr`：要查找的子串

**返回值：**

- 找到：返回指向位置的指针
- 未找到：返回 `NULL`

**示例：**

```cpp
char str[] = "Hello World";
char* p1 = strchr(str, 'o');   // 查找字符 'o'
char* p2 = strstr(str, "World"); // 查找子串 "World"

if (p1 != NULL) {
    cout << "找到 'o' 在位置: " << (p1 - str) << endl;
}
if (p2 != NULL) {
    cout << "找到 'World' 在位置: " << (p2 - str) << endl;
}
```

**注意事项：**

- `strchr` 找第一次出现的位置
- 返回的是指针，可以通过减法得到位置

## 数学函数

### 基本数学函数（<cmath>）

#### 1. abs() - 绝对值

**函数签名：**

```cpp
int abs(int x);
double abs(double x);
```

**参数说明：**

- `x`：要求绝对值的数（可以是 int 或 double）

**返回值：** 返回绝对值（类型与参数相同）

**示例：**

```cpp
#include <cmath>

int a = -10;
double b = -3.14;

cout << abs(a) << endl;   // 10
cout << abs(b) << endl;   // 3.14
```

**注意事项：**

- 需要包含 `<cmath>` 头文件

---

#### 2. sqrt() - 平方根

**函数签名：**

```cpp
double sqrt(double x);
```

**参数说明：**

- `x`：要求平方根的数（必须 >= 0）

**返回值：** 返回平方根（double 类型）

**示例：**

```cpp
double x = 16.0;
double result = sqrt(x);  // 4.0
cout << result << endl;

double y = 2.0;
cout << sqrt(y) << endl;  // 1.41421
```

**注意事项：**

- ⚠️ 如果 `x < 0`，返回 NaN（Not a Number）
- 返回类型总是 `double`

---

#### 3. pow() - 幂运算

**函数签名：**

```cpp
double pow(double base, double exp);
```

**参数说明：**

- `base`：底数
- `exp`：指数

**返回值：** 返回 `base^exp`（double 类型）

**示例：**

```cpp
double base = 2.0;
double exp = 3.0;
double result = pow(base, exp);  // 8.0 (2^3)
cout << result << endl;

cout << pow(10, 2) << endl;  // 100
cout << pow(2, 10) << endl;  // 1024
```

**注意事项：**

- 返回类型总是 `double`
- `pow(0, 0)` 的结果可能是 1 或实现相关

---

#### 4. floor() - 向下取整

**函数签名：**

```cpp
double floor(double x);
```

**参数说明：**

- `x`：浮点数

**返回值：** 返回小于或等于 `x` 的最大整数（double 类型）

**示例：**

```cpp
double x1 = 3.7;
double x2 = 3.2;
double x3 = -3.7;

cout << floor(x1) << endl;  // 3
cout << floor(x2) << endl;  // 3
cout << floor(x3) << endl;  // -4
```

**注意事项：**

- 向负无穷方向取整
- 返回类型是 `double`

---

#### 5. ceil() - 向上取整

**函数签名：**

```cpp
double ceil(double x);
```

**参数说明：**

- `x`：浮点数

```cpp
double x1 = 3.7;
```

#### 6. round() - 四舍五入

**函数签名：**

```cpp
double round(double x);
```

**参数说明：**

- `x`：浮点数

**返回值：** 返回四舍五入后的整数（double 类型）

```cpp
double x1 = 3.4;
```

#### 7. sin() / cos() / tan() - 三角函数

**函数签名：**

```cpp
double sin(double x);
double cos(double x);
double tan(double x);
```

**参数说明：**

- `x`：角度（以弧度为单位，不是度数）

**返回值：** 返回三角函数值（double 类型）

**示例：**

```cpp
#include <cmath>

double angle = 3.14159 / 4;  // 45度（弧度）

cout << sin(angle) << endl;  // 0.707
cout << cos(angle) << endl;  // 0.707
cout << tan(angle) << endl;  // 1.0
```

**注意事项：**

- ⚠️ 参数必须是弧度，不是度数
- 度数转弧度：`弧度 = 度数 * PI / 180`
- `PI` 可以定义为 `3.14159265358979323846`

---

#### 8. max() / min() - 最大值/最小值

**函数签名：**

```cpp
template<typename T>
const T& max(const T& a, const T& b);
template<typename T>
const T& min(const T& a, const T& b);
```

**参数说明：**

- `a`：第一个值
- `b`：第二个值

**返回值：** 返回较大/较小的值的引用

**示例：**

```cpp
#include <algorithm>

int a = 10, b = 20;

cout << max(a, b) << endl;  // 20
cout << min(a, b) << endl;  // 10

double x = 3.14, y = 2.71;
cout << max(x, y) << endl;  // 3.14
cout << min(x, y) << endl;  // 2.71
```

**注意事项：**

- 需要包含 `<algorithm>` 头文件
- 可以用于任意可比较的类型

---

#### 9. rand() - 随机数

**函数签名：**

```cpp
int rand();
```

**参数说明：** 无参数

**返回值：** 返回 0 到 RAND_MAX 之间的伪随机整数

**示例：**

```cpp
#include <cstdlib>
#include <ctime>

srand(time(NULL));  // 设置随机种子（只需调用一次）

int randomNum = rand();              // 0 到 RAND_MAX
int randomRange = rand() % 100;      // 0-99 的随机数
int randomMin = rand() % (max - min + 1) + min;  // min-max 范围的随机数

cout << randomNum << endl;
```

**注意事项：**

- ⚠️ 使用前必须调用 `srand()` 设置种子
- 如果不设置种子，每次运行程序产生的随机数序列相同
- `RAND_MAX` 通常定义为 32767
- `rand() % n` 的分布可能不均匀

---

#### 10. srand() - 设置随机种子

**函数签名：**

```cpp
void srand(unsigned int seed);
```

**参数说明：**

- `seed`：种子值

**返回值：** 无（void）

**示例：**

```cpp
#include <cstdlib>
#include <ctime>

srand(time(NULL));  // 用当前时间作为种子
srand(12345);      // 用固定值作为种子（用于调试）
```

**注意事项：**

- ⚠️ 只需在程序开始时调用一次
- 用 `time(NULL)` 作为种子可以每次产生不同序列
- 用固定种子可以复现随机序列（用于调试）

## 容器操作函数

### vector 常用函数

#### 1. push_back() - 尾部插入元素

**函数签名：**

```cpp
void push_back(const T& value);
```

**参数说明：**

- `value`：要插入的元素

**返回值：** 无（void）

**示例：**

```cpp
#include <vector>

vector<int> v;
v.push_back(10);  // v = {10}
v.push_back(20);  // v = {10, 20}
v.push_back(30);  // v = {10, 20, 30}

for (int x : v) {
    cout << x << " ";  // 10 20 30
}
```

**注意事项：**

- 在容器末尾添加元素
- 时间复杂度：平均 O(1)，最坏 O(n)（需要重新分配）

---

#### 2. pop_back() - 删除尾部元素

**函数签名：**


## 排序和查找算法

### sort() - 排序

**函数签名：**

```cpp
void sort(RandomAccessIterator first, RandomAccessIterator last);
void sort(RandomAccessIterator first, RandomAccessIterator last, Compare comp);
```

**参数说明：**

- `first`：排序范围起始位置
- `last`：排序范围结束位置
- `comp`：自定义比较函数（可选）

**返回值：** 无（void）

**示例：**

```cpp
#include <algorithm>

vector<int> v = {5, 2, 8, 1, 9};

// 升序排序（默认）
sort(v.begin(), v.end());
for (int x : v) cout << x << " ";  // 1 2 5 8 9
cout << endl;

// 降序排序
sort(v.begin(), v.end(), greater<int>());
for (int x : v) cout << x << " ";  // 9 8 5 2 1
cout << endl;

// 自定义排序
sort(v.begin(), v.end(), [](int a, int b) {
    return a > b;  // 降序
});
```

**注意事项：**

- 时间复杂度：O(n log n)
- 默认是升序排序
- 相等元素的相对位置可能改变（不稳定排序）

### find() - 查找元素

**函数签名：**

- `last`：查找范围结束位置
- `value`：要查找的值

**注意事项：**

- 时间复杂度：O(n)

- `value`：要查找的值

**返回值：** 返回迭代器

- `lower_bound`：第一个 `>= value` 的位置
- `upper_bound`：第一个 `> value` 的位置

**示例：**

```cpp
vector<int> v = {1, 3, 3, 3, 5};

auto it1 = lower_bound(v.begin(), v.end(), 3);
cout << "lower_bound: " << (it1 - v.begin()) << endl;  // 1 (第一个>=3)

auto it2 = upper_bound(v.begin(), v.end(), 3);
cout << "upper_bound: " << (it2 - v.begin()) << endl;  // 4 (第一个>3)
```

**注意事项：**

- ⚠️ 容器必须已经排序！
- 可以用来统计某个值的个数：`upper - lower`

### 迭代器基础：begin() / end()

STL 容器和算法大量使用**迭代器**来表示位置范围。可以把迭代器理解为"高级指针"。

**常用函数：**

```cpp
container.begin();   // 指向容器第一个元素
container.end();     // 指向容器最后一个元素的**下一个位置**
container.rbegin();  // 反向迭代器，指向最后一个元素
container.rend();    // 反向迭代器，指向第一个元素的前一个位置
```

**示例：**

```cpp
#include <vector>
#include <iostream>
using namespace std;

int main() {
    vector<int> v = {10, 20, 30, 40, 50};

    // 用迭代器遍历
    for (auto it = v.begin(); it != v.end(); ++it) {
        cout << *it << " ";   // *it 解引用，获取元素的值
    }
    // 输出: 10 20 30 40 50

    // begin() + n 可以偏移位置
    cout << *(v.begin() + 2) << endl;  // 30（第三个元素）

    // end() - 1 指向最后一个元素
    cout << *(v.end() - 1) << endl;    // 50

    return 0;
}
```

**注意事项：**

- `begin()` 到 `end()` 是**左闭右开**区间 `[begin, end)`，即包含 `begin` 但不包含 `end`
- 算法函数（如 `sort`、`find`、`count`）都接受迭代器参数
- `auto` 关键字可以自动推导迭代器类型，避免写长长的类型名

---

### auto 关键字

`auto` 让编译器**自动推导**变量类型，简化代码。

**示例：**

```cpp
auto x = 10;          // int
auto d = 3.14;        // double
auto s = "hello";     // const char*
auto str = string("hi"); // std::string

// 常用于迭代器（类型很长）
map<string, int> m;
auto it = m.find("apple");   // 自动推导为 map<string,int>::iterator

// 常用于 range-based for
vector<int> v = {1, 2, 3};
for (auto& val : v) {
    val *= 2;   // 引用，可以修改元素
}
```

**注意事项：**

- 需要 C++11 或更高
- `auto` 推导的是**值类型**（会拷贝），加 `&` 是引用（不会拷贝）

### Range-based for 循环

遍历容器/数组的简洁写法。

**语法：**

```cpp
for (类型 变量 : 容器) {
    // 使用变量
}
```

**示例：**

```cpp
vector<int> v = {1, 2, 3, 4, 5};

// 只读遍历
for (int x : v) {
    cout << x << " ";   // 1 2 3 4 5
}

// 修改元素（用引用 &）
for (int& x : v) {
    x *= 2;             // 每个元素乘以2
}

// 用 auto 更简洁
for (auto x : v) {
    cout << x << " ";   // 2 4 6 8 10
}

// 遍历字符串
string str = "Hello";
for (char c : str) {
    cout << c << " ";   // H e l l o
}

// 遍历数组
int arr[] = {10, 20, 30};
for (int x : arr) {
    cout << x << " ";   // 10 20 30
}
```

### pair 和 make_pair

`pair` 可以把**两个值**打包成一个整体，常用于 map 和函数返回两个值。

**定义：**

```cpp
#include <utility>   // pair 在此头文件中（大多数STL头文件已间接包含）

pair<int, int> p1;                   // 两个 int
pair<string, double> p2;             // string 和 double
pair<int, int> p3 = {1, 2};          // 初始化
pair<int, int> p4 = make_pair(1, 2); // 用 make_pair 创建
```

**访问成员：**

- `p.first` — 第一个值
- `p.second` — 第二个值

**示例：**

```cpp
pair<string, int> student = {"张三", 90};
cout << student.first << ": " << student.second << endl;
// 张三: 90

// pair 排序：默认按 first 排序，first 相同按 second
vector<pair<int, int>> vp = {{3, 1}, {1, 4}, {2, 2}};
sort(vp.begin(), vp.end());
for (auto& p : vp) {
    cout << p.first << "," << p.second << " ";
}
// 输出: 1,4 2,2 3,1

// map 的每个元素就是一个 pair
map<string, int> m;
m["apple"] = 5;
for (auto& p : m) {
    cout << p.first << " = " << p.second << endl;
    // apple = 5
}
```

### 格式化输出补充

#### 1. fixed - 固定小数点输出

**作用：** 让浮点数以**小数形式**输出（不用科学计数法）。

#### 5. endl 与 "\n" 的区别

## 常见考试陷阱

### 1. string::find() 的返回值

**错误：**

```cpp
string str = "Hello";
size_t pos = str.find("World");
if (pos == -1) {  // ❌ 错误！
    cout << "未找到" << endl;
}
```

**正确：**

```cpp
string str = "Hello";
size_t pos = str.find("World");
if (pos == string::npos) {  // ✅ 正确
    cout << "未找到" << endl;
}
```

---

### 2. vector 下标越界

**不安全：**

```cpp
vector<int> v = {1, 2, 3};
cout << v[10] << endl;  // ❌ 未定义行为，可能崩溃
```

**安全：**

```cpp
vector<int> v = {1, 2, 3};
cout << v.at(10) << endl;  // ✅ 抛出异常
```

---

### 3. getline() 和 cin 混用

**错误：**

```cpp
int n;
cin >> n;
string line;
getline(cin, line);  // ❌ 读取到空行！
```

**正确：**

```cpp
int n;
cin >> n;
cin.ignore();  // ✅ 忽略换行符
string line;
getline(cin, line);
```

---

### 4. sort() 自定义比较

**升序（默认）：**

```cpp
sort(v.begin(), v.end());
```

**降序：**

```cpp
sort(v.begin(), v.end(), greater<int>());
```

**自定义：**

```cpp
sort(v.begin(), v.end(), [](int a, int b) {
    return a > b;  // 返回 true 表示 a 应该在 b 前面
});
```

---

### 5. map 的 count() 返回值

```cpp
map<string, int> m;
m["apple"] = 5;

// map 的 count() 只返回 0 或 1
if (m.count("apple")) {  // ✅ 正确
    cout << "存在" << endl;
}
```
