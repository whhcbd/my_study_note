# C++ 语法糖完全指南

## 什么是语法糖？

语法糖（Syntactic Sugar）是指编程语言中让代码更简洁、更易读的语法特性。它们不改变程序的功能，但能让代码写得更快、读起来更清晰。C++ 从 C++11 开始引入了大量语法糖，让现代 C++ 代码变得更加优雅。

## 1. 类型推断（Type Deduction）

### auto 关键字

`auto` 让编译器自动推断变量类型，不用手动写类型名称。

```cpp
// 传统方式
std::vector<int> numbers = {1, 2, 3, 4, 5};
std::vector<int>::iterator it = numbers.begin();

// 使用 auto
std::vector<int> numbers = {1, 2, 3, 4, 5};
auto it = numbers.begin();  // 编译器自动推断为 std::vector<int>::iterator

// 更多例子
auto x = 42;              // int
auto y = 3.14;            // double
auto z = "hello";         // const char*
auto& ref = x;            // int& (引用)
auto* ptr = &x;           // int* (指针)
auto&& rref = std::move(x); // int&& (右值引用)
```

**优势：**

- 减少代码量，特别是复杂类型
- 避免类型名称拼写错误
- 当类型可能改变时，auto 会自动适应

**注意事项：**

- 不要过度使用，确保代码可读性
- `auto` 会忽略顶层 const，如需保留请使用 `const auto`

### decltype 关键字

`decltype` 用于查询表达式的类型，可以获取表达式的确切类型。

```cpp
int x = 10;
decltype(x) y = 20;  // y 的类型是 int
decltype(x + 1) z = 30;  // z 的类型是 int

const int& ref = x;
decltype(ref) ref2 = y;  // ref2 的类型是 const int&

// 在函数中使用
template<typename T, typename U>
auto add(T t, U u) -> decltype(t + u) {
    return t + u;
}
```

### 尾返回类型（Trailing Return Type）

在函数参数列表后声明返回类型，特别适用于返回类型依赖于参数的情况。

```cpp
// 传统方式
template<typename T, typename U>
std::add_const<decltype(t + u)>::type add_old(T t, U u) {
    return t + u;
}

// 尾返回类型（C++11）
template<typename T, typename U>
auto add(T t, U u) -> decltype(t + u) {
    return t + u;
}

// C++14 可以省略尾返回类型
template<typename T, typename U>
auto add(T t, U u) {
    return t + u;  // 自动推断返回类型
}
```

## 2. 循环语法糖

### 范围 for 循环（Range-based for loop）

C++11 引入的简洁遍历容器的方式。

```cpp
// 传统方式
std::vector<int> numbers = {1, 2, 3, 4, 5};
for (std::vector<int>::iterator it = numbers.begin(); it != numbers.end(); ++it) {
    std::cout << *it << " ";
}

// 使用范围 for 循环
std::vector<int> numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    std::cout << num << " ";
}

// 使用引用避免拷贝
for (int& num : numbers) {
    num *= 2;  // 修改原元素
}

// 使用 const 引用只读访问
for (const int& num : numbers) {
    std::cout << num << " ";
}

// 配合 auto 使用
for (auto& num : numbers) {
    num *= 2;
}
```

**适用场景：**

- 遍历数组、容器（vector、list、map 等）
- 遍历初始化列表
- 自定义类型（需提供 begin() 和 end() 方法）

## 3. 初始化语法糖

### 初始化列表（Initializer List）

统一的初始化语法，使用花括号 `{}`。

```cpp
// 基本类型
int x = 42;
int y{42};      // C++11 初始化列表
int z{3.14};    // 编译错误！禁止隐式窄化转换

// 数组
int arr1[] = {1, 2, 3};
int arr2[]{1, 2, 3};  // 可以省略 =

// 容器
std::vector<int> v1 = {1, 2, 3, 4, 5};
std::vector<int> v2{1, 2, 3, 4, 5};

std::map<std::string, int> scores = {
    {"Alice", 95},
    {"Bob", 87},
    {"Charlie", 92}
};

// 自定义结构体
struct Point {
    int x, y;
};
Point p1 = {10, 20};
Point p2{10, 20};

// 使用 std::initializer_list 接收初始化列表
void print_list(std::initializer_list<int> list) {
    for (auto num : list) {
        std::cout << num << " ";
    }
}
print_list({1, 2, 3, 4, 5});
```

### std::array

固定大小数组的现代化替代品，提供 STL 容器接口。

```cpp
// 传统数组
int arr[5] = {1, 2, 3, 4, 5};
int size = sizeof(arr) / sizeof(arr[0]);  // 手动计算大小

// std::array
#include <array>
std::array<int, 5> arr = {1, 2, 3, 4, 5};
int size = arr.size();  // 直接获取大小
arr[0] = 10;            // 访问元素

std::sort(arr.begin(), arr.end());
```

## 4. 函数相关语法糖

### Lambda 表达式

匿名函数，可以就地定义函数，特别适合用于算法回调。

```cpp
// 基本语法
auto lambda = [](int x, int y) {
    return x + y;
};
int result = lambda(3, 4);  // result = 7

// 捕获外部变量
int factor = 2;
auto multiply = [factor](int x) {
    return x * factor;  // factor 被捕获
};

// 引用捕获
int sum = 0;
std::vector<int> numbers = {1, 2, 3, 4, 5};
for_each(numbers.begin(), numbers.end(), [&sum](int x) {
    sum += x;  // 通过引用修改外部变量
});

// 混合捕获
int a = 1, b = 2;
auto func = [a, &b]() {
    // a 是值捕获（只读）
    // b 是引用捕获（可修改）
    b = a + b;
};

// 捕获所有（慎用）
auto capture_all_by_value = [=]() { /* 所有外部变量按值捕获 */ };
auto capture_all_by_ref = [&]() { /* 所有外部变量按引用捕获 */ };

// 可变 lambda
auto mutable_lambda = [factor](int x) mutable {
    factor++;  // 修改捕获的副本
    return x * factor;
};
```

### std::function

通用函数包装器，可以存储任何可调用对象。

```cpp
#include <functional>

// 存储普通函数指针
int add(int a, int b) {
    return a + b;
}
std::function<int(int, int)> func = add;

// 存储 lambda
auto lambda = [](int a, int b) { return a - b; };
func = lambda;

// 存储函数对象（仿函数）
struct Multiply {
    int operator()(int a, int b) {
        return a * b;
    }
};
func = Multiply();

// 使用
std::cout << func(3, 4) << std::endl;

// 作为函数参数
void apply_operation(std::function<int(int, int)> op, int a, int b) {
    std::cout << op(a, b) << std::endl;
}

apply_operation(add, 5, 3);
apply_operation([](int a, int b) { return a / b; }, 10, 2);
```

### 函数默认参数

为函数参数提供默认值，调用时可以省略。

```cpp
void greet(std::string name = "Guest", int times = 1) {
    for (int i = 0; i < times; ++i) {
        std::cout << "Hello, " << name << "!" << std::endl;
    }
}

greet();                    // Hello, Guest!
greet("Alice");             // Hello, Alice!
greet("Bob", 3);            // Hello, Bob! (3次)
```

### constexpr 函数

编译期常量表达式，可以在编译时计算。

```cpp
constexpr int factorial(int n) {
    return n <= 1 ? 1 : n * factorial(n - 1);
}

// 编译期计算
constexpr int result = factorial(5);  // 编译时计算，result = 120

// 数组大小需要编译时常量
int arr[factorial(4)];  // 大小为 24 的数组

constexpr int square(int x) {
    return x * x;
}

static_assert(square(10) == 100, "Square of 10 should be 100");
```

## 5. 智能指针（Smart Pointers）

自动管理内存的指针，避免内存泄漏。

### std::unique_ptr

独占所有权的智能指针，同一时间只能有一个 unique_ptr 指向对象。

```cpp
#include <memory>

// 创建 unique_ptr
std::unique_ptr<int> ptr1(new int(42));
auto ptr2 = std::make_unique<int>(42);  // 推荐方式

// 转移所有权
std::unique_ptr<int> ptr3 = std::move(ptr1);  // ptr1 变为空
// ptr1 现在不能使用

// 访问对象
*ptr3 = 100;
ptr3.reset();  // 显式释放对象

// 自定义对象
class Widget {
public:
    Widget() { std::cout << "Widget created\n"; }
    ~Widget() { std::cout << "Widget destroyed\n"; }
    void doSomething() { std::cout << "Doing something\n"; }
};

auto widget = std::make_unique<Widget>();
widget->doSomething();
```

### std::shared_ptr

共享所有权的智能指针，多个 shared_ptr 可以指向同一个对象，使用引用计数。

```cpp
#include <memory>

// 创建 shared_ptr
std::shared_ptr<int> ptr1 = std::make_shared<int>(42);

// 复制共享所有权
std::shared_ptr<int> ptr2 = ptr1;  // 引用计数 = 2
std::shared_ptr<int> ptr3 = ptr1;  // 引用计数 = 3

// 访问对象
*ptr1 = 100;
std::cout << *ptr2 << std::endl;  // 输出 100

// 查看引用计数
std::cout << ptr1.use_count() << std::endl;  // 3

// 释放所有权
ptr1.reset();  // 引用计数 = 2
ptr2.reset();  // 引用计数 = 1
// ptr3 析构时，对象被真正释放
```

### std::weak_ptr

弱引用，不增加引用计数，用于解决 shared_ptr 的循环引用问题。

```cpp
#include <memory>

std::shared_ptr<int> shared = std::make_shared<int>(42);
std::weak_ptr<int> weak = shared;

// weak_ptr 不能直接访问对象
if (auto locked = weak.lock()) {
    // lock() 返回 shared_ptr，如果对象还存在
    std::cout << *locked << std::endl;
}

// 检查对象是否还存在
if (!weak.expired()) {
    std::cout << "Object still exists\n";
}
```

**循环引用示例：**

```cpp
class Node {
public:
    std::shared_ptr<Node> next;
    std::weak_ptr<Node> prev;  // 使用 weak_ptr 避免循环引用

    ~Node() { std::cout << "Node destroyed\n"; }
};

auto node1 = std::make_shared<Node>();
auto node2 = std::make_shared<Node>();
node1->next = node2;
node2->prev = node1;  // 弱引用，不会增加引用计数
```

## 6. 容器和数据结构语法糖

### std::tuple

固定大小的异构容器，可以存储不同类型的元素。

```cpp
#include <tuple>

// 创建 tuple
auto person = std::make_tuple("Alice", 25, 95.5);

// 访问元素
std::string name = std::get<0>(person);  // "Alice"
int age = std::get<1>(person);            // 25
double score = std::get<2>(person);       // 95.5

// 使用结构化绑定（C++17）
auto [n, a, s] = person;
std::cout << name << ", " << age << ", " << score << std::endl;

// 修改元素
std::get<1>(person) = 26;

// tuple 大小
int size = std::tuple_size<decltype(person)>::value;  // 3

// 遍历 tuple（需要模板元编程技巧）
```

### 结构化绑定（Structured Binding，C++17）

将复合类型的成员解构到单独变量中。

```cpp
// 对数组
int arr[3] = {1, 2, 3};
auto [x, y, z] = arr;  // x=1, y=2, z=3

// 对 pair
std::pair<std::string, int> person = {"Bob", 30};
auto [name, age] = person;  // name="Bob", age=30

// 对 tuple
auto data = std::make_tuple(10, 3.14, "hello");
auto [i, d, s] = data;  // i=10, d=3.14, s="hello"

// 对结构体
struct Point {
    int x, y;
};
Point p{10, 20};
auto [px, py] = p;  // px=10, py=20

// 在范围 for 循环中使用
std::map<std::string, int> scores = {{"Alice", 95}, {"Bob", 87}};
for (const auto& [name, score] : scores) {
    std::cout << name << ": " << score << std::endl;
}
```

### std::optional

表示可能不存在的值，比指针更清晰地表达"可能为空"。

```cpp
#include <optional>

// 函数可能返回值，也可能不返回
std::optional<int> divide(int a, int b) {
    if (b == 0) {
        return std::nullopt;  // 表示没有值
    }
    return a / b;
}

auto result = divide(10, 2);
if (result) {
    std::cout << "Result: " << *result << std::endl;  // 解引用获取值
} else {
    std::cout << "Cannot divide by zero\n";
}

// 提供默认值
int value = result.value_or(0);  // 如果没有值，返回 0

// 直接访问（如果有值）
if (result.has_value()) {
    std::cout << result.value() << std::endl;
}
```

### std::variant

类型安全的联合体，可以存储多种类型中的一种。

```cpp
#include <variant>

// 可以存储 int, double, 或 string
std::variant<int, double, std::string> value;

value = 42;                    // 存储整数
value = 3.14;                  // 存储浮点数
value = "hello";               // 存储字符串

// 访问当前值
if (std::holds_alternative<int>(value)) {
    int i = std::get<int>(value);
}

// 使用访问者模式（Visitor）
std::visit([](auto&& arg) {
    std::cout << arg << std::endl;
}, value);

// 函数重载访问者（C++17）
auto visitor = [](auto&& arg) {
    using T = std::decay_t<decltype(arg)>;
    if constexpr (std::is_same_v<T, int>)
        std::cout << "int: " << arg << std::endl;
    else if constexpr (std::is_same_v<T, double>)
        std::cout << "double: " << arg << std::endl;
    else if constexpr (std::is_same_v<T, std::string>)
        std::cout << "string: " << arg << std::endl;
};
std::visit(visitor, value);
```

## 7. 字符串处理语法糖

### 原始字符串字面量

避免转义字符，直接写字符串内容。

```cpp
// 传统方式（需要转义）
std::string path1 = "C:\\Users\\Documents\\file.txt";
std::string regex1 = "\\d+\\w*\\.";

// 原始字符串字面量
std::string path2 = R"(C:\Users\Documents\file.txt)";
std::string regex2 = R"(\d+\w*\.)";

// 多行字符串
std::string html = R"(
<html>
    <body>
        <h1>Hello, World!</h1>
    </body>
</html>
)";

// 如果字符串中包含 )" ，可以自定义分隔符
std::string text = R"delimiter(Contains )" inside)delimiter";
```

### std::string_view（C++17）

轻量级的字符串视图，不拥有数据，避免字符串拷贝。

```cpp
#include <string_view>

// 传统方式：会产生字符串拷贝
void print_string(const std::string& str) {
    std::cout << str << std::endl;
}

// 使用 string_view：零拷贝
void print_string(std::string_view sv) {
    std::cout << sv << std::endl;
}

// 可以接受 string、字符串字面量、字符数组
std::string s = "hello";
print_string(s);           // std::string
print_string("world");     // const char*
print_string(s.c_str());   // const char*

// 子串操作（不产生拷贝）
std::string text = "Hello, World!";
std::string_view view = text;
std::string_view sub = view.substr(0, 5);  // "Hello"
```

### 字符串字面量运算符

```cpp
// std::string 字面量
using namespace std::string_literals;
auto s1 = "hello"s;  // std::string 类型
auto s2 = "world";   // const char* 类型

// 拼接
auto combined = s1 + " " + s2;

// std::chrono 字面量
using namespace std::chrono_literals;
auto duration = 5s;      // 5 秒
auto milliseconds = 100ms;  // 100 毫秒
```

## 8. 并发语法糖

### std::thread

创建和管理线程。

```cpp
#include <thread>
#include <iostream>

void worker(int id) {
    std::cout << "Worker " << id << " is working\n";
}

// 创建线程
std::thread t1(worker, 1);
std::thread t2(worker, 2);

// 等待线程完成
t1.join();
t2.join();

// 使用 lambda
std::thread t3([]() {
    std::cout << "Lambda thread\n";
});
t3.join();
```

### std::future 和 std::async

异步执行函数并获取结果。

```cpp
#include <future>
#include <iostream>

int compute(int x) {
    std::this_thread::sleep_for(std::chrono::seconds(1));
    return x * x;
}

// 异步执行
auto future = std::async(std::launch::async, compute, 10);

// 做其他事情...
std::cout << "Doing other work...\n";

// 获取结果（会阻塞直到完成）
int result = future.get();
std::cout << "Result: " << result << std::endl;
```

## 9. 模板语法糖

### 别名模板（Alias Template）

为模板类型定义别名。

```cpp
// 传统 typedef（不支持模板）
typedef std::map<std::string, int> StringIntMap;

// using 别名（支持模板）
template<typename T>
using Vec = std::vector<T>;

Vec<int> numbers = {1, 2, 3, 4, 5};
Vec<std::string> words = {"hello", "world"};

// 复杂类型别名
template<typename T>
using Map = std::map<std::string, T>;

Map<int> scores;
Map<double> prices;
```

### 可变参数模板（Variadic Template）

接受可变数量模板参数。

```cpp
// 递归终止条件
void print() {
    std::cout << std::endl;
}

// 可变参数模板函数
template<typename T, typename... Args>
void print(T first, Args... args) {
    std::cout << first << " ";
    print(args...);  // 递归调用
}

// 使用
print(1, 2.5, "hello", 'a');

// 折叠表达式（C++17）
template<typename... Args>
auto sum(Args... args) {
    return (args + ...);  // 一元右折叠
}

int total = sum(1, 2, 3, 4, 5);  // 15
```

### if constexpr（C++17）

编译期条件判断，根据类型选择不同实现。

```cpp
template<typename T>
auto get_value(T t) {
    if constexpr (std::is_pointer_v<T>) {
        return *t;  // 如果是指针，解引用
    } else {
        return t;   // 否则直接返回
    }
}

int x = 10;
int* ptr = &x;

auto v1 = get_value(x);    // 10
auto v2 = get_value(ptr);  // 10
```

## 10. 其他有用语法糖

### nullptr

空指针常量，替代 NULL。

```cpp
// 传统方式（NULL 通常是 0）
int* ptr1 = NULL;

// 使用 nullptr（类型安全）
int* ptr2 = nullptr;

// 函数重载
void func(int);
void func(char*);

func(NULL);     // 调用 func(int)
func(nullptr);  // 调用 func(char*)
```

### static_assert

编译期断言，在编译时检查条件。

```cpp
// 检查类型大小
static_assert(sizeof(int) >= 4, "int must be at least 4 bytes");

// 检查模板参数
template<typename T>
void process() {
    static_assert(std::is_arithmetic_v<T>, "T must be arithmetic type");
}

process<int>();     // OK
process<std::string>();  // 编译错误
```

### noexcept

声明函数不抛出异常。

```cpp
// 声明不抛出异常
int safe_divide(int a, int b) noexcept {
    return a / b;
}

// 条件 noexcept
template<typename T>
void swap(T& a, T& b) noexcept(std::is_nothrow_move_constructible_v<T>) {
    T temp = std::move(a);
    a = std::move(b);
    b = std::move(temp);
}
```

### std::move 和 std::forward

移动语义，避免不必要的拷贝。

```cpp
#include <utility>

// std::move：将左值转换为右值引用
std::string str1 = "Hello";
std::string str2 = std::move(str1);  // 移动，str1 变为空
// str1 现在处于有效但未定义的状态

// 完美转发
template<typename T>
void wrapper(T&& arg) {
    process(std::forward<T>(arg));  // 保持值的类别
}

// 使用示例
class Widget {
public:
    Widget(const Widget& other) {
        std::cout << "Copy constructor\n";
    }

    Widget(Widget&& other) noexcept {
        std::cout << "Move constructor\n";
    }
};

Widget w1;
Widget w2 = w1;              // 拷贝构造
Widget w3 = std::move(w1);   // 移动构造
```

### 属性（Attributes）

为编译器提供额外信息。

```cpp
// [[nodiscard]]：忽略返回值时产生警告
[[nodiscard]] int get_value() {
    return 42;
}

int x = get_value();  // OK
get_value();         // 警告：忽略返回值

// [[deprecated]]：标记已弃用
[[deprecated("Use new_function instead")]]
void old_function() {}

old_function();  // 警告：已弃用

// [[maybe_unused]]：避免未使用变量警告
void process([[maybe_unused]] int param) {
    // 即使 param 未使用也不会警告
}

// [[fallthrough]]：在 switch 中有意穿透
switch (value) {
    case 1:
        do_something();
        [[fallthrough]];  // 明确表示有意穿透
    case 2:
        do_another_thing();
        break;
}
```

## 总结

C++ 语法糖让代码更简洁、更安全、更易读：

| 语法糖           | 主要优势                 | 使用场景                 |
| ---------------- | ------------------------ | ------------------------ |
| auto             | 减少类型重复，提高可读性 | 复杂类型、迭代器、lambda |
| 范围 for         | 简化遍历代码             | 遍历容器、数组           |
| 初始化列表       | 统一初始化语法           | 所有类型的初始化         |
| Lambda           | 就地定义函数             | 算法回调、事件处理       |
| 智能指针         | 自动内存管理             | 动态内存分配             |
| 结构化绑定       | 解构复合类型             | 访问 tuple、pair、结构体 |
| optional/variant | 类型安全的可选值/多态    | API 设计、配置处理       |
| string_view      | 零拷贝字符串操作         | 字符串参数传递           |

**学习建议：**

1. 从简单开始：先掌握 auto、范围 for、初始化列表
2. 逐步深入：学习 lambda、智能指针
3. 实践应用：在新项目中使用这些特性
4. 理解原理：了解语法糖背后的机制

现代 C++ 语法糖让代码更优雅，但记住：清晰的代码比简洁的代码更重要。选择合适的工具，不要为了使用语法糖而使用语法糖。
