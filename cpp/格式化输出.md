## 什么是格式化输出

**格式化输出**是指按照指定的格式、宽度、精度等要求，将数据输出到屏幕、文件或其他输出设备的过程。

### 为什么需要格式化输出？

想象你在打印一张成绩单：
- 姓名需要对齐
- 分数要保留小数点后两位
- 百分比要显示百分号
- 日期要按照年-月-日的格式

格式化输出就是为了让数据显示得整齐、美观、易读。

### 简单对比

```cpp
#include <iostream>

int main() {
    int score = 85;
    double average = 92.5678;
    std::string name = "张三";

    // 没有格式化
    std::cout << name << score << average << std::endl;
    // 输出：张三8592.5678（难以阅读）

    // 格式化后
    std::cout << "姓名: " << name << ", 成绩: " << score << ", 平均分: " << average << std::endl;
    // 输出：姓名: 张三, 成绩: 85, 平均分: 92.5678（清晰易读）

    return 0;
}
```

## 基本输出方法

### 1. std::cout（标准输出流）

`std::cout` 是 C++ 最常用的输出方式，使用 `<<` 运算符将数据输出到控制台。

```cpp
#include <iostream>

int main() {
    // 输出整数
    int age = 25;
    std::cout << "年龄: " << age << std::endl;

    // 输出浮点数
    double price = 19.99;
    std::cout << "价格: " << price << std::endl;

    // 输出字符串
    std::string name = "李四";
    std::cout << "姓名: " << name << std::endl;

    // 链式输出
    std::cout << "姓名: " << name << ", 年龄: " << age << ", 价格: " << price << std::endl;

    return 0;
}
```

### 2. std::cerr（标准错误流）

用于输出错误信息，不会被重定向到文件。

```cpp
#include <iostream>

int main() {
    std::cout << "这是正常输出" << std::endl;
    std::cerr << "这是错误信息" << std::endl;

    return 0;
}
```

### 3. std::clog（标准日志流）

用于输出日志信息，通常用于调试。

```cpp
#include <iostream>

int main() {
    std::clog << "程序开始运行" << std::endl;
    std::clog << "正在处理数据..." << std::endl;

    return 0;
}
```

### 4. 换行和刷新

```cpp
#include <iostream>

int main() {
    // std::endl - 换行并刷新缓冲区
    std::cout << "第一行" << std::endl;
    std::cout << "第二行" << std::endl;

    // '\n' - 仅换行，不刷新缓冲区（更快）
    std::cout << "第三行\n";
    std::cout << "第四行\n";

    // std::flush - 刷新缓冲区（不换行）
    std::cout << "立即输出";
    std::cout << std::flush;

    return 0;
}
```

## 格式化控制符

### 基本格式化标志

C++ 提供了一系列格式化标志来控制输出格式。

```cpp
#include <iostream>

    // 十六进制（小写）
    std::cout << "十六进制: " << std::hex << number << std::endl;

    // 十六进制（大写）
    std::cout << "十六进制(大写): " << std::uppercase << std::hex << number << std::endl;

    // 恢复十进制
    std::cout << std::dec << "恢复十进制: " << number << std::endl;

    // 显示前缀（0x 或 0）
    std::cout << "带前缀: " << std::showbase << std::hex << number << std::endl;

    return 0;
}
```

## 格式化操作符（iomanip）

`<iomanip>` 头文件提供了丰富的格式化操作符。

### 1. setw() - 设置宽度

```cpp
#include <iostream>
#include <iomanip>

int main() {
    std::cout << "姓名" << "  " << "年龄" << "  " << "成绩" << std::endl;
    std::cout << "----" << "  " << "----" << "  " << "----" << std::endl;

    std::cout << std::setw(10) << "张三"
              << std::setw(5) << 20
              << std::setw(8) << 85.5 << std::endl;

    std::cout << std::setw(10) << "李四"
              << std::setw(5) << 22
              << std::setw(8) << 92.3 << std::endl;

    return 0;
}
```

### 2. setprecision() - 设置精度

```cpp
#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.141592653589793;

    std::cout << "原始值: " << pi << std::endl;

    // 设置有效数字位数
    std::cout << "有效数字3位: " << std::setprecision(3) << pi << std::endl;
    std::cout << "有效数字5位: " << std::setprecision(5) << pi << std::endl;

    // 固定小数点格式
    std::cout << "固定小数2位: " << std::fixed << std::setprecision(2) << pi << std::endl;
    std::cout << "固定小数4位: " << std::fixed << std::setprecision(4) << pi << std::endl;

    // 科学计数法
    std::cout << "科学计数法: " << std::scientific << std::setprecision(2) << pi << std::endl;

    return 0;
}
```

### 3. setfill() - 设置填充字符

```cpp
#include <iostream>
#include <iomanip>

int main() {
    // 使用 * 填充
    std::cout << std::setfill('*');
    std::cout << std::setw(20) << "姓名" << std::endl;
    std::cout << std::setw(20) << "张三" << std::endl;

    // 使用 - 填充
    std::cout << std::setfill('-');
    std::cout << std::setw(20) << "李四" << std::endl;

    // 使用 0 填充（常用于数字）
    std::cout << std::setfill('0');
    std::cout << std::setw(5) << 42 << std::endl;

    return 0;
}
```

### 4. left / right - 对齐方式

```cpp
#include <iostream>
#include <iomanip>

int main() {
    // 左对齐
    std::cout << std::left << std::setfill('.');
    std::cout << std::setw(20) << "左对齐文本" << std::endl;

    // 右对齐
    std::cout << std::right << std::setfill('.');
    std::cout << std::setw(20) << "右对齐文本" << std::endl;

    return 0;
}
```

### 5. showpos / noshowpos - 显示正号

```cpp
#include <iostream>
#include <iomanip>

int main() {
    int positive = 42;
    int negative = -17;

    // 显示正号
    std::cout << std::showpos;
    std::cout << "正数: " << positive << std::endl;
    std::cout << "负数: " << negative << std::endl;

    // 不显示正号
    std::cout << std::noshowpos;
    std::cout << "正数: " << positive << std::endl;
    std::cout << "负数: " << negative << std::endl;

    return 0;
}
```

## printf 风格格式化

C++ 也支持 C 语言的 `printf` 函数，使用格式化字符串。

### 基本 printf 用法

```cpp
#include <cstdio>

int main() {
    int age = 25;
    double score = 92.5;
    char grade = 'A';
    const char* name = "张三";

    printf("姓名: %s\n", name);
    printf("年龄: %d\n", age);
    printf("成绩: %.2f\n", score);
    printf("等级: %c\n", grade);

    return 0;
}
```

### printf 格式化说明符

| 说明符 | 类型 | 示例 |
|--------|------|------|
| `%d` | 整数 | `printf("%d", 42)` |
| `%f` | 浮点数 | `printf("%f", 3.14)` |
| `%.2f` | 保留2位小数 | `printf("%.2f", 3.14159)` |
| `%e` | 科学计数法 | `printf("%e", 12345.67)` |
| `%c` | 字符 | `printf("%c", 'A')` |
| `%s` | 字符串 | `printf("%s", "Hello")` |
| `%x` | 十六进制 | `printf("%x", 255)` |
| `%o` | 八进制 | `printf("%o", 8)` |
| `%%` | 百分号 | `printf("完成度: %d%%", 85)` |

### printf 宽度和对齐

```cpp
#include <cstdio>

int main() {
    // 设置宽度
    printf("%10d\n", 42);     // 右对齐，宽度10
    printf("%-10d\n", 42);    // 左对齐，宽度10

    // 浮点数
    printf("%10.2f\n", 3.14159);
    printf("%-10.2f\n", 3.14159);

    // 字符串
    printf("%15s\n", "Hello");
    printf("%-15s\n", "Hello");

    return 0;
}
```

### sprintf - 输出到字符串

```cpp
#include <cstdio>

    char buffer[100];
```

### snprintf - 安全的字符串格式化

```cpp

}
```

### 场景 1：成绩表

```cpp
#include <iostream>
#include <iomanip>
#include <vector>

struct Student {
    std::string name;
    int chinese;
    int math;
    int english;
};

void printReport(const std::vector<Student>& students) {
    std::cout << "\n===== 成绩表 =====\n";
    std::cout << std::left;
    std::cout << std::setw(10) << "姓名"
              << std::setw(8) << "语文"
              << std::setw(8) << "数学"
              << std::setw(8) << "英语"
              << std::setw(8) << "总分"
              << std::setw(8) << "平均分" << std::endl;
    std::cout << std::string(50, '-') << std::endl;

    for (const auto& s : students) {
        int total = s.chinese + s.math + s.english;
        double average = total / 3.0;

        std::cout << std::setw(10) << s.name
                  << std::setw(8) << s.chinese
                  << std::setw(8) << s.math
                  << std::setw(8) << s.english
                  << std::setw(8) << total
                  << std::fixed << std::setprecision(2)
                  << std::setw(8) << average << std::endl;
    }
}

int main() {
    std::vector<Student> students = {
        {"张三", 85, 92, 78},
        {"李四", 90, 88, 95},
        {"王五", 76, 85, 82}
    };

    printReport(students);

    return 0;
}
```

### 场景 2：商品清单

```cpp
#include <iostream>

struct Product {

void printInvoice(const std::vector<Product>& products) {

}
```

### 场景 3：进度条

```cpp
#include <iostream>
#include <iomanip>
#include <thread>
#include <chrono>

void printProgress(int progress, int total) {
    int barWidth = 50;
    float percentage = static_cast<float>(progress) / total;

    std::cout << "\r[";
    int pos = static_cast<int>(barWidth * percentage);
    for (int i = 0; i < barWidth; ++i) {
        if (i < pos)
            std::cout << "=";
        else if (i == pos)
            std::cout << ">";
        else
            std::cout << " ";
    }
    std::cout << "] " << std::fixed << std::setprecision(1)
              << (percentage * 100) << "%";
    std::cout.flush();
}

int main() {
    int total = 100;

    for (int i = 0; i <= total; ++i) {
        printProgress(i, total);
        std::this_thread::sleep_for(std::chrono::milliseconds(50));
    }

    std::cout << std::endl;
    std::cout << "完成!" << std::endl;

    return 0;
}
```

### 场景 4：时间格式化

## 总结

格式化输出是 C++ 编程中的重要技能，掌握它需要理解：

### 核心概念
1. **基本输出**：std::cout、std::cerr、std::clog
2. **格式化操作符**：setw、setprecision、setfill 等
3. **格式化标志**：left、right、fixed、scientific 等

### 实践要点
1. **对齐方式**：使用 left/right 控制对齐
2. **精度控制**：使用 setprecision 控制小数位数
3. **宽度设置**：使用 setw 控制输出宽度
4. **填充字符**：使用 setfill 设置填充字符

### 应用场景
1. **表格输出**：成绩表、数据表
2. **清单打印**：购物清单、商品清单
3. **进度显示**：进度条、百分比
4. **数据报告**：各种格式化的数据输出

### 最佳实践
1. **选择合适的方法**：简单场景用 cout，复杂格式用 printf
2. **注意效率**：频繁输出时考虑性能
3. **保持一致**：同一程序中保持格式一致
4. **用户友好**：输出要清晰、易读、美观

通过熟练使用格式化输出，你可以让程序的数据显示更加专业和美观。
