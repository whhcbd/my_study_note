# C++ 听课笔记 - 数据类型 (Data Types)

> 📌 **来源**：菜鸟教程 C++ 教程  
> 🎯 **目标**：掌握 C++ 内置类型、修饰符、派生类型与类型转换  
> 💡 **核心思想**：数据类型 = 内存空间的"使用说明书"

---

## 一、为什么需要数据类型？🤔

```cpp
/*
 * 变量 = 内存中的"小盒子"，用于存储数据
 * 数据类型 = 告诉编译器：
 *   ① 这个盒子多大？(占用多少字节)
 *   ② 里面能装什么？(整数/小数/字符/真值...)
 *   ③ 怎么解读里面的二进制？(编码方式)
 *
 * ✅ 好处：
 * • 类型安全：防止"用字符串做加法"这类错误
 * • 内存高效：按需分配，不浪费空间
 * • 代码清晰：读代码就知道变量用途
 */
```

## 二、基本内置类型 (Built-in Types) 🧱

### 2.1 七种基础类型速览

| 类型 | 关键字 | 用途 | 典型大小 | 取值范围/说明 |
|------|--------|------|---------|--------------|
| **布尔型** | `bool` | 真/假逻辑值 | 1 字节 | `true`(1) / `false`(0) |
| **字符型** | `char` | ASCII 字符/小整数 | 1 字节 | -128~127 或 0~255 |
| **整型** | `int` | 整数运算 | 4 字节 | ±21 亿左右 |
| **浮点型** | `float` | 单精度小数 | 4 字节 | ±3.4e±38，6-7 位有效数字 |
| **双浮点型** | `double` | 双精度小数 | 8 字节 | ±1.7e±308，15 位有效数字 |
| **无类型** | `void` | 表示"无值" | - | 用于函数无返回值、泛型指针 |
| **宽字符型** | `wchar_t` | Unicode 字符 | 2 或 4 字节 | 平台相关，用于国际化 |

```cpp
// 💡 重要提示：
// • 具体大小与编译器、操作系统、CPU 架构相关
// • 现代 64 位系统：int 通常 4 字节，long 通常 8 字节
// • 不确定时用 sizeof(类型) 查询
```

### 2.2 类型修饰符 (Modifiers) ✨

```cpp
/*
 * 修饰符用于调整基础类型的特性
 */

| 修饰符 | 作用 | 示例 | 说明 |
|--------|------|------|------|
| `signed` | 有符号（默认） | `signed int x = -10;` | 可表示正负数 |
| `unsigned` | 无符号 | `unsigned int y = 100;` | 只能 ≥0，范围翻倍 |
| `short` | 短整型 | `short z = 100;` | 通常 2 字节，范围小 |
| `long` | 长整型 | `long a = 100000L;` | 通常 4 或 8 字节 |
| `const` | 常量 | `const int b = 5;` | 值不可修改 |
| `volatile` | 易变变量 | `volatile int c;` | 禁止编译器优化，用于硬件寄存器 |
| `mutable` | 可变成员 | `mutable int counter;` | const 对象中也可修改 |

// 📋 组合使用示例：
unsigned long long bigNum = 18446744073709551615ULL;  // 最大无符号 64 位整数
const short MAX_SIZE = 100;  // 常量短整型
```

## 三、C++11 新增类型特性 🆕

### 3.1 自动类型推断

```cpp
// 🔹 auto：编译器自动推导类型
auto x = 10;           // x 是 int
auto y = 3.14;         // y 是 double
auto name = "Hello";   // name 是 const char*

// 🔹 decltype：获取表达式的类型
int a = 5;
decltype(a) b = 10;    // b 的类型与 a 相同，即 int

// 💡 适用场景：
// • 模板编程中类型复杂时
// • 避免重复书写长类型名
// • 返回值类型依赖于参数时
```

### 3.2 空指针与初始化

```cpp
// 🔹 nullptr：类型安全的空指针 (C++11)
int* ptr1 = NULL;      // 传统写法，NULL 本质是 0
int* ptr2 = nullptr;   // ✅ 推荐：明确表示指针为空

// 🔹 std::initializer_list：初始化列表
#include <initializer_list>
std::initializer_list<int> nums = {1, 2, 3, 4};

// 🔹 std::tuple：元组（存储不同类型的数据）
#include <tuple>
std::tuple<int, double, char> person(25, 175.5, 'M');
```

## 四、派生数据类型 (Derived Types) 🔗

### 4.1 类型分类速查

| 类型 | 描述 | 示例 |
|------|------|--------|
| **数组** | 相同类型元素的连续集合 | `int arr[5] = {1,2,3,4,5};` |
| **指针** | 存储变量内存地址 | `int* ptr = &x;` |
| **引用** | 变量的别名（必须初始化） | `int& ref = x;` |
| **函数** | 可执行的代码块 | `int add(int a, int b);` |
| **结构体** | 用户自定义复合类型 | `struct Point { int x, y; };` |
| **类** | 支持封装/继承/多态 | `class Student { ... };` |
| **联合体** | 成员共享同一内存 | `union Data { int i; float f; };` |
| **枚举** | 命名的整数常量集合 | `enum Color { RED, GREEN };` |

### 4.2 枚举类型详解 ⭐

```cpp
// 🔹 基本定义
enum Color { RED, GREEN, BLUE };  // 默认: RED=0, GREEN=1, BLUE=2
Color c = GREEN;

// 🔹 自定义初始值
enum Status { 
    OFF = 0, 
    ON = 100, 
    ERROR = -1  // 可以任意赋值
};

// 🔹 枚举变量使用
if (c == RED) {
    cout << "红色警告" << endl;
}

// 💡 现代 C++ 推荐：强类型枚举 (C++11)
enum class Direction { NORTH, SOUTH, EAST, WEST };
Direction d = Direction::NORTH;  // 必须加作用域，类型更安全
```

## 五、类型别名 (Type Aliases) 🏷️

### 5.1 两种定义方式

```cpp
// 🔹 传统方式：typedef
typedef unsigned long long ULL;
typedef std::vector<int> IntVec;

ULL bigNum = 123456789ULL;
IntVec numbers = {1, 2, 3};

// 🔹 现代方式：using (C++11，更清晰)
using ULL = unsigned long long;
using IntVec = std::vector<int>;
using StringMap = std::map<std::string, int>;

// 💡 优势：
// • 简化复杂类型书写
// • 提高代码可读性
// • 便于后期统一修改类型
```

## 六、标准库常用类型 📦

```cpp
#include <string>
#include <vector>
#include <array>
#include <map>
#include <set>

// 🔹 std::string：动态字符串（比 char[] 更安全方便）

std::set<int> uniqueNums = {3, 1, 2, 1};  // 实际存储 {1,2,3}
```

## 七、类型转换 (Type Casting) 🔄

### 7.1 C++ 四种安全转换（推荐）

| 转换类型 | 关键字 | 用途 | 安全性 |
|---------|--------|------|--------|
| **静态转换** | `static_cast` | 基本类型转换、向上转型 | ⚠️ 编译时检查，无运行时验证 |
| **动态转换** | `dynamic_cast` | 继承体系向下转型 | ✅ 运行时类型检查，失败返回 nullptr |
| **常量转换** | `const_cast` | 移除/添加 const 属性 | ⚠️ 仅改 const 属性，不改类型 |
| **重解释转换** | `reinterpret_cast` | 底层二进制重解释 | ❌ 极危险，慎用！ |

### 7.2 使用示例

```cpp
// 🔹 static_cast：基本类型转换
int i = 10;
double d = static_cast<double>(i);  // int → double

// 🔹 dynamic_cast：安全向下转型（需虚函数）
class Base { public: virtual ~Base() = default; };
class Derived : public Base { public: void show() { ... } };

Base* basePtr = new Derived;
Derived* derivedPtr = dynamic_cast<Derived*>(basePtr);
if (derivedPtr) {
    derivedPtr->show();  // ✅ 转换成功
}
delete basePtr;

// 🔹 const_cast：移除 const（谨慎使用！）
const int x = 10;
int& ref = const_cast<int&>(x);  // ⚠️ 修改 const 变量是未定义行为！
// ref = 20;  // ❌ 危险！不要这样做

// 🔹 reinterpret_cast：底层重解释（高级用法）
int* intPtr = new int(42);
char* charPtr = reinterpret_cast<char*>(intPtr);  // ⚠️ 直接操作内存，极易出错
```

### 7.3 C 风格转换（不推荐）

```cpp
// ❌ 避免使用 C 风格强制转换
double d = 3.14;
int i = (int)d;              // C 风格
int j = int(d);              // 函数风格

// ✅ 推荐：使用 C++ 风格转换
int k = static_cast<int>(d);  // 意图明确，便于搜索和维护
```

## 八、实用技巧：查询类型信息 🔍

### 8.1 使用 sizeof 和 numeric_limits

```cpp
#include <iostream>
#include <limits>  // 数值范围查询

using namespace std;

int main() {
    // 🔹 查询类型大小（字节）
    cout << "int 大小: " << sizeof(int) << " 字节" << endl;
    
    // 🔹 查询数值范围
    cout << "int 最小值: " << numeric_limits<int>::min() << endl;
    cout << "int 最大值: " << numeric_limits<int>::max() << endl;
    
    // 🔹 查询浮点数精度
    cout << "float 有效位数: " << numeric_limits<float>::digits10 << endl;
    
    return 0;
}
```

### 8.2 类型选择建议 🎯

```cpp
/*
 * ✅ 选择数据类型的原则：
 * 
 * 1️⃣ 整数：
 *    • 小范围计数 → short / int
 *    • 大范围/文件偏移 → long long
 *    • 只存非负数 → unsigned（但注意溢出行为）
 * 
 * 2️⃣ 小数：
 *    • 一般计算 → double（精度更高）
 *    • 内存敏感/大量数据 → float
 *    • 金融计算 → 考虑定点数或专用库（避免浮点误差）
 * 
 * 3️⃣ 字符：
 *    • ASCII 文本 → char
 *    • 国际化/Unicode → char16_t / char32_t / std::wstring
 * 
 * 4️⃣ 布尔：
 *    • 逻辑判断 → bool（比 int 更语义化）
 * 
 * 5️⃣ 指针/引用：
 *    • 需要修改原值 → 引用 &
 *    • 可能为空/动态内存 → 指针 *
 *    • 现代 C++ → 智能指针 unique_ptr/shared_ptr
 */
```
