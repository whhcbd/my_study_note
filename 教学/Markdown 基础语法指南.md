
Markdown 是一种轻量级标记语言，广泛用于文档编写、技术博客、README 文件等场景。以下是其常用语法及示例：

---

## 1. 标题

使用 `#` 表示标题级别，最多支持六级：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

---

## 2. 文本格式化

| 效果  | 语法                  | 示例          |
| --- | ------------------- | ----------- |
| 粗体  | `**文本**` 或 `__文本__` | **粗体文本**    |
| 斜体  | `*文本*` 或 `_文本_`     | *斜体文本*      |
| 粗斜体 | `***文本***`          | ***粗斜体文本*** |
| 删除线 | `~~文本~~`            | ~~删除线文本~~   |
| 高亮  | `==文本==`（部分支持）      | ==高亮文本==    |
| 下标  | `H~2~O`             | H~2~O       |
| 上标  | `x^2^`              | x^2^        |

---

## 3. 列表

### 无序列表

```markdown
- 项目一
- 项目二
  - 子项目
```

### 有序列表

```markdown
1. 第一项
2. 第二项
   3. 子项
```

### 任务列表

```markdown
- [ ] 未完成
- [x] 已完成
```

---

## 4. 链接与图片

| 类型   | 语法                                      | 示例                                               |
|--------|-------------------------------------------|----------------------------------------------------|
| 链接   | `[显示文本](URL)`                         | `[百度](https://www.baidu.com)`                   |
| 图片   | `![替代文本](图片URL)`                    | `![Logo](https://example.com/logo.png)`           |
| 带标题 | `[文本](URL "标题")`                      | `[链接](https://example.com "示例链接")`          |

---

## 5. 代码

### 行内代码

```markdown
使用 `code` 标记行内代码
```

### 代码块

````markdown
```python
def hello():
    print("Hello, World!")
```
````

支持指定语言以实现语法高亮，如 `python`、`javascript`、`bash` 等。

---

## 6. 引用

```markdown
> 这是一段引用文本
> 可以有多行
>> 嵌套引用
```

---

## 7. 表格

```markdown
| 姓名 | 年龄 | 城市     |
|------|------|----------|
| 张三 | 25   | 北京     |
| 李四 | 30   | 上海     |
```

对齐方式：

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容   | 内容     | 内容   |
```

---

## 8. 分隔线

以下任一方式均可创建水平线：

```markdown
---
***
___
```

---

## 9. 其他常用语法

### 转义字符

```markdown
\* 显示星号 \*
\# 显示井号 \#
```

### 自动链接

```markdown
<https://example.com>
<email@example.com>
```

### 脚注（部分平台支持）

```markdown
这是内容[^1]

[^1]: 这是脚注内容
```

### 数学公式（需支持 LaTeX）

```markdown
行内公式：$E = mc^2$

块级公式：
$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$
```

### HTML 支持（部分平台）

```markdown
<kbd>Ctrl</kbd> + <kbd>C</kbd>
<details>
<summary>点击展开</summary>
隐藏内容
</details>
```
