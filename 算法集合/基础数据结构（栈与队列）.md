## 一、抽象数据类型（ADT）

### 1.1 什么是ADT
- **定义**：定义了数据的逻辑操作，但不关心具体实现
- **好处**：
  - 分离接口与实现
  - 便于替换不同实现
  - 提高代码可维护性

### 1.2 常见ADT操作
| 操作类型 | 说明 | 示例 |
|---------|------|------|
| **构造函数** | 创建新的数据结构实例 | `stack = Stack()` |
| **判断函数** | 检查数据结构的状态 | `is_empty()` |
| **存取函数** | 修改数据结构内容 | `push()`, `pop()` |
| **访问函数** | 获取数据结构的信息 | `peek()`, `size()` |

## 二、栈

### 2.1 栈的定义
- **定义**：后进先出（LIFO, Last In First Out）的线性数据结构
- **形象理解**：像叠盘子，只能从顶部取放
- **特点**：
  - 只能在一端（栈顶）进行插入和删除
  - 最后插入的元素最先被取出

### 2.2 栈的基本操作
| 操作 | 功能 | 时间复杂度 |
|------|------|-----------|
| `push(x)` | 将元素x压入栈顶 | O(1) |
| `pop()` | 弹出并返回栈顶元素 | O(1) |
| `peek()` / `top()` | 返回栈顶元素但不弹出 | O(1) |
| `is_empty()` | 判断栈是否为空 | O(1) |
| `size()` | 返回栈中元素个数 | O(1) |

### 2.3 栈的代码实现（Python）
```python
class Stack:
    def __init__(self):
        self.items = []
    
    def is_empty(self):
        return len(self.items) == 0
    
    def push(self, item):
        self.items.append(item)
    
    def pop(self):
        if not self.is_empty():
            return self.items.pop()
        return None
    
    def peek(self):
        if not self.is_empty():
            return self.items[-1]
        return None
    
    def size(self):
        return len(self.items)

# 使用示例
stack = Stack()
stack.push(1)
stack.push(2)
stack.push(3)
print(stack.peek())  # 3
print(stack.pop())   # 3
print(stack.pop())   # 2
print(stack.is_empty())  # False
```

### 2.4 栈的应用场景

#### 2.4.2 括号匹配
```python
def is_valid_parentheses(s):
    stack = Stack()
    
    for char in s:
        if char == '(':
            stack.push(char)
        elif char == ')':
            if stack.is_empty():
                return False
            stack.pop()
    
    return stack.is_empty()

# 示例
print(is_valid_parentheses("(())"))  # True
print(is_valid_parentheses("(()"))  # False
print(is_valid_parentheses("())("))  # False
```

#### 2.4.3 表达式求值
```python
def evaluate_postfix(expression):
    stack = Stack()
    
    for token in expression.split():
        if token.isdigit():
            stack.push(int(token))
        else:
            b = stack.pop()
            a = stack.pop()
            if token == '+':
                stack.push(a + b)
            elif token == '-':
                stack.push(a - b)
            elif token == '*':
                stack.push(a * b)
            elif token == '/':
                stack.push(a / b)
    
    return stack.pop()

# 示例：后缀表达式 "3 4 + 5 *"
# 等价于 (3 + 4) * 5 = 35
print(evaluate_postfix("3 4 + 5 *"))  # 35
```

## 三、队列

### 3.1 队列的定义
- **定义**：先进先出（FIFO, First In First Out）的线性数据结构
- **形象理解**：像排队买票，先来的人先买
- **特点**：
  - 在一端（队尾）插入元素
  - 在另一端（队头）删除元素
  - 最先插入的元素最先被取出

### 3.2 队列的基本操作
| 操作 | 功能 | 时间复杂度 |
|------|------|-----------|
| `enqueue(x)` | 将元素x插入队尾 | O(1) |
| `dequeue()` | 删除并返回队头元素 | O(1) |
| `front()` | 返回队头元素但不删除 | O(1) |
| `is_empty()` | 判断队列是否为空 | O(1) |
| `size()` | 返回队列中元素个数 | O(1) |

### 3.3 队列的代码实现（Python）
```python
class Queue:
    def __init__(self):
        self.items = []
    
    def is_empty(self):
        return len(self.items) == 0
    
    def enqueue(self, item):
        self.items.append(item)
    
    def dequeue(self):
        if not self.is_empty():
            return self.items.pop(0)
        return None
    
    def front(self):
        if not self.is_empty():
            return self.items[0]
        return None
    
    def size(self):
        return len(self.items)

# 使用示例
queue = Queue()
queue.enqueue(1)
queue.enqueue(2)
queue.enqueue(3)
print(queue.front())  # 1
print(queue.dequeue())  # 1
print(queue.dequeue())  # 2
print(queue.is_empty())  # False
```

### 3.4 队列的应用场景

#### 3.4.1 任务调度
```python
import time

class TaskScheduler:
    def __init__(self):
        self.task_queue = Queue()
    
    def add_task(self, task):
        self.task_queue.enqueue(task)
        print(f"任务 {task} 已添加到队列")
    
    def process_tasks(self):
        while not self.task_queue.is_empty():
            task = self.task_queue.dequeue()
            print(f"正在执行任务: {task}")
            time.sleep(1)  # 模拟任务执行
            print(f"任务 {task} 完成")

# 使用示例
scheduler = TaskScheduler()
scheduler.add_task("任务A")
scheduler.add_task("任务B")
scheduler.add_task("任务C")
scheduler.process_tasks()
```

#### 3.4.2 广度优先搜索（BFS）
```python
from collections import deque

def bfs(graph, start, goal):
    queue = Queue()
    visited = set()
    
    queue.enqueue((start, [start]))  # (当前节点, 路径)
    visited.add(start)
    
    while not queue.is_empty():
        current, path = queue.dequeue()
        
        if current == goal:
            return path
        
        for neighbor in graph[current]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.enqueue((neighbor, path + [neighbor]))
    
    return None  # 未找到路径

# 示例图
graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E']
}

path = bfs(graph, 'A', 'F')
print(f"从A到F的路径: {path}")  # ['A', 'C', 'F']
```

#### 3.4.3 缓冲区管理
```python
class Buffer:
    def __init__(self, capacity):
        self.queue = Queue()
        self.capacity = capacity
    
    def add_data(self, data):
        if self.queue.size() < self.capacity:
            self.queue.enqueue(data)
            return True
        return False  # 缓冲区已满
    
    def get_data(self):
        if not self.queue.is_empty():
            return self.queue.dequeue()
        return None

# 使用示例
buffer = Buffer(capacity=3)
buffer.add_data("数据1")
buffer.add_data("数据2")
buffer.add_data("数据3")

print(buffer.get_data())  # 数据1
print(buffer.get_data())  # 数据2
print(buffer.add_data("数据4"))  # True
print(buffer.add_data("数据5"))  # False（缓冲区已满）
```

## 四、栈与队列的对比

### 4.1 特性对比表

| 特性 | 栈（LIFO） | 队列（FIFO） |
|------|------------|--------------|
| **插入/删除规则** | 同一端（栈顶） | 不同端（队尾插入，队头删除）|
| **访问顺序** | 后进先出 | 先进先出 |
| **典型应用** | 函数调用、括号匹配 | 任务调度、BFS、缓冲区 |
| **形象理解** | 叠盘子 | 排队买票 |

### 4.2 选择指南

**选择栈的情况**：
- 需要回溯到之前的状态（如递归、撤销操作）
- 处理嵌套结构（如括号、HTML标签）
- 表达式求值（后缀表达式）

**选择队列的情况**：
- 需要按顺序处理任务（如任务调度）
- 需要按层次遍历（如BFS）
- 缓冲数据（如消息队列）

## 五、实现优化

### 5.1 栈的实现优化

#### 使用列表实现
```python
class StackList:
    def __init__(self):
        self.items = []
    
    def push(self, item):
        self.items.append(item)
    
    def pop(self):
        return self.items.pop()
```

#### 使用链表实现
```python
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class StackLinkedList:
    def __init__(self):
        self.top = None
        self.size = 0
    
    def push(self, item):
        new_node = Node(item)
        new_node.next = self.top
        self.top = new_node
        self.size += 1
    
    def pop(self):
        if self.top is None:
            return None
        data = self.top.data
        self.top = self.top.next
        self.size -= 1
        return data
```

### 5.2 队列的实现优化

#### 使用双端队列（deque）
```python
from collections import deque

class QueueDeque:
    def __init__(self):
        self.items = deque()
    
    def enqueue(self, item):
        self.items.append(item)
    
    def dequeue(self):
        if self.items:
            return self.items.popleft()
        return None
```

#### 使用循环队列
```python
class CircularQueue:
    def __init__(self, capacity):
        self.capacity = capacity
        self.queue = [None] * capacity
        self.front = 0
        self.rear = -1
        self.size = 0
    
    def enqueue(self, item):
        if self.size == self.capacity:
            return False  # 队列已满
        
        self.rear = (self.rear + 1) % self.capacity
        self.queue[self.rear] = item
        self.size += 1
        return True
    
    def dequeue(self):
        if self.size == 0:
            return None  # 队列为空
        
        item = self.queue[self.front]
        self.front = (self.front + 1) % self.capacity
        self.size -= 1
        return item
```

## 八、常见错误

### 8.1 栈相关错误
- 栈溢出：不断push而不检查大小
- 空栈pop：没有检查is_empty就pop
- 错误理解栈的"顶"概念

### 8.2 队列相关错误
- 队列满时继续enqueue
- 空队列时dequeue
- 用列表的pop(0)效率低（应使用deque）

### 8.3 实现错误
- 混淆LIFO和FIFO的顺序
- 忘记处理边界情况（空/满）
- 时间复杂度计算错误
