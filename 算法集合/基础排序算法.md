## 一、排序算法基础

### 1.1 什么是排序
- **定义**：将一组数据按照特定规则（升序或降序）重新排列
- **常见类型**：
  - 内部排序：所有数据都在内存中
  - 外部排序：数据量太大，需要外部存储

### 1.2 评价标准
| 标准 | 说明 |
|------|------|
| **时间复杂度** | 算法执行时间随数据规模的增长 |
| **空间复杂度** | 算法需要的额外存储空间 |
| **稳定性** | 相等元素的相对位置是否保持不变 |

## 二、冒泡排序

### 2.1 算法思想
- 依次比较相邻元素，如果顺序错误就交换
- 每轮遍历后，最大元素"冒泡"到末尾
- 重复n-1轮，直到所有元素有序

### 2.2 算法步骤
```
for i = 0 to n-2:
    for j = 0 to n-2-i:
        if arr[j] > arr[j+1]:
            交换 arr[j] 和 arr[j+1]
```

### 2.3 代码实现（Python）
```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n-1):
        for j in range(n-1-i):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

# 示例
arr = [64, 34, 25, 12, 22, 11, 90]
print(bubble_sort(arr))  # [11, 12, 22, 25, 34, 64, 90]
```

### 2.4 算法分析
- **时间复杂度**：O(n²)
- **空间复杂度**：O(1)
- **稳定性**：稳定

### 2.5 优化版本
- 添加标志位，如果某轮没有发生交换，说明已经有序
- 可以提前结束排序

## 三、选择排序

### 3.1 算法思想
- 每轮从未排序部分找到最小元素
- 将最小元素放到已排序部分的末尾
- 重复n-1轮

### 3.2 算法步骤
```
for i = 0 to n-2:
    min_index = i
    for j = i+1 to n-1:
        if arr[j] < arr[min_index]:
            min_index = j
    交换 arr[i] 和 arr[min_index]
```

### 3.3 代码实现（Python）
```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n-1):
        min_index = i
        for j in range(i+1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

# 示例
arr = [64, 34, 25, 12, 22, 11, 90]
print(selection_sort(arr))  # [11, 12, 22, 25, 34, 64, 90]
```

### 3.4 算法分析
- **时间复杂度**：O(n²)
- **空间复杂度**：O(1)
- **稳定性**：不稳定（交换可能改变相等元素的相对位置）

## 四、插入排序

### 4.1 算法思想
- 将数组分为已排序和未排序两部分
- 每次从未排序部分取出一个元素
- 插入到已排序部分的合适位置
- 类似整理扑克牌

### 4.2 算法步骤
```
for i = 1 to n-1:
    key = arr[i]
    j = i - 1
    while j >= 0 and arr[j] > key:
        arr[j+1] = arr[j]
        j = j - 1
    arr[j+1] = key
```

### 4.3 代码实现（Python）
```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j+1] = arr[j]
            j -= 1
        arr[j+1] = key
    return arr

# 示例
arr = [64, 34, 25, 12, 22, 11, 90]
print(insertion_sort(arr))  # [11, 12, 22, 25, 34, 64, 90]
```

### 4.4 算法分析
- **时间复杂度**：O(n²)（最好情况O(n)，已排序时）
- **空间复杂度**：O(1)
- **稳定性**：稳定
- **适用场景**：小规模数据或基本有序的数据

## 五、快速排序

### 5.1 算法思想
- 选择一个基准元素（pivot）
- 将数组分为两部分：小于pivot的和大于pivot的
- 对两部分递归排序
- 分治法的典型应用

### 5.2 算法步骤
```
quick_sort(arr, left, right):
    if left < right:
        pivot_index = partition(arr, left, right)
        quick_sort(arr, left, pivot_index-1)
        quick_sort(arr, pivot_index+1, right)

partition(arr, left, right):
    pivot = arr[right]
    i = left - 1
    for j = left to right-1:
        if arr[j] <= pivot:
            i += 1
            交换 arr[i] 和 arr[j]
    交换 arr[i+1] 和 arr[right]
    return i + 1
```

### 5.3 代码实现（Python）
```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    
    return quick_sort(left) + middle + quick_sort(right)

# 示例
arr = [64, 34, 25, 12, 22, 11, 90]
print(quick_sort(arr))  # [11, 12, 22, 25, 34, 64, 90]
```

### 5.4 算法分析
- **时间复杂度**：
  - 平均：O(n log n)
  - 最坏：O(n²)（已排序且选择最差pivot）
- **空间复杂度**：O(log n)（递归栈）
- **稳定性**：不稳定
- **适用场景**：大规模数据，平均性能最好

## 六、算法对比总结

### 6.1 性能对比表

| 算法 | 时间复杂度（平均）| 时间复杂度（最坏）| 空间复杂度 | 稳定性 | 适用场景 |
|-------|------------------|------------------|-----------|-------|---------|
| **冒泡排序** | O(n²) | O(n²) | O(1) | 稳定 | 小规模数据，简单实现 |
| **选择排序** | O(n²) | O(n²) | O(1) | 不稳定 | 小规模数据 |
| **插入排序** | O(n²) | O(n²) | O(1) | 稳定 | 小规模或基本有序数据 |
| **快速排序** | O(n log n) | O(n²) | O(log n) | 不稳定 | 大规模数据 |

### 6.2 选择建议

**小规模数据（n < 100）**：
- 插入排序（若数据基本有序）
- 选择排序（简单实现）

**中等规模数据（100 < n < 1000）**：
- 插入排序
- 冒泡排序（学习用）

**大规模数据（n > 1000）**：
- 快速排序（首选）
- 归并排序（需要稳定性时）
