# 标记算法（桶计数）笔记

## 问题描述
给定 `n` 组数据，每组数据包含若干个整数，以 `-1` 作为结束标志。要求判断 1 到 30 这 30 个数字是否**全部至少出现一次**。  
- 如果全部出现，输出 `"yes"`，否则输出 `"no"`。  
- 输入的数字若不在 1~30 范围内则忽略。

## 算法思想
使用一个**标记数组（桶）**记录每个数字是否出现过。  
- 数组长度为 31（下标 0 不使用），初始全部为 0（或 `false`）。  
- 遍历所有输入，当读到 1~30 的数字时，将对应下标的值设为 1（或 `true`），表示该数字已出现。  
- 最后检查 1~30 的每个位置是否都被标记为 1，若全部标记则输出 `"yes"`，否则输出 `"no"`。

这种方法利用了数字范围有限（1~30）的特点，用数组直接映射，实现简单且高效。

## 算法步骤
1. **初始化**  
   - 定义大小为 31 的标记数组 `arr`，所有元素初始化为 0。  
   - 读取整数 `n`，表示有 `n` 组数据。

2. **处理输入**  
   - 外层循环 `for (int i = 0; i < n; i++)` 处理每组数据。  
   - 内层循环 `while (cin >> day && day != -1)` 持续读取当前组的整数，直到遇到 `-1`。  
   - 对于每个读入的数字 `day`，如果 `day` 在 1 到 30 之间（含），则将 `arr[day]` 置为 1。

3. **检查结果**  
   - 遍历 `d` 从 1 到 30：  
     - 如果 `arr[d] == 0`，说明数字 `d` 从未出现，将 `all_covered` 置为 `false` 并跳出循环。  
   - 根据 `all_covered` 的值输出 `"yes"` 或 `"no"`。

## 代码实现
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    int arr[32] = {0};          // 标记数组，下标 1~30 有效
    cin >> n;

    for (int i = 0; i < n; i++) {
        int day;
        while (cin >> day && day != -1) {
            if (day >= 1 && day <= 30) {
                arr[day] = 1;   // 标记该数字已出现
            }
        }
    }

    bool all_covered = true;
    for (int d = 1; d <= 30; d++) {
        if (arr[d] == 0) {
            all_covered = false;
            break;
        }
    }

    if (all_covered) {
        cout << "yes" << endl;
    } else {
        cout << "no" << endl;
    }

    return 0;
}
```

## 复杂度分析
- **时间复杂度**：  
  - 输入处理部分：总共读取的数字个数记为 `m`，每个数字处理时间为 O(1)，因此为 O(m)。  
  - 检查部分：遍历 30 次，常数时间 O(30)。  
  - 总体：O(m + 30) ≈ O(m)。
- **空间复杂度**：  
  - 固定大小的数组 `arr[32]`，空间复杂度 O(1)。

## 算法特点
- **适用范围**：数字范围已知且较小（如本题的 1~30），可用数组直接映射。
- **优点**：实现简单，时间效率高，空间开销小。
- **局限性**：如果数字范围很大（如 1~10^9），则无法用数组直接标记，需改用哈希表或排序等方法。

## 注意事项
- **数组下标**：为了使用下标直接对应数字，数组大小至少为 `max_value + 1`，此处 30，所以定义为 `arr[31]`（下标 0 未用）。
- **输入格式**：每组数据以 `-1` 结束，`-1` 本身不参与标记。程序通过 `while (cin >> day && day != -1)` 确保读到 `-1` 时停止当前组，并继续下一组。
- **边界检查**：只处理 1~30 的数字，其他数字（如 0、负数、大于 30 的数）直接忽略，不影响结果。
- **提前终止**：检查过程中一旦发现某个数字未出现，立即设置 `all_covered = false` 并 `break`，无需继续检查，提高效率。

## 扩展思考
- 若需要统计每个数字出现的次数，可将标记数组改为计数数组（`int cnt[31]`），每次遇到数字 `++cnt[day]`。
- 若数字范围不固定但不大，可使用 `vector<bool>` 动态分配。
- 若数字范围大或未知，可用 `unordered_set` 记录出现过的数字，最后判断集合大小是否等于预期总数。
