## 一、搜索算法基础

### 1.1 什么是搜索
- **定义**：在数据集合中查找特定元素的过程
- **输入**：数据集合、目标值
- **输出**：目标值的位置或不存在的信息

### 1.2 搜索算法分类
| 类型 | 特点 | 示例 |
|------|------|------|
| **线性搜索** | 逐个比较，无序数据 | 在未排序列表中查找 |
| **二分搜索** | 折半查找，有序数据 | 在排序列表中查找 |
| **哈希搜索** | 通过哈希函数直接定位 | 哈希表查找 |

## 二、线性搜索

### 2.1 算法思想
- 从数组的第一个元素开始
- 逐个与目标值比较
- 找到目标值则返回索引
- 遍历完整个数组都没找到则返回-1

### 2.2 算法步骤
```
for i = 0 to n-1:
    if arr[i] == target:
        return i
return -1  # 未找到
```

### 2.3 代码实现（Python）
```python
def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

# 示例
arr = [4, 2, 7, 1, 9, 3]
target = 7
index = linear_search(arr, target)
print(f"目标值 {target} 在索引 {index}")  # 目标值 7 在索引 2
```

### 2.4 算法分析
- **时间复杂度**：O(n)
  - 最好情况：O(1)（目标在第一个位置）
  - 最坏情况：O(n)（目标在最后一个位置或不存在）
- **空间复杂度**：O(1)
- **优点**：
  - 实现简单
  - 不要求数据有序
- **缺点**：
  - 大规模数据效率低

## 三、二分搜索

### 3.1 算法思想
- 要求数组必须有序（升序或降序）
- 每次比较中间元素与目标值
- 根据比较结果，确定目标在左半部分还是右半部分
- 重复此过程，直到找到目标或确定不存在

### 3.2 算法步骤
```
left = 0, right = n-1

while left <= right:
    mid = (left + right) // 2
    
    if arr[mid] == target:
        return mid
    elif arr[mid] < target:
        left = mid + 1  # 在右半部分
    else:
        right = mid - 1  # 在左半部分

return -1  # 未找到
```

### 3.3 代码实现（Python）
```python
def binary_search(arr, target):
    left = 0
    right = len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2
        
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return -1

# 示例
arr = [1, 2, 3, 4, 7, 9]  # 有序数组
target = 7
index = binary_search(arr, target)
print(f"目标值 {target} 在索引 {index}")  # 目标值 7 在索引 4
```

### 3.4 算法分析
- **时间复杂度**：O(log n)
  - 每次搜索范围减半
  - 搜索100万个元素最多需要20次比较
- **空间复杂度**：O(1)
- **优点**：
  - 大规模数据效率高
- **缺点**：
  - 要求数组有序
  - 插入和删除需要维护有序性

## 四、算法对比

### 4.1 性能对比表

| 算法 | 时间复杂度 | 空间复杂度 | 要求数据有序 | 适用场景 |
|-------|-----------|-----------|-------------|---------|
| **线性搜索** | O(n) | O(1) | 否 | 小规模、无序数据 |
| **二分搜索** | O(log n) | O(1) | 是 | 大规模、有序数据 |

### 4.2 选择建议

**小规模数据（n < 100）**：
- 线性搜索（简单实现）

**中等规模数据（100 < n < 10,000）**：
- 如果数据经常变动：线性搜索
- 如果数据基本不变且有序：二分搜索

**大规模数据（n > 10,000）**：
- 二分搜索（首选）
- 考虑使用更高级的数据结构（如哈希表、B树）

## 五、算法变体

### 5.1 递归版二分搜索
```python
def binary_search_recursive(arr, target, left, right):
    if left > right:
        return -1
    
    mid = (left + right) // 2
    
    if arr[mid] == target:
        return mid
    elif arr[mid] < target:
        return binary_search_recursive(arr, target, mid + 1, right)
    else:
        return binary_search_recursive(arr, target, left, mid - 1)

# 调用
arr = [1, 2, 3, 4, 7, 9]
index = binary_search_recursive(arr, 7, 0, len(arr) - 1)
```

### 5.2 返回插入位置
如果目标值不存在，返回应该插入的位置：
```python
def binary_search_insert_position(arr, target):
    left = 0
    right = len(arr)
    
    while left < right:
        mid = (left + right) // 2
        
        if arr[mid] < target:
            left = mid + 1
        else:
            right = mid
    
    return left  # 插入位置
```

## 六、实践应用

### 6.1 查找第一个等于目标的元素
```python
def binary_search_first(arr, target):
    result = -1
    left = 0
    right = len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2
        
        if arr[mid] == target:
            result = mid
            right = mid - 1  # 继续在左半部分查找
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return result
```

### 6.2 查找最后一个等于目标的元素
```python
def binary_search_last(arr, target):
    result = -1
    left = 0
    right = len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2
        
        if arr[mid] == target:
            result = mid
            left = mid + 1  # 继续在右半部分查找
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return result
```

### 8.3 常见错误
- 二分搜索忘记数组必须有序
- 计算mid时整数溢出（建议用 `left + (right - left) // 2`）
- 边界条件处理不当（left <= right vs left < right）
- 递归版本的基准情况错误
