# 二分查找求整数平方根（上取整）笔记

## 问题描述
给定一个非负整数 n（0 ≤ n ≤ 2^63），求最小的非负整数 q，使得  
q² ≥ n。  
等价于 q = ⌈√n⌉。

## 适用性分析
- **单调性**：函数 f(q) = q² 在 q ≥ 0 上严格递增。
- **有界性**：因为 (2^32)² = 2^64 > 2^63，且 n ≤ 2^63，所以答案 q 必在区间 [0, 2^32] 内。
- 因此可以在有序区间上使用二分查找找到第一个满足 q² ≥ n 的位置。

## 算法思想（循环不变式）
维护两个指针（或边界）l 和 r，使得：
- l 左侧的所有数平方后 < n（不满足条件）
- r 右侧的所有数平方后 ≥ n（满足条件）
- 初始时 l = 0，r = 2^32（开区间，r 本身是一个安全的上界）
- 每次取中点 mid = l + floor((r-l)/2)，计算 mid² 并与 n 比较：
  - 若 mid² ≥ n，说明 mid 可能是答案，但左边可能有更小的解，故将 r 缩小到 mid（保留候选）。
  - 若 mid² < n，说明 mid 太小，解在右边，故将 l 增大到 mid+1。
- 循环直到 l = r，此时 l 即为第一个满足条件的位置。

## 算法步骤
1. **初始化**  
   l = 0  
   r = 2^32 （用 `1ULL << 32` 表示）
2. **二分循环**（while (l < r)）
   - 计算中点：`mid = l + (r - l) / 2` （避免 l+r 溢出）
   - 计算平方：使用 __int128 类型存储 `(__int128)mid * mid` 防止溢出
   - 比较：
     - 若平方值 ≥ n，则 `r = mid`
     - 否则 `l = mid + 1`
3. **输出结果**  
   cout << l << endl

## 边界处理与细节
- **左闭右开区间**：采用 [l, r) 的写法，最终 l 和 r 相等，该位置就是答案。
- **中点计算**：`mid = l + (r - l) / 2` 避免直接 (l+r)/2 可能导致的整数溢出。
- **平方溢出**：mid 最大为 2^32，其平方 2^64 已超出 unsigned long long 的最大值（2^64-1），但 2^64 正好溢出为 0，可能引发错误。因此使用 128 位整数类型（如 GCC 的 __int128）安全计算。
- **上界选择**：取 r = 2^32 而非 2^32-1，保证平方严格大于任何可能的 n，且是 2 的幂，便于位运算。若取 2^32-1，其平方为 2^64 - 2^33 + 1，也大于 2^63，但代码中用 `1ULL << 32` 更简洁。

## 复杂度分析
- **时间复杂度**：循环次数为 ⌈log₂ 2^32⌉ = 32 次，常数时间。
- **空间复杂度**：仅使用几个变量，O(1)。

## 示例验证
输入：1223334445555  
二分查找过程（简略）：
- 初始区间 [0, 4294967296)
- 中点约 2147483648，平方远大于 n，r 缩小
- 不断二分，最终得到 q = 11060446  
验证：  
11060446² = 122333572597316 ≥ 1223334445555  
11060445² = 122333316638025 < 1223334445555 ✅

## 注意事项
- 此算法求解的是 **上取整** 的平方根（第一个满足 q² ≥ n 的整数）。若需要下取整（floor），可改为查找最后一个满足 q² ≤ n 的位置。
- 对于更大的数据范围，需相应调整上界和数据类型。
- 代码中使用了 __int128，这是 GCC 扩展，在其他编译器中可能需要改用其他大数方法（如 Python 的整数自动支持大数，但此题是 C++ 环境）。

## 完整 C++ 代码
```cpp
#include <iostream>
using namespace std;

int main() {
    unsigned long long n;
    cin >> n;
    unsigned long long l = 0, r = 1ULL << 32; // 2^32
    while (l < r) {
        unsigned long long mid = l + (r - l) / 2;
        __int128 square = (__int128)mid * mid;
        if (square >= n) {
            r = mid;
        } else {
            l = mid + 1;
        }
    }
    cout << l << endl;
    return 0;
}
```
