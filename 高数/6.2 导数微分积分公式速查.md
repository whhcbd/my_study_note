本文档收集了高等数学中最常用的导数、微分和积分公式，便于快速查阅。

### 2. 导数运算法则

#### 基本四则运算

- **和差法则**：$(u \pm v)' = u' \pm v'$
- **乘法法则**：$(uv)' = u'v + uv'$
- **除法法则**：$\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$

#### 复合函数（链式法则）

$$[f(g(x))]' = f'(g(x)) \cdot g'(x)$$

或写作：$y = f(u), u = g(x)$，则 $\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$

#### 反函数导数

$$\frac{dy}{dx} = \frac{1}{\frac{dx}{dy}}$$

#### 参数方程导数

设 $x = \varphi(t), y = \psi(t)$，则：
$$\frac{dy}{dx} = \frac{\psi'(t)}{\varphi'(t)}$$

#### 隐函数导数

方程 $F(x,y) = 0$ 确定的隐函数：
$$\frac{dy}{dx} = -\frac{F_x}{F_y}$$

## 二、基本微分公式

### 1. 微分与导数的关系

$$dy = f'(x)dx$$

### 2. 基本微分公式表

| 函数        | 微分                                          |
| ----------- | --------------------------------------------- |
| $C$（常数） | $dC = 0$                                      |
| $x^n$       | $d(x^n) = nx^{n-1}dx$                         |
| $a^x$       | $d(a^x) = a^x \ln a \, dx$                    |
| $e^x$       | $d(e^x) = e^x dx$                             |
| $\log_a x$  | $d(\log_a x) = \frac{1}{x \ln a} dx$          |
| $\ln x$     | $d(\ln x) = \frac{1}{x} dx$                   |
| $\sin x$    | $d(\sin x) = \cos x \, dx$                    |
| $\cos x$    | $d(\cos x) = -\sin x \, dx$                   |
| $\tan x$    | $d(\tan x) = \sec^2 x \, dx$                  |
| $\cot x$    | $d(\cot x) = -\csc^2 x \, dx$                 |
| $\sec x$    | $d(\sec x) = \sec x \tan x \, dx$             |
| $\csc x$    | $d(\csc x) = -\csc x \cot x \, dx$            |
| $\arcsin x$ | $d(\arcsin x) = \frac{1}{\sqrt{1-x^2}} dx$    |
| $\arccos x$ | $d(\arccos x) = -\frac{1}{\sqrt{1-x^2}} dx$   |
| $\arctan x$ | $d(\arctan x) = \frac{1}{1+x^2} dx$           |
| $\arcsec x$ | $d(\arcsec x) = \frac{1}{\lvert x \rvert \sqrt{x^2-1}} dx$ |
| $\arccsc x$ | $d(\arccsc x) = -\frac{1}{\lvert x \rvert \sqrt{x^2-1}} dx$ |
| $\sinh x$   | $d(\sinh x) = \cosh x \, dx$                  |
| $\cosh x$   | $d(\cosh x) = \sinh x \, dx$                  |
| $\tanh x$   | $d(\tanh x) = \operatorname{sech}^2 x \, dx$  |
| $\coth x$   | $d(\coth x) = -\operatorname{csch}^2 x \, dx$ |

### 3. 微分运算法则

- **和差法则**：$d(u \pm v) = du \pm dv$
- **乘法法则**：$d(uv) = u\,dv + v\,du$
- **除法法则**：$d\left(\frac{u}{v}\right) = \frac{v\,du - u\,dv}{v^2}$
- **复合函数**：$d[f(g(x))] = f'(g(x))g'(x)dx$

## 三、基本不定积分公式

### 1. 基本积分公式表

| 被积函数                       | 不定积分                                |
| ------------------------------ | --------------------------------------- |
| $0$                            | $C$                                     |
| $k$（常数）                    | $kx + C$                                |
| $x^n$                          | $\frac{x^{n+1}}{n+1} + C$ ($n \neq -1$) |
| $\frac{1}{x}$                  | $\ln\lvert x\rvert + C$                 |
| $a^x$                          | $\frac{a^x}{\ln a} + C$                 |
| $e^x$                          | $e^x + C$                               |
| $\sin x$                       | $-\cos x + C$                           |
| $\cos x$                       | $\sin x + C$                            |
| $\sec^2 x$                     | $\tan x + C$                            |
| $\csc^2 x$                     | $-\cot x + C$                           |
| $\sec x \tan x$                | $\sec x + C$                            |
| $\csc x \cot x$                | $-\csc x + C$                           |
| $\frac{1}{\sqrt{1-x^2}}$       | $\arcsin x + C$                         |
| $\frac{1}{1+x^2}$              | $\arctan x + C$                         |
| $\frac{1}{\lvert x \rvert \sqrt{x^2-1}}$         | $\arcsec x + C$ |
| $\frac{1}{1-x^2}$              | $\frac{1}{2}\ln\left\lvert\frac{1+x}{1-x}\right\rvert + C$  |
| $\frac{1}{a^2-x^2}$            | $\frac{1}{2a}\ln\left\lvert\frac{a+x}{a-x}\right\rvert + C$ |
| $\frac{1}{a^2+x^2}$            | $\frac{1}{a}\arctan\frac{x}{a} + C$     |
| $\frac{1}{\sqrt{a^2-x^2}}$     | $\arcsin\frac{x}{a} + C$                |
| $\frac{1}{\sqrt{x^2 \pm a^2}}$ | $\ln\lvert x + \sqrt{x^2 \pm a^2}\rvert + C$ |
| $\sinh x$                      | $\cosh x + C$                           |
| $\cosh x$                      | $\sinh x + C$                           |
| $\operatorname{sech}^2 x$      | $\tanh x + C$                           |
| $\operatorname{csch}^2 x$      | $-\coth x + C$                          |
| $\frac{1}{\sqrt{x^2-a^2}}$     | $\ln\lvert x + \sqrt{x^2-a^2}\rvert + C$ ($x > a$) |

### 2. 积分运算法则

- **线性性质**：$\int [f(x) \pm g(x)]dx = \int f(x)dx \pm \int g(x)dx$
- **常数提取**：$\int k f(x)dx = k\int f(x)dx$
- **凑微分法**：$\int f[g(x)]g'(x)dx = \int f(u)du$（其中 $u=g(x)$）
- **分部积分法**：$\int u\,dv = uv - \int v\,du$

## 五、常见积分技巧公式

### 1. 三角恒等式积分

$$\int \sin^2 x \, dx = \frac{x}{2} - \frac{\sin 2x}{4} + C$$

$$\int \cos^2 x \, dx = \frac{x}{2} + \frac{\sin 2x}{4} + C$$

$$\int \tan^2 x \, dx = \tan x - x + C$$

$$\int \sec^3 x \, dx = \frac{1}{2}\sec x \tan x + \frac{1}{2}\ln\lvert\sec x + \tan x\rvert + C$$

### 2. 降幂公式

$$\sin^2 x = \frac{1 - \cos 2x}{2}$$

$$\cos^2 x = \frac{1 + \cos 2x}{2}$$

$$\sin x \cos x = \frac{1}{2}\sin 2x$$

### 3. 万能代换

设 $t = \tan\frac{x}{2}$，则：
$$\sin x = \frac{2t}{1+t^2}, \quad \cos x = \frac{1-t^2}{1+t^2}, \quad dx = \frac{2}{1+t^2}dt$$

### 4. 指数与三角函数组合

$$\int e^{ax}\sin(bx)dx = \frac{e^{ax}}{a^2+b^2}(a\sin bx - b\cos bx) + C$$

$$\int e^{ax}\cos(bx)dx = \frac{e^{ax}}{a^2+b^2}(b\sin bx + a\cos bx) + C$$

## 六、重要极限与导数关系

### 1. 重要极限

$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$

$$\lim_{x \to 0} \frac{1 - \cos x}{x^2} = \frac{1}{2}$$

$$\lim_{x \to 0} \frac{e^x - 1}{x} = 1$$

$$\lim_{x \to 0} \frac{\ln(1+x)}{x} = 1$$

$$\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$$

### 2. 导数定义

$$f'(x) = \lim_{\Delta x \to 0} \frac{f(x + \Delta x) - f(x)}{\Delta x}$$

或：

$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

## 七、微积分基本定理

### 1. 牛顿-莱布尼茨公式

若 $F(x)$ 是 $f(x)$ 的原函数，则：

$$\int_{a}^{b} f(x)dx = F(b) - F(a)$$

### 2. 变限积分求导

$$\frac{d}{dx}\int_{a}^{x} f(t)dt = f(x)$$

$$\frac{d}{dx}\int_{x}^{b} f(t)dt = -f(x)$$

$$\frac{d}{dx}\int_{a(x)}^{b(x)} f(t)dt = f[b(x)]b'(x) - f[a(x)]a'(x)$$

## 八、常用微分方程

### 1. 可分离变量方程

$$\frac{dy}{dx} = f(x)g(y)$$

解：$\int \frac{dy}{g(y)} = \int f(x)dx$

### 2. 一阶线性微分方程

$$y' + P(x)y = Q(x)$$

通解：$y = e^{-\int P(x)dx}\left[\int Q(x)e^{\int P(x)dx}dx + C\right]$

### 3. 二阶常系数齐次方程

$$y'' + py' + qy = 0$$

特征方程：$r^2 + pr + q = 0$

根据判别式 $\Delta = p^2 - 4q$：

- $\Delta > 0$：$y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$（其中 $r_{1,2} = \frac{-p \pm \sqrt{p^2-4q}}{2}$）
- $\Delta = 0$：$y = (C_1 + C_2 x)e^{-\frac{p}{2}x}$（其中 $r = -\frac{p}{2}$）
- $\Delta < 0$：$y = e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x)$（其中 $\alpha = -\frac{p}{2}$，$\beta = \frac{\sqrt{4q-p^2}}{2}$）

## 九、参数方程与极坐标

### 1. 参数方程求导

设 $x = \varphi(t), y = \psi(t)$

一阶导数：$\frac{dy}{dx} = \frac{\psi'(t)}{\varphi'(t)}$

二阶导数：$\frac{d^2y}{dx^2} = \frac{d}{dt}\left(\frac{\psi'(t)}{\varphi'(t)}\right) \cdot \frac{1}{\varphi'(t)}$

### 2. 极坐标公式

弧长：$ds = \sqrt{r^2 + (r')^2}d\theta$

面积：$S = \frac{1}{2}\int_{\alpha}^{\beta} r^2(\theta)d\theta$

## 十、常用技巧口诀

### 1. 导数记忆口诀

- 常数导数为零，幂函数降一幂
- 指数导数不变，对数倒数形式
- 正弦变余弦，余弦负正弦
- 正切记为sec²，余切记为-csc²
- 反正切记为1/(1+x²)，反余切记为负

### 2. 积分记忆口诀

- 积分导数逆运算，幂函数升一幂除以新指数
- 指数函数不变除ln a，自然指数不用除
- 正弦积分负余弦，余弦积分正弦
- 分部积分：反对幂三指，反对幂三指

### 3. 微分记忆要点

- 微分=导数×dx
- 运算法则与导数类似
- 凑微分法的关键：找复合关系

## 十一、常见易错点

### 1. 导数计算错误

- ❌ $(uv)' = u'v'$（错误）
- ✅ $(uv)' = u'v + uv'$（正确）

- ❌ $\left(\frac{u}{v}\right)' = \frac{u'}{v'}$（错误）
- ✅ $\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$（正确）

### 2. 积分错误

- ❌ $\int \frac{1}{x}dx = \ln x + C$（错误）
- ✅ $\int \frac{1}{x}dx = \ln\lvert x\rvert + C$（正确）

- ❌ $\int \cos x dx = -\sin x + C$（错误）
- ✅ $\int \cos x dx = \sin x + C$（正确）

### 3. 链式法则遗漏

- ❌ $[\sin(2x)]' = \cos(2x)$（错误）
- ✅ $[\sin(2x)]' = 2\cos(2x)$（正确）
