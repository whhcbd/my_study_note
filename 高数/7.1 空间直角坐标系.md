## 一、空间直角坐标系 (Space Rectangular Coordinate System)

### 1.1 坐标系的建立
*   **二维 vs 三维**：
    *   二维 (高数上)：$x, y$ 轴，平面解析几何。
    *   三维 (高数下)：$x, y, z$ 轴，空间解析几何。
*   **右手系规则 (Right-Hand System)**：
    *   **方法一**：四指从 $x$ 轴正向沿 $90^\circ$ 转向 $y$ 轴正向，大拇指指向即为 $z$ 轴正向。
    *   **方法二**：右手掌心向上，拇指指 $x$，食指指 $y$，中指垂直弹出指 $z$。
    *   **结论**：$x, y, z$ 方向关系固定，做题按此顺序画轴。

### 1.2 点的坐标表示
*   **表示方法**：$M(x, y, z)$
    *   $x$：横坐标
    *   $y$：纵坐标
    *   $z$：**竖坐标** (常考概念)
*   **特殊位置点的坐标特征**：
    | 位置 | 坐标特征 | 示例 |
    | :--- | :--- | :--- |
    | **坐标轴上** | 仅该轴坐标非零，其余为 0 | $z$ 轴上点：$(0, 0, z)$ |
    | **坐标面上** | 垂直于该面的轴坐标为 0 | $xOz$ 面上点：$(x, 0, z)$即 $y=0$ |
    | **原点** | 全为 0 | $(0, 0, 0)$ |

### 1.3 卦限 (Octants)
*   **定义**：三个坐标平面将空间分为 **8 个部分**，称为卦限。
*   **顺序**：
    *   第 I 卦限：$x>0, y>0, z>0$ (全正)
    *   逆时针旋转排列，上方 4 个，下方 4 个。
    *   *注：实际做题很少考“第几卦限”，通常直接描述正负号。*

### 1.4 对称点 (Symmetry Points) ⭐考点
设点 $P(a, b, c)$，其对称点坐标规律如下：

| 对称对象 | 规律口诀 | 对称点坐标 |
| :--- | :--- | :--- |
| **关于 $xOy$ 面** | 谁垂直谁变号 ($z$ 变) | $(a, b, -c)$ |
| **关于 $z$ 轴** | 谁对称谁不变 ($z$ 不变)，其余变号 | $(-a, -b, c)$ |
| **关于原点** | 全部变号 | $(-a, -b, -c)$ |
| **关于 $x$ 轴** | $x$ 不变，其余变号 | $(a, -b, -c)$ |

### 1.5 两点间距离公式
*   **公式**：设 $M_1(x_1, y_1, z_1)$，$M_2(x_2, y_2, z_2)$
    $$ |M_1M_2| = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2} $$
*   **推导原理**：长方体体对角线长度 (多次勾股定理)。

## 二、向量基础概念 (Vector Basics)

### 2.1 向量的定义
*   **定义**：既有**大小**又有**方向**的量 (物理称矢量)。
*   **表示法**：
    *   几何：有向线段 $\vec{AB}$ (起点 $A$, 终点 $B$)。
    *   字母：手写加箭头 $\vec{a}$，印刷体加粗 $\mathbf{a}$ 或希腊字母 $\boldsymbol{\alpha}$。
    *   *注意：线性代数中手写通常不加箭头，高数中建议加箭头区分。*
*   **自由向量**：
    *   只要**大小相等、方向相同**，即为相等向量。
    *   可以平行移动，起点不限。

### 2.2 向量的模 (Modulus)
*   **定义**：向量的长度，记作 $|\vec{AB}|$ 或 $|\vec{a}|$ (双竖线)。
*   **特殊向量**：
    *   **单位向量**：模为 1 的向量。
    *   **零向量**：模为 0 的向量 ($\vec{0}$)，方向任意。

### 2.3 向量的夹角
*   **定义**：起点搬到一起，两向量间 $\in [0, \pi]$ 的角。
*   **特殊关系**：
    *   **平行 (共线)**：夹角为 $0$ (同向) 或 $\pi$ (反向)。
    *   **垂直 (正交)**：夹角为 $\frac{\pi}{2}$，记作 $\vec{a} \perp \vec{b}$。
    *   *零向量与任何向量既平行又垂直。*

## 三、向量的线性运算 (Linear Operations)

### 3.1 加法 (Addition)
*   **几何法则**：
    1.  **三角形法则**：首尾相接，起点连终点 ($\vec{AB} + \vec{BC} = \vec{AC}$)。
    2.  **平行四边形法则**：起点重合，画对角线。
*   **物理意义**：位移的合成 (家→姥姥家→奶奶家 = 家→奶奶家)。
*   **运算律**：交换律、结合律。

### 3.2 减法 (Subtraction)
*   **定义**：$\vec{a} - \vec{b} = \vec{a} + (-\vec{b})$。
*   **几何做法**：**起点同，终点连，指向被减数**。
    *   即 $\vec{AB} - \vec{AC} = \vec{CB}$ (指向 $B$)。

### 3.3 数乘 (Scalar Multiplication)
*   **定义**：实数 $\lambda$ 乘向量 $\vec{a}$，记作 $\lambda \vec{a}$。
*   **效果**：
    *   $\lambda > 0$：方向不变，长度伸缩。
    *   $\lambda < 0$：方向相反，长度伸缩。
    *   $\lambda = 0$：零向量。
*   **模长性质**：$|\lambda \vec{a}| = |\lambda| \cdot |\vec{a}|$ (数字取绝对值)。
*   **平行充要条件**：$\vec{b} \parallel \vec{a} \iff \vec{b} = \lambda \vec{a}$ ($\vec{a} \neq \vec{0}$)。

### 3.4 单位向量公式 ⭐必考
*   求与 $\vec{a}$ **同向**的单位向量 $\vec{e}$：
    $$ \vec{e} = \frac{\vec{a}}{|\vec{a}|} $$
*   **推导**：数乘不改变方向，除以模长使新模长为 1。

## 四、向量的坐标表示 (Coordinate Representation)

### 4.1 基底与分解
*   **单位基底**：$\vec{i}, \vec{j}, \vec{k}$ 分别表示 $x, y, z$ 轴正向的单位向量。
*   **分解式**：设 $\vec{OM}$ 终点为 $M(x, y, z)$
    $$ \vec{OM} = x\vec{i} + y\vec{j} + z\vec{k} $$
*   **坐标式**：$\vec{OM} = (x, y, z)$
    *   *注意：点的坐标 $(x,y,z)$ 与向量坐标 $\{x,y,z\}$ 形式相同，但含义不同。*

### 4.2 坐标运算公式
设 $\vec{a} = (a_x, a_y, a_z)$，$\vec{b} = (b_x, b_y, b_z)$

| 运算 | 坐标计算公式 |
| :--- | :--- |
| **加法** | $\vec{a} + \vec{b} = (a_x+b_x, \ a_y+b_y, \ a_z+b_z)$ |
| **减法** | $\vec{a} - \vec{b} = (a_x-b_x, \ a_y-b_y, \ a_z-b_z)$ |
| **数乘** | $\lambda \vec{a} = (\lambda a_x, \ \lambda a_y, \ \lambda a_z)$ |
| **平行** | $\frac{b_x}{a_x} = \frac{b_y}{a_y} = \frac{b_z}{a_z} = \lambda$ (对应分量成比例) |

### 4.3 向量坐标的求法
*   **已知两点 $A(x_1, y_1, z_1)$, $B(x_2, y_2, z_2)$**：
    $$ \vec{AB} = \vec{OB} - \vec{OA} = (x_2-x_1, \ y_2-y_1, \ z_2-z_1) $$
    *   **口诀**：**终点坐标 减 起点坐标**。

### 4.4 向量的模 (坐标版)
*   $$ |\vec{a}| = \sqrt{a_x^2 + a_y^2 + a_z^2} $$
*   $$ |\vec{AB}| = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2} $$

## 五、方向角与方向余弦 (Direction Angles & Cosines) ⭐重点

### 5.1 定义
*   **方向角**：向量 $\vec{a}$ 与三个坐标轴正向的夹角。
    *   与 $x$ 轴夹角：$\alpha$
    *   与 $y$ 轴夹角：$\beta$
    *   与 $z$ 轴夹角：$\gamma$
    *   范围：$\alpha, \beta, \gamma \in [0, \pi]$
*   **方向余弦**：三个方向角的余弦值。
    $$ \cos\alpha, \quad \cos\beta, \quad \cos\gamma $$

### 5.2 计算公式
设 $\vec{a} = (a_x, a_y, a_z)$，模为 $|\vec{a}|$
$$ \cos\alpha = \frac{a_x}{|\vec{a}|}, \quad \cos\beta = \frac{a_y}{|\vec{a}|}, \quad \cos\gamma = \frac{a_z}{|\vec{a}|} $$
*   **理解**：某轴方向余弦 = 该轴分量 / 向量模长。

### 5.3 重要性质
1.  **平方和为 1**：
    $$ \cos^2\alpha + \cos^2\beta + \cos^2\gamma = 1 $$
2.  **与单位向量的关系**：
    方向余弦构成的向量即为**同向单位向量**：
    $$ \vec{e} = (\cos\alpha, \cos\beta, \cos\gamma) = \frac{\vec{a}}{|\vec{a}|} $$
3.  **符号意义**：
    *   余弦为正：锐角 (与轴正向大致同向)。
    *   余弦为负：钝角 (与轴正向大致反向)。

## 六、典型例题思路总结

### 6.1 平行四边形向量表示
*   **场景**：已知邻边向量 $\vec{AB}, \vec{AD}$，求对角线或中点向量。
*   **技巧**：
    *   对角线 $\vec{AC} = \vec{AB} + \vec{AD}$ (平行四边形法则)。
    *   中点向量 = $\frac{1}{2}$ 对角线向量。
    *   反向向量 = 加负号。

### 6.2 方向余弦实战步骤
1.  **求向量坐标**：终点减起点。
2.  **求模长**：平方和开根号。
3.  **求方向余弦**：分量除以模长。
4.  **求方向角**：反余弦 $\arccos$ (注意范围 $0 \sim \pi$)。

## 七、核心公式速查表 ✅

| 概念        | 公式/结论                                                                                                                  |
| :-------- | :--------------------------------------------------------------------------------------------------------------------- |
| **两点间距离** | $d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2}$                                                                   |
| **向量模长**  | $\|\vec{a}\| = \sqrt{a_x^2 + a_y^2 + a_z^2}$                                                                           |
| **单位向量**  | $\vec{e} = \dfrac{\vec{a}}{\|\vec{a}\|}$ （与 $\vec{a}$ 同向）                                                              |
| **向量平行**  | $\vec{b} \parallel \vec{a} \iff \dfrac{b_x}{a_x} = \dfrac{b_y}{a_y} = \dfrac{b_z}{a_z}$                                |
| **方向余弦**  | $\cos\alpha = \dfrac{a_x}{\|\vec{a}\|},\ \cos\beta = \dfrac{a_y}{\|\vec{a}\|},\ \cos\gamma = \dfrac{a_z}{\|\vec{a}\|}$ |
| **余弦平方和** | $\cos^2\alpha + \cos^2\beta + \cos^2\gamma = 1$                                                                        |
| **对称点规律** | 关于谁对称，谁不变；关于原点对称，全变号                                                                                                   |
