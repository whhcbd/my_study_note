# 📐 空间平面方程（第三关）听课笔记

## 🎯 本关目标
掌握平面的四种常用方程（点法式、一般式、三点式、截距式），学会处理平面间的关系（夹角、平行、垂直）以及点到平面的距离。

---

## 一、平面的四种方程

### 1.1 点法式方程 —— 核心基础
- **确定平面的关键**：**一个点 + 一个法向量**
  - **法向量 `N`**：与平面内所有向量都垂直的非零向量。方向只有两个（互为反向），长度任意。
  - **点 `M₀(x₀, y₀, z₀)`**：确定平面的具体位置。
- **方程推导**：
  - 在平面上任取一点 `M(x, y, z)`，则向量 `M₀M = (x-x₀, y-y₀, z-z₀)`。
  - `M₀M` 与法向量 `N = (A, B, C)` 垂直 ⇒ **内积为 0**。
  - **点法式方程**：
    $$A(x - x₀) + B(y - y₀) + C(z - z₀) = 0$$

- **例题 1**：已知平面过点 `(2, -3, 0)`，法向量 `(1, -2, 3)`，求方程。
  - 解：`1·(x-2) + (-2)·(y+3) + 3·(z-0) = 0` ⇒ `x - 2y + 3z - 8 = 0`

### 1.2 一般式方程
- **形式**：$$Ax + By + Cz + D = 0$$
- **与点法式的关系**：将点法式化简合并常数项即得一般式。
- **关键信息**：
  - **法向量 `N = (A, B, C)`**（直接从系数读出）
  - **求平面上一点**：令两个变量为0，解第三个。例如令 `x=0, y=0`，则 `z = -D/C`（若 `C≠0`）。

#### 特殊情况分析（系数为0时的平面位置）

| 方程特征 | 几何含义 | 示例 |
| :--- | :--- | :--- |
| **D = 0** | 平面过原点 `(0,0,0)` | `x + 2y - 3z = 0` |
| **缺一个变量**（某系数为0） | 平面**平行于**缺失的那个坐标轴 | `x + y = 1` 平行于 z 轴 |
| **缺一个变量且 D=0** | 平面**包含**缺失的那个坐标轴 | `x + y = 0` 包含 z 轴 |
| **缺两个变量**（如 `z = 3`） | 平面**平行于**对应的坐标面（此处为 xOy 面） | `z = 3` 平行于 xOy 面 |
| **缺两个变量且 D=0**（如 `z=0`） | 就是坐标面本身 | `z = 0` 即 xOy 面 |

- **例题 2**：求包含 x 轴和点 `(4, -3, -1)` 的平面方程。
  - **法一（利用结论）**：
    - 包含 x 轴 ⇒ 平行于 x 轴且过原点 ⇒ 方程缺 x 项 (`A=0`) 且 `D=0`。
    - 设方程：`By + Cz = 0`。代入点 `(4, -3, -1)`：`-3B - C = 0` ⇒ `C = -3B`。
    - 取 `B=1`，得 `y - 3z = 0`。（`B` 不能为0，否则方程无意义）
  - **法二（设一般式）**：
    - 设 `Ax + By + Cz + D = 0`。
    - 过原点 ⇒ `D=0`。
    - 包含 x 轴 ⇒ x 轴上点 `(1,0,0)` 满足方程 ⇒ `A·1 = 0` ⇒ `A=0`。
    - 代入点 `(4, -3, -1)`：`-3B - C = 0` ⇒ `C = -3B`。带回得 `By - 3Bz = 0` ⇒ `y - 3z = 0`。

### 1.3 三点式方程
- **原理**：不共线的三点唯一确定一个平面。
- **推导思路**：设平面上任一点 `M(x,y,z)`，与已知三点 `M₁, M₂, M₃` 构成三个向量 `M₁M, M₁M₂, M₁M₃`。它们共面 ⇒ **混合积为 0**。
- **方程（行列式形式）**：
  $$ \begin{vmatrix} x - x_1 & y - y_1 & z - z_1 \\ x_2 - x_1 & y_2 - y_1 & z_2 - z_1 \\ x_3 - x_1 & y_3 - y_1 & z_3 - z_1 \end{vmatrix} = 0 $$

- **例题 4**：已知三点 `M₁(2,-1,4)`, `M₂(-1,3,-2)`, `M₃(0,2,3)`，求平面方程。
  - **法一（点法式 + 叉积求法向量）**：
    1. 求两个向量：`M₁M₂ = (-3, 4, -6)`，`M₁M₃ = (-2, 3, -1)`
    2. 求法向量 `N = M₁M₂ × M₁M₃`：
       - 用口诀：写两遍 `(-3,4,-6, -3,4)` 和 `(-2,3,-1, -2,3)`
       - 第一分量：`|4 -6; 3 -1| = 4*(-1) - (-6)*3 = -4 + 18 = 14`
       - 第二分量：`|-6 -3; -1 -2| = (-6)*(-2) - (-3)*(-1) = 12 - 3 = 9` （注意顺序）
         - 更保险：按公式第二分量 = `-( | -3 -6; -2 -1 | )?` 其实用口诀直接：看 `a₃ a₁` 和 `b₃ b₁`：`(-6, -3)` 和 `(-1, -2)` ⇒ `(-6)*(-2) - (-3)*(-1) = 12 - 3 = 9` ✅
       - 第三分量：`|-3 4; -2 3| = (-3)*3 - 4*(-2) = -9 + 8 = -1`
       - 得 `N = (14, 9, -1)` （可乘任意非零倍数）
    3. 用点 `M₁` 和 `N` 写点法式：
       $$14(x-2) + 9(y+1) + (-1)(z-4) = 0$$ ⇒ $$14x + 9y - z - 15 = 0$$
  - **法二（直接三点式行列式）**：
    $$ \begin{vmatrix} x-2 & y+1 & z-4 \\ -3 & 4 & -6 \\ -2 & 3 & -1 \end{vmatrix} = 0 $$
    展开计算（结果相同）。

### 1.4 截距式方程
- **形式**：$$\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$$，其中 `a, b, c ≠ 0`。
- **几何意义**：平面在 x, y, z 轴上的截距分别为 `a, b, c`。
- **来源**：由一般式 `Ax+By+Cz+D=0` 变形得来，要求 `D≠0`。

## 二、平面间的关系

### 2.1 两平面的夹角
- **定义**：两平面法向量之间的**锐角**（或直角）。
- **公式**：
  $$\cos \theta = \frac{|\vec{n_1} \cdot \vec{n_2}|}{|\vec{n_1}| |\vec{n_2}|}$$
  - 分子加绝对值，保证 `θ ∈ [0, π/2]`。
- **例题 5**：求平面 `x - y + 2z - 6 = 0` 与 `2x + y + z = 5` 的夹角。
  - 法向量：`n₁=(1,-1,2)`，`n₂=(2,1,1)`
  - `|n₁·n₂| = |1*2 + (-1)*1 + 2*1| = |2 -1 +2| = 3`
  - `|n₁| = √(1+1+4)=√6`，`|n₂| = √(4+1+1)=√6`
  - `cosθ = 3/(√6*√6) = 3/6 = 1/2` ⇒ `θ = π/3`

### 2.2 特殊位置关系
- **垂直** ⇔ 法向量垂直 ⇔ `n₁·n₂ = 0`
- **平行** ⇔ 法向量平行 ⇔ `n₁ = λ n₂`（对应分量成比例）
  - **进一步区分**：
    - **平行但不重合**：两平面无公共点。验证：取一个平面上一点，代入另一个平面方程，**不满足**。
    - **重合**：有公共点。验证：取一个平面上一点，代入另一个平面方程，**满足**。

### 2.3 综合例题 6
> **题目**：求过点 `M₁(1,1,1)` 和 `M₂(0,1,-1)` 且垂直于平面 `x + y + z = 0` 的平面方程。

- **分析**：求平面方程 ⇒ 需要 **一个点**（已有）和 **法向量 `N`**。
  - `N` 必须满足：
    1. 垂直于已知平面 ⇒ `N` 垂直于已知平面的法向量 `n₁ = (1,1,1)`
    2. 垂直于平面内的向量 `M₁M₂` ⇒ `N` 垂直于 `M₁M₂ = (-1, 0, -2)`
- **解法**：`N` 同时垂直于两个已知向量 ⇒ `N = M₁M₂ × n₁`
  - 计算叉积：
    $$ \begin{vmatrix} i & j & k \\ -1 & 0 & -2 \\ 1 & 1 & 1 \end{vmatrix} $$
    - 第一分量：`0*1 - (-2)*1 = 0 + 2 = 2`
    - 第二分量：`-(-1*1 - (-2)*1) = -(-1 + 2) = -1` （或直接算：`| -2 -1; 1 1| = -2*1 - (-1)*1 = -2+1=-1`）
    - 第三分量：`-1*1 - 0*1 = -1`
    - 得 `N = (2, -1, -1)`（或任何倍数）
- **写方程**（用点 `M₁(1,1,1)`）：
  $$2(x-1) + (-1)(y-1) + (-1)(z-1) = 0$$ ⇒ $$2x - y - z = 0$$

## 三、点到平面的距离公式

- **公式**：点 `P₀(x₀, y₀, z₀)` 到平面 `Ax + By + Cz + D = 0` 的距离为
  $$ d = \frac{|A x₀ + B y₀ + C z₀ + D|}{\sqrt{A^2 + B^2 + C^2}} $$

- **记忆**：**“带到左边取绝对值，除以法向量模长”**

- **例题 7**：求点 `(2,1,1)` 到平面 `x + y - z + 1 = 0` 的距离。
  - 分子：`|2 + 1 - 1 + 1| = |3| = 3`
  - 分母：`√(1²+1²+(-1)²) = √3`
  - 距离：`d = 3/√3 = √3`

- **推导思路**（理解即可）：
  1. 在平面上任取一点 `P₁(x₁,y₁,z₁)`。
  2. 距离 `d` 等于向量 `P₁P₀` 在法向量 `n=(A,B,C)` 上投影的绝对值。
  3. 投影公式：`d = |(P₁P₀ · n)| / |n|`。
  4. 计算 `P₁P₀ · n = A(x₀-x₁) + B(y₀-y₁) + C(z₀-z₁) = (Ax₀+By₀+Cz₀) - (Ax₁+By₁+Cz₁)`。
  5. 因为 `P₁` 在平面上，所以 `Ax₁+By₁+Cz₁ = -D`，代入得分子 `|Ax₀+By₀+Cz₀ + D|`。

## 📝 本讲总结

| 方程类型 | 形式 | 已知条件 | 关键点 |
| :--- | :--- | :--- | :--- |
| **点法式** | `A(x-x₀)+B(y-y₀)+C(z-z₀)=0` | 一点 `M₀` + 法向量 `n=(A,B,C)` | 核心基础，求平面首选 |
| **一般式** | `Ax+By+Cz+D=0` | 可直接读出法向量 `(A,B,C)` | 处理特殊情况（缺项）很方便 |
| **三点式** | 三阶行列式为0 | 不共线三点 | 直接用行列式，避免求法向量 |
| **截距式** | `x/a + y/b + z/c = 1` | 三轴截距 `a, b, c` | 直观，便于画图 |

- **求平面方程的通用思维**：**“一点 + 一法”**。点常直接给出，法向量常通过以下方式获得：
  - 与平面内两个不共线向量垂直 ⇒ 叉积。
  - 与另一平面垂直 ⇒ 法向量与已知法向量垂直。
- **两平面夹角**：转化为法向量夹角（取锐角）。
- **平行与重合**：先看法向量是否平行，再验证一个点是否在另一平面上。
- **距离公式**：必须熟练掌握，后续多元函数微积分常用。
