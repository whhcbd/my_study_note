# 8.4 多元函数的极值与最值

> 📌 **本节核心**：掌握无条件极值、条件极值（拉格朗日乘数法）、最值问题的求解方法

---

## 一、无条件极值

### 1. 极值的定义

设函数 $z = f(x, y)$ 在点 $(x_0, y_0)$ 的某邻域内有定义：

- **极大值**：若 $f(x, y) \leq f(x_0, y_0)$，则 $f(x_0, y_0)$ 为极大值
- **极小值**：若 $f(x, y) \geq f(x_0, y_0)$，则 $f(x_0, y_0)$ 为极小值

### 2. 极值存在的必要条件

📐 **定理**：若 $f(x, y)$ 在 $(x_0, y_0)$ 处可偏导且取得极值，则：

$$f_x(x_0, y_0) = 0, \quad f_y(x_0, y_0) = 0$$

满足此条件的点称为**驻点**（或稳定点）。

⚠️ **注意**：驻点不一定是极值点！

### 3. 极值存在的充分条件

设 $(x_0, y_0)$ 是驻点，记：

$$A = f_{xx}(x_0, y_0), \quad B = f_{xy}(x_0, y_0), \quad C = f_{yy}(x_0, y_0)$$

计算判别式：

$$\Delta = AC - B^2$$

🎯 **判别口诀**：

| 判别式 $\Delta$ | 条件 | 结论 |
|---|---|---|
| $\Delta > 0$ | $A > 0$ | **极小值** |
| $\Delta > 0$ | $A < 0$ | **极大值** |
| $\Delta < 0$ | — | **不是极值**（鞍点） |
| $\Delta = 0$ | — | **无法判定** |

💡 **记忆技巧**：
- $\Delta > 0$ 且 $A > 0$：开口向上的碗 → 极小值
- $\Delta > 0$ 且 $A < 0$：开口向下的碗 → 极大值
- $\Delta < 0$：马鞍形 → 鞍点

## 二、条件极值与拉格朗日乘数法

### 1. 条件极值问题

在约束条件 $\varphi(x, y) = 0$ 下，求 $f(x, y)$ 的极值。

### 2. 拉格朗日乘数法

📌 **核心步骤**：

**Step 1**：构造拉格朗日函数

$$L(x, y, \lambda) = f(x, y) + \lambda \varphi(x, y)$$

**Step 2**：求偏导数并令其为零

$$\begin{cases}
L_x = f_x + \lambda \varphi_x = 0 \\
L_y = f_y + \lambda \varphi_y = 0 \\
L_\lambda = \varphi(x, y) = 0
\end{cases}$$

**Step 3**：解方程组得驻点 $(x_0, y_0)$

**Step 4**：比较各驻点的函数值，确定极值

### 3. 三元函数的条件极值

在约束条件 $\varphi(x, y, z) = 0$ 下，求 $f(x, y, z)$ 的极值：

$$L(x, y, z, \lambda) = f(x, y, z) + \lambda \varphi(x, y, z)$$

解方程组：

$$\begin{cases}
L_x = f_x + \lambda \varphi_x = 0 \\
L_y = f_y + \lambda \varphi_y = 0 \\
L_z = f_z + \lambda \varphi_z = 0 \\
L_\lambda = \varphi(x, y, z) = 0
\end{cases}$$

## 三、最值问题

### 1. 有界闭区域上的最值

在有界闭区域 $D$ 上求 $f(x, y)$ 的最值：

📝 **求解步骤**：

1. 求区域 $D$ 内部的所有驻点
2. 求边界上的最值（化为一元函数或条件极值）
3. 比较所有候选点的函数值，确定最大值和最小值

### 2. 实际应用问题

常见类型：
- 几何问题：体积最大、距离最短
- 经济问题：成本最低、利润最大

## 四、例题实战

### 📝 例1：求无条件极值

求 $f(x, y) = x^3 + y^3 - 3xy$ 的极值。

**解**：

**Step 1**：求驻点

$$f_x = 3x^2 - 3y = 0 \Rightarrow y = x^2$$
$$f_y = 3y^2 - 3x = 0 \Rightarrow x = y^2$$

联立得：$x = x^4 \Rightarrow x(x^3 - 1) = 0$

解得：$(0, 0)$ 和 $(1, 1)$

**Step 2**：计算二阶偏导数

$$A = f_{xx} = 6x, \quad B = f_{xy} = -3, \quad C = f_{yy} = 6y$$

**Step 3**：判别

- 点 $(0, 0)$：$A = 0, B = -3, C = 0$，$\Delta = 0 \cdot 0 - 9 = -9 < 0$ → **鞍点**
- 点 $(1, 1)$：$A = 6, B = -3, C = 6$，$\Delta = 36 - 9 = 27 > 0$，$A > 0$ → **极小值**

$$f(1, 1) = 1 + 1 - 3 = -1$$

✅ **答案**：极小值为 $-1$，在点 $(1, 1)$ 处取得。

---

### 📝 例2：拉格朗日乘数法

求椭球面 $x^2 + 2y^2 + 3z^2 = 1$ 上的点到原点的最大距离和最小距离。

**解**：

问题转化为：在约束 $x^2 + 2y^2 + 3z^2 = 1$ 下，求 $f(x, y, z) = x^2 + y^2 + z^2$ 的极值。

**Step 1**：构造拉格朗日函数

$$L = x^2 + y^2 + z^2 + \lambda(x^2 + 2y^2 + 3z^2 - 1)$$

**Step 2**：求偏导

$$\begin{cases}
L_x = 2x + 2\lambda x = 0 \Rightarrow x(1 + \lambda) = 0 \\
L_y = 2y + 4\lambda y = 0 \Rightarrow y(1 + 2\lambda) = 0 \\
L_z = 2z + 6\lambda z = 0 \Rightarrow z(1 + 3\lambda) = 0 \\
x^2 + 2y^2 + 3z^2 = 1
\end{cases}$$

**Step 3**：分类讨论

- 若 $x \neq 0$，则 $\lambda = -1$，代入得 $y = z = 0$，$x = \pm 1$，$f = 1$
- 若 $y \neq 0$，则 $\lambda = -\frac{1}{2}$，代入得 $x = z = 0$，$y = \pm \frac{\sqrt{2}}{2}$，$f = \frac{1}{2}$
- 若 $z \neq 0$，则 $\lambda = -\frac{1}{3}$，代入得 $x = y = 0$，$z = \pm \frac{\sqrt{3}}{3}$，$f = \frac{1}{3}$

✅ **答案**：最大距离为 $1$，最小距离为 $\frac{\sqrt{3}}{3}$。

### 📝 例3：实际应用

某工厂生产两种产品，成本函数为 $C(x, y) = 2x^2 + y^2 - xy + 100$，总产量限制为 $x + y = 10$。求使成本最低的生产方案。

**解**：

在约束 $x + y - 10 = 0$ 下，求 $C(x, y)$ 的最小值。

**Step 1**：构造拉格朗日函数

$$L = 2x^2 + y^2 - xy + 100 + \lambda(x + y - 10)$$

**Step 2**：求偏导

$$\begin{cases}
L_x = 4x - y + \lambda = 0 \\
L_y = 2y - x + \lambda = 0 \\
x + y = 10
\end{cases}$$

**Step 3**：解方程组

由前两式相减：$4x - y - 2y + x = 0 \Rightarrow 5x = 3y$

代入 $x + y = 10$：$x = \frac{15}{4}, y = \frac{25}{4}$

$$C\left(\frac{15}{4}, \frac{25}{4}\right) = 2 \cdot \frac{225}{16} + \frac{625}{16} - \frac{375}{16} + 100 = \frac{925}{16} + 100 \approx 157.8$$

✅ **答案**：生产 $x = 3.75$ 单位产品1，$y = 6.25$ 单位产品2，成本最低。

## 五、核心公式速查表

| 类型 | 公式/方法 |
|---|---|
| **驻点条件** | $f_x = 0, f_y = 0$ |
| **判别式** | $\Delta = f_{xx} f_{yy} - (f_{xy})^2$ |
| **极值判别** | $\Delta > 0, A > 0$ → 极小值；$\Delta > 0, A < 0$ → 极大值；$\Delta < 0$ → 鞍点 |
| **拉格朗日函数** | $L(x, y, \lambda) = f(x, y) + \lambda \varphi(x, y)$ |
| **条件极值方程** | $L_x = 0, L_y = 0, L_\lambda = 0$ |

---

## 六、常见题型总结

| 题型 | 方法 | 注意事项 |
|---|---|---|
| 无条件极值 | 求驻点 + 判别式 | 驻点不一定是极值点 |
| 条件极值 | 拉格朗日乘数法 | 注意分类讨论 |
| 闭区域最值 | 内部驻点 + 边界最值 | 边界可能需要参数化 |
| 实际应用 | 建立目标函数和约束条件 | 注意实际意义（如 $x, y > 0$） |
