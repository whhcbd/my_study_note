# 11.3 傅里叶级数

> 📌 **本节核心**：理解傅里叶级数的概念，掌握傅里叶系数的计算，学会将函数展开成傅里叶级数

---

## 一、三角级数与三角函数系

### 1. 三角级数

形如

$$\frac{a_0}{2} + \sum_{n=1}^{\infty} (a_n \cos nx + b_n \sin nx)$$

的级数称为**三角级数**，其中 $a_0, a_n, b_n$ 是常数。

### 2. 三角函数系的正交性

三角函数系：

$$1, \cos x, \sin x, \cos 2x, \sin 2x, \ldots, \cos nx, \sin nx, \ldots$$

在 $[-\pi, \pi]$ 上具有**正交性**：

$$\int_{-\pi}^{\pi} \cos nx \, dx = 0, \quad \int_{-\pi}^{\pi} \sin nx \, dx = 0 \quad (n = 1, 2, 3, \ldots)$$

$$\int_{-\pi}^{\pi} \cos mx \cos nx \, dx = \begin{cases}
0, & m \neq n \\
\pi, & m = n \neq 0
\end{cases}$$

$$\int_{-\pi}^{\pi} \sin mx \sin nx \, dx = \begin{cases}
0, & m \neq n \\
\pi, & m = n \neq 0
\end{cases}$$

$$\int_{-\pi}^{\pi} \cos mx \sin nx \, dx = 0 \quad (\text{任意 } m, n)$$

💡 **正交性的意义**：不同频率的三角函数在一个周期内的"平均值"为零。

---

## 二、傅里叶级数

### 1. 傅里叶系数

设 $f(x)$ 是周期为 $2\pi$ 的函数，在 $[-\pi, \pi]$ 上可积。若 $f(x)$ 能展开成三角级数：

$$f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} (a_n \cos nx + b_n \sin nx)$$

则系数由下列公式确定（称为**傅里叶系数**）：

$$a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \, dx$$

$$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \cos nx \, dx \quad (n = 1, 2, 3, \ldots)$$

$$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \sin nx \, dx \quad (n = 1, 2, 3, \ldots)$$

🎯 **傅里叶系数口诀**：
- $a_0$：函数的平均值（乘以 $2$）
- $a_n$：函数与 $\cos nx$ 的"相关性"
- $b_n$：函数与 $\sin nx$ 的"相关性"

### 2. 傅里叶级数的定义

由上述公式确定的三角级数称为 $f(x)$ 的**傅里叶级数**，记作：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} (a_n \cos nx + b_n \sin nx)$$

符号 $\sim$ 表示"对应"，不一定相等。

---

## 三、傅里叶级数的收敛定理

### 狄利克雷（Dirichlet）充分条件

设 $f(x)$ 是周期为 $2\pi$ 的函数，若 $f(x)$ 在 $[-\pi, \pi]$ 上满足：

1. 连续或只有有限个第一类间断点
2. 只有有限个极值点

则 $f(x)$ 的傅里叶级数收敛，且：

- 在 $f(x)$ 的连续点处，级数收敛于 $f(x)$
- 在 $f(x)$ 的间断点 $x_0$ 处，级数收敛于 $\frac{f(x_0^-) + f(x_0^+)}{2}$（左右极限的平均值）

💡 **重要结论**：在间断点处，傅里叶级数收敛于左右极限的平均值。

---

## 四、正弦级数与余弦级数

### 1. 奇函数与偶函数的傅里叶级数

**奇函数**：$f(-x) = -f(x)$

$$a_n = 0 \quad (n = 0, 1, 2, \ldots)$$

$$b_n = \frac{2}{\pi} \int_0^{\pi} f(x) \sin nx \, dx$$

傅里叶级数只含正弦项：

$$f(x) \sim \sum_{n=1}^{\infty} b_n \sin nx \quad \text{（正弦级数）}$$

**偶函数**：$f(-x) = f(x)$

$$b_n = 0 \quad (n = 1, 2, 3, \ldots)$$

$$a_n = \frac{2}{\pi} \int_0^{\pi} f(x) \cos nx \, dx$$

傅里叶级数只含余弦项：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} a_n \cos nx \quad \text{（余弦级数）}$$

🎯 **记忆口诀**：
- **奇函数 → 正弦级数**（奇对奇）
- **偶函数 → 余弦级数**（偶对偶）

### 2. 函数的奇延拓与偶延拓

对于定义在 $[0, \pi]$ 上的函数 $f(x)$：

**奇延拓**：将 $f(x)$ 延拓为奇函数，展开成正弦级数

**偶延拓**：将 $f(x)$ 延拓为偶函数，展开成余弦级数

## 五、周期为 $2l$ 的傅里叶级数

### 1. 一般周期函数的傅里叶级数

设 $f(x)$ 的周期为 $2l$，则其傅里叶级数为：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} \left(a_n \cos \frac{n\pi x}{l} + b_n \sin \frac{n\pi x}{l}\right)$$

其中：

$$a_0 = \frac{1}{l} \int_{-l}^{l} f(x) \, dx$$

$$a_n = \frac{1}{l} \int_{-l}^{l} f(x) \cos \frac{n\pi x}{l} \, dx$$

$$b_n = \frac{1}{l} \int_{-l}^{l} f(x) \sin \frac{n\pi x}{l} \, dx$$

🎯 **变换规律**：
- $\pi \to l$（周期的一半）
- $nx \to \frac{n\pi x}{l}$
- 积分区间 $[-\pi, \pi] \to [-l, l]$
- 系数前的 $\frac{1}{\pi} \to \frac{1}{l}$

---

## 六、例题实战

### 📝 例1：展开周期为 $2\pi$ 的函数

将函数 $f(x) = x$（$-\pi < x < \pi$）展开成傅里叶级数，并求 $\sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n}$ 的和。

**解**：

**Step 1**：判断奇偶性

$f(-x) = -x = -f(x)$，$f(x)$ 是奇函数。

**Step 2**：计算傅里叶系数

$$a_n = 0 \quad (n = 0, 1, 2, \ldots)$$

$$b_n = \frac{2}{\pi} \int_0^{\pi} x \sin nx \, dx$$

分部积分：$u = x, dv = \sin nx \, dx$

$$= \frac{2}{\pi} \left[-\frac{x\cos nx}{n}\bigg|_0^{\pi} + \frac{1}{n} \int_0^{\pi} \cos nx \, dx\right]$$

$$= \frac{2}{\pi} \left[-\frac{\pi\cos n\pi}{n} + \frac{\sin nx}{n^2}\bigg|_0^{\pi}\right]$$

$$= \frac{2}{\pi} \cdot \left(-\frac{\pi(-1)^n}{n}\right) = \frac{-2(-1)^n}{n} = \frac{2(-1)^{n+1}}{n}$$

**Step 3**：写出傅里叶级数

$$f(x) = \sum_{n=1}^{\infty} \frac{2(-1)^{n+1}}{n} \sin nx = 2\left(\sin x - \frac{\sin 2x}{2} + \frac{\sin 3x}{3} - \cdots\right)$$

**Step 4**：求级数和

令 $x = \frac{\pi}{2}$：

$$\frac{\pi}{2} = 2\left(\sin\frac{\pi}{2} - \frac{\sin\pi}{2} + \frac{\sin\frac{3\pi}{2}}{3} - \cdots\right)$$

$$= 2\left(1 - 0 - \frac{1}{3} + 0 + \frac{1}{5} - \cdots\right) = 2\sum_{n=0}^{\infty} \frac{(-1)^n}{2n+1}$$

这给出了莱布尼茨公式：$\frac{\pi}{4} = 1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \cdots$

对于 $\sum \frac{(-1)^{n-1}}{n}$，令 $x = \pi$（注意 $x = \pi$ 是间断点）：

$$\frac{f(\pi^-) + f(\pi^+)}{2} = \frac{\pi + (-\pi)}{2} = 0$$

但这不能直接给出答案。实际上，通过 $\ln(1+x) = \sum \frac{(-1)^{n-1}}{n} x^n$，令 $x = 1$：

$$\sum_{n=1}^{\infty} \frac{(-1)^{n-1}}{n} = \ln 2$$

✅ **答案**：$f(x) = 2\sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n} \sin nx$，$\sum \frac{(-1)^{n-1}}{n} = \ln 2$。

---

### 📝 例2：展开偶函数

将函数 $f(x) = |x|$（$-\pi \leq x \leq \pi$）展开成傅里叶级数。

**解**：

**Step 1**：判断奇偶性

$f(-x) = |-x| = |x| = f(x)$，$f(x)$ 是偶函数。

**Step 2**：计算傅里叶系数

$$b_n = 0 \quad (n = 1, 2, 3, \ldots)$$

$$a_0 = \frac{2}{\pi} \int_0^{\pi} x \, dx = \frac{2}{\pi} \cdot \frac{\pi^2}{2} = \pi$$

$$a_n = \frac{2}{\pi} \int_0^{\pi} x \cos nx \, dx$$

分部积分：

$$= \frac{2}{\pi} \left[\frac{x\sin nx}{n}\bigg|_0^{\pi} - \frac{1}{n} \int_0^{\pi} \sin nx \, dx\right]$$

$$= \frac{2}{\pi} \left[0 + \frac{\cos nx}{n^2}\bigg|_0^{\pi}\right] = \frac{2}{\pi n^2} (\cos n\pi - 1)$$

$$= \frac{2}{\pi n^2} ((-1)^n - 1) = \begin{cases}
0, & n \text{ 偶数} \\
-\frac{4}{\pi n^2}, & n \text{ 奇数}
\end{cases}$$

**Step 3**：写出傅里叶级数

$$f(x) = \frac{\pi}{2} - \frac{4}{\pi} \sum_{k=0}^{\infty} \frac{\cos(2k+1)x}{(2k+1)^2}$$

$$= \frac{\pi}{2} - \frac{4}{\pi} \left(\cos x + \frac{\cos 3x}{9} + \frac{\cos 5x}{25} + \cdots\right)$$

✅ **答案**：$f(x) = \frac{\pi}{2} - \frac{4}{\pi} \sum_{k=0}^{\infty} \frac{\cos(2k+1)x}{(2k+1)^2}$

---

### 📝 例3：周期为 $2l$ 的函数

将函数 $f(x) = x$（$0 < x < 2$）展开成正弦级数（周期延拓为 $4$）。

**解**：

周期 $2l = 4$，$l = 2$。

**Step 1**：奇延拓

将 $f(x)$ 延拓为奇函数，周期为 $4$。

**Step 2**：计算傅里叶系数

$$a_n = 0, \quad b_n = \frac{2}{l} \int_0^{l} f(x) \sin \frac{n\pi x}{l} \, dx$$

$$= \frac{2}{2} \int_0^{2} x \sin \frac{n\pi x}{2} \, dx = \int_0^{2} x \sin \frac{n\pi x}{2} \, dx$$

分部积分：$u = x, dv = \sin \frac{n\pi x}{2} \, dx$

$$= \left[-\frac{2x}{n\pi} \cos \frac{n\pi x}{2}\right]_0^{2} + \frac{2}{n\pi} \int_0^{2} \cos \frac{n\pi x}{2} \, dx$$

$$= -\frac{4}{n\pi} \cos n\pi + \frac{2}{n\pi} \cdot \frac{2}{n\pi} \sin \frac{n\pi x}{2}\bigg|_0^{2}$$

$$= -\frac{4(-1)^n}{n\pi} + 0 = \frac{4(-1)^{n+1}}{n\pi}$$

**Step 3**：写出傅里叶级数

$$f(x) = \sum_{n=1}^{\infty} \frac{4(-1)^{n+1}}{n\pi} \sin \frac{n\pi x}{2}$$

$$= \frac{4}{\pi} \left(\sin \frac{\pi x}{2} - \frac{1}{2}\sin \pi x + \frac{1}{3}\sin \frac{3\pi x}{2} - \cdots\right)$$

✅ **答案**：$f(x) = \frac{4}{\pi} \sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n} \sin \frac{n\pi x}{2}$

---

### 📝 例4：利用傅里叶级数求级数和

利用例2的结果，求 $\sum_{n=1}^{\infty} \frac{1}{(2n-1)^2}$ 的和。

**解**：

由例2：

$$|x| = \frac{\pi}{2} - \frac{4}{\pi} \sum_{k=0}^{\infty} \frac{\cos(2k+1)x}{(2k+1)^2}$$

令 $x = 0$：

$$0 = \frac{\pi}{2} - \frac{4}{\pi} \sum_{k=0}^{\infty} \frac{1}{(2k+1)^2}$$

$$\sum_{k=0}^{\infty} \frac{1}{(2k+1)^2} = \frac{\pi^2}{8}$$

即：

$$\sum_{n=1}^{\infty} \frac{1}{(2n-1)^2} = 1 + \frac{1}{9} + \frac{1}{25} + \cdots = \frac{\pi^2}{8}$$

✅ **答案**：$\frac{\pi^2}{8}$

| 周期 | 傅里叶系数 |
|---|---|
| **$2\pi$** | $a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \, dx$ |

|  | $b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \sin nx \, dx$ |

|  | $b_n = \frac{1}{l} \int_{-l}^{l} f(x) \sin \frac{n\pi x}{l} \, dx$ |

## 八、奇偶函数的傅里叶级数

| 函数性质 | 傅里叶级数 | 系数计算 |
|---|---|---|
| **奇函数** | $\sum b_n \sin nx$ | $b_n = \frac{2}{\pi} \int_0^{\pi} f(x) \sin nx \, dx$ |
| **偶函数** | $\frac{a_0}{2} + \sum a_n \cos nx$ | $a_n = \frac{2}{\pi} \int_0^{\pi} f(x) \cos nx \, dx$ |

## 九、常见题型总结

| 题型 | 方法 | 注意事项 |
|---|---|---|
| 计算傅里叶系数 | 分部积分 | 注意奇偶性简化 |
| 奇函数展开 | 只计算 $b_n$ | 正弦级数 |
| 偶函数展开 | 只计算 $a_n$ | 余弦级数 |
| 周期为 $2l$ | 变换公式 | $\pi \to l$，$nx \to \frac{n\pi x}{l}$ |
| 求级数和 | 代入特殊值 | 注意间断点 |
