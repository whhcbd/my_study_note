# 8.5 方向导数与梯度

> 📌 **本节核心**：理解方向导数的几何意义，掌握梯度的计算与性质

---

## 一、方向导数

### 1. 方向导数的定义

设函数 $z = f(x, y)$ 在点 $P_0(x_0, y_0)$ 的某邻域内有定义，$\vec{l} = (\cos\alpha, \cos\beta)$ 是一个单位方向向量。

若极限

$$\lim_{t \to 0^+} \frac{f(x_0 + t\cos\alpha, y_0 + t\cos\beta) - f(x_0, y_0)}{t}$$

存在，则称此极限为 $f$ 在点 $P_0$ 沿方向 $\vec{l}$ 的**方向导数**，记作：

$$\frac{\partial f}{\partial \vec{l}}\bigg|_{P_0} \quad \text{或} \quad f_{\vec{l}}(x_0, y_0)$$

💡 **几何意义**：函数在某点沿指定方向的变化率。

### 2. 方向导数的计算公式

若 $f(x, y)$ 在点 $P_0(x_0, y_0)$ 可微，则方向导数存在，且：

$$\frac{\partial f}{\partial \vec{l}}\bigg|_{P_0} = f_x(x_0, y_0)\cos\alpha + f_y(x_0, y_0)\cos\beta$$

其中 $\cos\alpha, \cos\beta$ 是方向 $\vec{l}$ 的**方向余弦**。

🎯 **计算口诀**：
- **Step 1**：求偏导数 $f_x, f_y$
- **Step 2**：确定方向余弦 $\cos\alpha, \cos\beta$
- **Step 3**：代入公式计算

### 3. 三元函数的方向导数

对于 $u = f(x, y, z)$，方向 $\vec{l} = (\cos\alpha, \cos\beta, \cos\gamma)$：

$$\frac{\partial f}{\partial \vec{l}}\bigg|_{P_0} = f_x\cos\alpha + f_y\cos\beta + f_z\cos\gamma$$

## 二、梯度

### 1. 梯度的定义

函数 $f(x, y)$ 在点 $P_0(x_0, y_0)$ 的**梯度**是一个向量：

$$\text{grad}\,f(x_0, y_0) = \nabla f(x_0, y_0) = \left(f_x(x_0, y_0), f_y(x_0, y_0)\right)$$

或记作：

$$\nabla f = \frac{\partial f}{\partial x}\vec{i} + \frac{\partial f}{\partial y}\vec{j}$$

### 2. 梯度的几何意义

📐 **核心性质**：

1. **方向**：梯度的方向是函数增长最快的方向
2. **大小**：梯度的模 $|\nabla f|$ 是函数在该点的最大方向导数
3. **垂直性**：梯度垂直于等值线（或等值面）

💡 **形象理解**：
- 梯度像"山坡的最陡方向"
- 沿梯度方向爬山最快
- 等高线与梯度垂直

### 3. 方向导数与梯度的关系

设 $\vec{l} = (\cos\alpha, \cos\beta)$ 是单位方向向量，则：

$$\frac{\partial f}{\partial \vec{l}} = \nabla f \cdot \vec{l} = |\nabla f| \cos\theta$$

其中 $\theta$ 是梯度 $\nabla f$ 与方向 $\vec{l}$ 的夹角。

🎯 **重要结论**：

| 条件 | 方向导数 |
|---|---|
| $\theta = 0$（沿梯度方向） | 最大值 $\|\nabla f\|$ |
| $\theta = \pi$（反梯度方向） | 最小值 $-\|\nabla f\|$ |
| $\theta = \frac{\pi}{2}$（垂直梯度） | 0（等值线方向） |

---

## 三、三元函数的梯度

对于 $u = f(x, y, z)$：

$$\nabla f = \left(f_x, f_y, f_z\right) = f_x\vec{i} + f_y\vec{j} + f_z\vec{k}$$

方向导数：

$$\frac{\partial f}{\partial \vec{l}} = \nabla f \cdot \vec{l} = |\nabla f| \cos\theta$$

## 四、例题实战

### 📝 例1：计算方向导数

求函数 $f(x, y) = x^2 + y^2$ 在点 $(1, 2)$ 沿从点 $(1, 2)$ 到点 $(2, 3)$ 方向的方向导数。

**解**：

**Step 1**：求偏导数

$$f_x = 2x, \quad f_y = 2y$$

在点 $(1, 2)$：$f_x(1, 2) = 2, f_y(1, 2) = 4$

**Step 2**：确定方向向量

从 $(1, 2)$ 到 $(2, 3)$ 的方向向量：$\vec{v} = (2-1, 3-2) = (1, 1)$

单位化：$|\vec{v}| = \sqrt{1^2 + 1^2} = \sqrt{2}$

$$\vec{l} = \frac{1}{\sqrt{2}}(1, 1) = \left(\frac{\sqrt{2}}{2}, \frac{\sqrt{2}}{2}\right)$$

方向余弦：$\cos\alpha = \cos\beta = \frac{\sqrt{2}}{2}$

**Step 3**：计算方向导数

$$\frac{\partial f}{\partial \vec{l}}\bigg|_{(1,2)} = 2 \cdot \frac{\sqrt{2}}{2} + 4 \cdot \frac{\sqrt{2}}{2} = \sqrt{2} + 2\sqrt{2} = 3\sqrt{2}$$

✅ **答案**：$3\sqrt{2}$

---

### 📝 例2：求梯度

求函数 $f(x, y) = e^x \sin y$ 在点 $(0, \frac{\pi}{4})$ 的梯度，并求函数在该点增长最快的方向。

**解**：

**Step 1**：求偏导数

$$f_x = e^x \sin y, \quad f_y = e^x \cos y$$

在点 $(0, \frac{\pi}{4})$：

$$f_x\left(0, \frac{\pi}{4}\right) = e^0 \sin\frac{\pi}{4} = \frac{\sqrt{2}}{2}$$

$$f_y\left(0, \frac{\pi}{4}\right) = e^0 \cos\frac{\pi}{4} = \frac{\sqrt{2}}{2}$$

**Step 2**：写出梯度

$$\nabla f\left(0, \frac{\pi}{4}\right) = \left(\frac{\sqrt{2}}{2}, \frac{\sqrt{2}}{2}\right)$$

**Step 3**：增长最快的方向

增长最快的方向就是梯度方向：$\left(\frac{\sqrt{2}}{2}, \frac{\sqrt{2}}{2}\right)$

单位化后：$\vec{l} = \left(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}\right)$

最大方向导数：$|\nabla f| = \sqrt{\left(\frac{\sqrt{2}}{2}\right)^2 + \left(\frac{\sqrt{2}}{2}\right)^2} = 1$

✅ **答案**：梯度为 $\left(\frac{\sqrt{2}}{2}, \frac{\sqrt{2}}{2}\right)$，增长最快方向为 $\left(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}\right)$，最大方向导数为 $1$。

---

### 📝 例3：三元函数的方向导数

求函数 $u = xyz$ 在点 $(1, 1, 1)$ 沿方向 $\vec{l} = (1, 2, 2)$ 的方向导数。

**解**：

**Step 1**：求偏导数

$$u_x = yz, \quad u_y = xz, \quad u_z = xy$$

在点 $(1, 1, 1)$：$u_x = 1, u_y = 1, u_z = 1$

**Step 2**：单位化方向向量

$$|\vec{l}| = \sqrt{1^2 + 2^2 + 2^2} = 3$$

$$\vec{l}_0 = \frac{1}{3}(1, 2, 2) = \left(\frac{1}{3}, \frac{2}{3}, \frac{2}{3}\right)$$

**Step 3**：计算方向导数

$$\frac{\partial u}{\partial \vec{l}}\bigg|_{(1,1,1)} = 1 \cdot \frac{1}{3} + 1 \cdot \frac{2}{3} + 1 \cdot \frac{2}{3} = \frac{5}{3}$$

✅ **答案**：$\frac{5}{3}$

---

### 📝 例4：梯度与等值线

证明：函数 $f(x, y)$ 的梯度垂直于过该点的等值线。

**证明**：

设等值线方程为 $f(x, y) = C$（常数）。

将等值线参数化：$x = x(t), y = y(t)$，则：

$$f(x(t), y(t)) = C$$

两边对 $t$ 求导：

$$f_x \frac{dx}{dt} + f_y \frac{dy}{dt} = 0$$

即：

$$\nabla f \cdot \left(\frac{dx}{dt}, \frac{dy}{dt}\right) = 0$$

其中 $\left(\frac{dx}{dt}, \frac{dy}{dt}\right)$ 是等值线的切向量。

因此，梯度 $\nabla f$ 垂直于等值线的切向量，即垂直于等值线。

✅ **证毕**

## 五、核心公式速查表

| 概念 | 公式 |
|---|---|
| **方向导数（二元）** | $\frac{\partial f}{\partial \vec{l}} = f_x\cos\alpha + f_y\cos\beta$ |
| **方向导数（三元）** | $\frac{\partial f}{\partial \vec{l}} = f_x\cos\alpha + f_y\cos\beta + f_z\cos\gamma$ |
| **梯度（二元）** | $\nabla f = (f_x, f_y)$ |
| **梯度（三元）** | $\nabla f = (f_x, f_y, f_z)$ |
| **方向导数与梯度关系** | $\frac{\partial f}{\partial \vec{l}} = \nabla f \cdot \vec{l} = \|\nabla f\| \cos\theta$ |
| **最大方向导数** | $\max \frac{\partial f}{\partial \vec{l}} = \|\nabla f\|$（沿梯度方向） |

---

## 六、常见题型总结

| 题型 | 方法 | 注意事项 |
|---|---|---|
| 计算方向导数 | 求偏导 + 方向余弦 | 方向向量要单位化 |
| 求梯度 | 直接求偏导数 | 梯度是向量 |
| 最大方向导数 | 求梯度的模 | 方向为梯度方向 |
| 等值线/面问题 | 利用梯度垂直性 | 梯度垂直于等值线 |

⭐ **学习建议**：
1. 理解方向导数的几何意义：沿某方向的变化率
2. 掌握梯度的三个核心性质：方向、大小、垂直性
3. 注意方向向量必须单位化后才能用于计算方向导数
4. 梯度在优化问题中应用广泛（如梯度下降法）
