# 📐 二重积分的计算 听课笔记

> 📌 **课程来源**：KR 老师高数下预习课
> 🎯 **核心目标**：掌握二重积分在直角坐标系和极坐标系下的计算方法，学会交换积分次序和利用对称性
> 💡 **重要性**：二重积分计算是考试必考大题，交换积分次序是高频考点

---

## 一、直角坐标系下的计算

### 1.1 X 型区域（先 $y$ 后 $x$）

若 $D$ 可表示为 $a \leq x \leq b$，$\varphi_1(x) \leq y \leq \varphi_2(x)$，则：

$$\iint_D f(x,y)\,\mathrm{d}x\mathrm{d}y = \int_a^b \left[\int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y)\,\mathrm{d}y\right] \mathrm{d}x$$

> 💡 **口诀**：X 型区域，**先积 $y$ 后积 $x$**。$y$ 的上下限是 $x$ 的函数，$x$ 的上下限是常数。

**图形理解**：用平行于 $y$ 轴的线"穿"区域，穿入点为 $y = \varphi_1(x)$，穿出点为 $y = \varphi_2(x)$。

### 1.2 Y 型区域（先 $x$ 后 $y$）

若 $D$ 可表示为 $c \leq y \leq d$，$\psi_1(y) \leq x \leq \psi_2(y)$，则：

$$\iint_D f(x,y)\,\mathrm{d}x\mathrm{d}y = \int_c^d \left[\int_{\psi_1(y)}^{\psi_2(y)} f(x,y)\,\mathrm{d}x\right] \mathrm{d}y$$

> 💡 **口诀**：Y 型区域，**先积 $x$ 后积 $y$**。

### 1.3 如何选择积分次序 ⭐

| 情况 | 选择 |
| :--- | :--- |
| 区域既是 X 型又是 Y 型 | 看哪个方向**上下限更简单** |
| 被积函数对某个变量积不出 | 换次序！先积能积的那个变量 |
| 题目要求交换次序 | 先从累次积分还原区域 $D$，再重新定限 |

> **例题 1**：计算 $\iint_D xy\,\mathrm{d}x\mathrm{d}y$，其中 $D$ 由 $y = x$，$y = x^2$ 围成。

- **Step 1：确定区域**。联立 $x = x^2$ 得交点 $(0,0)$ 和 $(1,1)$。区域 $D$：$0 \leq x \leq 1$，$x^2 \leq y \leq x$。
- **Step 2：先积 $y$ 后积 $x$**
  $$\int_0^1 \left[\int_{x^2}^x xy\,\mathrm{d}y\right] \mathrm{d}x = \int_0^1 x \cdot \frac{y^2}{2}\bigg|_{x^2}^x \mathrm{d}x = \int_0^1 x \cdot \frac{x^2 - x^4}{2}\,\mathrm{d}x$$
  $$= \frac{1}{2}\int_0^1 (x^3 - x^5)\,\mathrm{d}x = \frac{1}{2}\left[\frac{x^4}{4} - \frac{x^6}{6}\right]_0^1 = \frac{1}{2}\left(\frac{1}{4} - \frac{1}{6}\right) = \frac{1}{2} \cdot \frac{1}{12} = \frac{1}{24}$$

> **例题 2**：计算 $\iint_D e^{-y^2}\,\mathrm{d}x\mathrm{d}y$，其中 $D$ 由 $y = x$，$y = 1$，$x = 0$ 围成。

- **分析**：$e^{-y^2}$ 对 $y$ 积不出原函数（没有初等表达式）！必须**先积 $x$ 后积 $y$**。
- **Step 1：化为 Y 型区域**。$0 \leq y \leq 1$，$0 \leq x \leq y$。
- **Step 2：先积 $x$**
  $$\int_0^1 \left[\int_0^y e^{-y^2}\,\mathrm{d}x\right] \mathrm{d}y = \int_0^1 e^{-y^2} \cdot y\,\mathrm{d}y$$
- **Step 3**：令 $u = -y^2$，$\mathrm{d}u = -2y\,\mathrm{d}y$
  $$= \int_0^1 y e^{-y^2}\,\mathrm{d}y = -\frac{1}{2}e^{-y^2}\bigg|_0^1 = -\frac{1}{2}(e^{-1} - 1) = \frac{1}{2}\left(1 - \frac{1}{e}\right)$$

## 二、极坐标系下的计算 ⭐⭐

### 2.1 变换公式

当积分区域是**圆、扇形、环形**或被积函数含 $x^2 + y^2$ 时，用极坐标更方便。

$$x = r\cos\theta, \quad y = r\sin\theta, \quad \mathrm{d}x\mathrm{d}y = r\,\mathrm{d}r\mathrm{d}\theta$$

> ⚠️ **注意**：面积元素多了一个 $r$！$\mathrm{d}\sigma = r\,\mathrm{d}r\mathrm{d}\theta$，千万别忘！

### 2.2 极坐标下的累次积分

$$\iint_D f(x,y)\,\mathrm{d}\sigma = \int_{\alpha}^{\beta} \left[\int_{r_1(\theta)}^{r_2(\theta)} f(r\cos\theta, r\sin\theta) \cdot r\,\mathrm{d}r\right] \mathrm{d}\theta$$

**常见区域的极坐标表示**：

| 区域形状 | 极坐标表示 |
| | :--- |
| **圆心在原点的圆** $x^2+y^2 \leq R^2$ | $0 \leq \theta \leq 2\pi$，$0 \leq r \leq R$ |
| **第一象限的扇形** | $0 \leq \theta \leq \frac{\pi}{2}$，$0 \leq r \leq R$ |
| **环形** $a^2 \leq x^2+y^2 \leq b^2$ | $0 \leq \theta \leq 2\pi$，$a \leq r \leq b$ |
| **偏心圆** $(x-a)^2+y^2 \leq a^2$ | $-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2}$，$0 \leq r \leq 2a\cos\theta$ |

> **例题 3**：计算 $\iint_D (x^2 + y^2)\,\mathrm{d}\sigma$，其中 $D = \{(x,y) \mid x^2 + y^2 \leq 1\}$。

- **Step 1**：区域是单位圆，用极坐标
  - $0 \leq \theta \leq 2\pi$，$0 \leq r \leq 1$
- **Step 2**：被积函数 $x^2 + y^2 = r^2$
  $$I = \int_0^{2\pi} \left[\int_0^1 r^2 \cdot r\,\mathrm{d}r\right] \mathrm{d}\theta = \int_0^{2\pi} \frac{r^4}{4}\bigg|_0^1 \mathrm{d}\theta = \int_0^{2\pi} \frac{1}{4}\,\mathrm{d}\theta = \frac{\pi}{2}$$

## 三、交换积分次序 ⭐⭐⭐ 高频考点！

### 3.1 方法

1. 从给定的累次积分中，读出区域 $D$ 的边界
2. **画出区域 $D$**（关键步骤！）
3. 按新的积分次序重新确定上下限

> **例题 4**：交换 $\int_0^1 \mathrm{d}x \int_0^{x^2} f(x,y)\,\mathrm{d}y$ 的积分次序。

- **Step 1：读出区域**。$0 \leq x \leq 1$，$0 \leq y \leq x^2$
- **Step 2：画图**。$y = x^2$ 是抛物线，区域在抛物线下方、$x$ 轴上方，$x$ 从 $0$ 到 $1$
- **Step 3：改为先 $x$ 后 $y$**
  - $y$ 的范围：$0 \leq y \leq 1$
  - 从 $y$ 处水平"穿"：$x$ 从 $\sqrt{y}$ 到 $1$
  - 结果：$\int_0^1 \mathrm{d}y \int_{\sqrt{y}}^1 f(x,y)\,\mathrm{d}x$ ✅

> **例题 5**：交换 $\int_0^1 \mathrm{d}x \int_x^1 e^{-y^2}\,\mathrm{d}y$ 的积分次序并计算。

- **Step 1**：区域 $D$：$0 \leq x \leq 1$，$x \leq y \leq 1$（即 $y = x$ 上方的三角形）
- **Step 2**：改为先 $x$ 后 $y$：$0 \leq y \leq 1$，$0 \leq x \leq y$
- **Step 3**：
  $$\int_0^1 \mathrm{d}y \int_0^y e^{-y^2}\,\mathrm{d}x = \int_0^1 e^{-y^2} \cdot y\,\mathrm{d}y = -\frac{1}{2}e^{-y^2}\bigg|_0^1 = \frac{1}{2}\left(1 - \frac{1}{e}\right)$$

## 四、利用对称性简化计算 ⭐

### 4.1 奇偶对称性

设 $D$ 关于 $y$ 轴对称（即 $(x,y) \in D \Rightarrow (-x,y) \in D$）：

| 被积函数性质 | 结论 |
| | :--- |
| $f(-x, y) = -f(x, y)$（关于 $x$ 为奇函数） | $\iint_D f\,\mathrm{d}\sigma = 0$ |
| $f(-x, y) = f(x, y)$（关于 $x$ 为偶函数） | $\iint_D f\,\mathrm{d}\sigma = 2\iint_{D_1} f\,\mathrm{d}\sigma$（$D_1$ 是 $D$ 的 $x \geq 0$ 部分） |

关于 $x$ 轴对称的情况类似。

### 4.2 轮换对称性

若 $D$ 关于 $y = x$ 对称，则：
$$\iint_D f(x,y)\,\mathrm{d}\sigma = \iint_D f(y,x)\,\mathrm{d}\sigma$$

> 💡 **口诀**：区域对称看奇偶，奇函数积分等于零，偶函数积分两半算！

> **例题 6**：计算 $\iint_D (x^3 + y\sin x)\,\mathrm{d}\sigma$，其中 $D = \{(x,y) \mid x^2 + y^2 \leq 1\}$。

- **Step 1**：$D$ 关于 $y$ 轴对称
- **Step 2**：$x^3$ 关于 $x$ 为奇函数，$\iint_D x^3\,\mathrm{d}\sigma = 0$
- **Step 3**：$y\sin x$ 关于 $x$ 为奇函数（$\sin(-x) = -\sin x$），$\iint_D y\sin x\,\mathrm{d}\sigma = 0$
- **结果**：$I = 0 + 0 = 0$ ✅

## 📝 总结与记忆要点

| 方法 | 适用场景 | 关键点 |
| | :--- | :--- |
| **直角坐标 X 型** | 先积 $y$ 后积 $x$ | $y$ 的限是 $x$ 的函数 |
| **直角坐标 Y 型** | 先积 $x$ 后积 $y$ | $x$ 的限是 $y$ 的函数 |
| **极坐标** | 圆/扇形/环形区域 | $\mathrm{d}\sigma = r\,\mathrm{d}r\mathrm{d}\theta$，别忘 $r$！ |
| **交换次序** | 先画图再定限 | 从累次积分还原区域是关键 |
| **奇偶对称** | 区域有对称轴 | 奇函数积分为 0，偶函数算一半 |

- **口诀**：二重积分两大招，直角坐标极坐标。圆的区域用极角，别忘乘 $r$ 这个宝。交换次序先画图，对称奇偶省功夫！
- **易错点**：
  - 极坐标**必须乘 $r$**！这是最常见的丢分点
  - 交换积分次序时，必须**画出区域图**，否则很容易定错限
  - $e^{-y^2}$、$\frac{\sin y}{y}$ 这类函数"积不出"，必须换次序
