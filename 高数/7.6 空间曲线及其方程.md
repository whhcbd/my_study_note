## 一、空间曲线的两种方程

### 1.1 参数方程
- **思想**：将曲线视为质点运动的轨迹，`t` 为时间参数。
- **形式**：
  $$\begin{cases} x = x(t) \\ y = y(t) \\ z = z(t) \end{cases}$$
- **示例**（螺旋线）：
  $$\begin{cases} x = \cos t \\ y = \sin t \\ z = t \end{cases}$$
  - 消去 `t` 得 `x² + y² = 1`（圆柱面），`z = t` 表示高度随 `t` 线性增加。
  - 几何意义：点在圆柱面上一边旋转一边上升，形成螺旋线。

### 1.2 一般方程（交面式）
- **思想**：空间曲线可看作两张空间曲面的交线。
- **形式**：
  $$\begin{cases} F(x,y,z) = 0 \\ G(x,y,z) = 0 \end{cases}$$
- **示例 1**：判断曲线形状
  $$\begin{cases} x^2 + y^2 = 1 \\ 2x + 3z = 6 \end{cases}$$
  - 第一方程：圆柱面（平行于 z 轴）
  - 第二方程：平面（平行于 y 轴，截距 x=3, z=2）
  - 交线：一个椭圆（平面斜截圆柱）

## 二、实用工具：GeoGebra 3D 绘图

### 2.1 基本操作
- 搜索“GeoGebra 3D计算器”进入。
- 直接输入方程，自动生成曲面。
- 可旋转、缩放，多角度观察。

### 2.2 示例应用
- 输入 `x^2 + y^2 = 1` → 圆柱面
- 输入 `2x + 3z = 6` → 平面
- 两者相交，直观看到交线形状（椭圆）
- 输入参数方程 `(cos(t), sin(t), t)`，调整 t 范围（如 0 到 8π），得到多圈螺旋线。

## 三、空间曲线在坐标面上的投影

### 3.1 核心概念
- **投影柱面**：过曲线 L 且母线平行于某坐标轴的柱面。
  - 例如，往 xOy 面投影 → 投影柱面平行于 z 轴。
- **投影曲线**：投影柱面与坐标平面的交线。

### 3.2 求解步骤（以 xOy 面投影为例）
1. **消元**：从曲线方程组中消去变量 `z`，得到方程 `H(x,y)=0`。
   - 此方程表示**投影柱面**（缺 z，母线平行 z 轴）。
2. **联立坐标平面**：将 `H(x,y)=0` 与 `z=0` 联立，即得投影曲线方程：
   $$\begin{cases} H(x,y) = 0 \\ z = 0 \end{cases}$$

### 3.3 推广到其他坐标面
- 往 **yOz 面** 投影：消去 `x`，得 `H(y,z)=0`，联立 `x=0`
- 往 **xOz 面** 投影：消去 `y`，得 `H(x,z)=0`，联立 `y=0`
- 若要求**过曲线且平行于某坐标轴的柱面**，则消去对应的变量即可（例如平行 y 轴 → 消 y）。

## 四、典型例题

### 4.1 例 2：参数方程表示的螺旋线
**题目**：分析曲线
$$\begin{cases} x = \cos t \\ y = \sin t \\ z = t \end{cases}$$
**分析**：
- 消去 t 得 `x² + y² = 1` → 曲线在圆柱面上。
- `z = t` 随 t 增大而增大，故曲线是螺旋上升的。

### 4.2 例 3：求交线在 xOy 面的投影
**题目**：曲线由
$$\begin{cases} \frac{x^2}{16} + \frac{y^2}{4} - \frac{z^2}{5} = 1 \\ x - 2z + 3 = 0 \end{cases}$$
给出，求其在 xOy 面上的投影曲线。

**解**：
1. 从平面方程解出 `z = (x+3)/2`。
2. 代入双曲面方程：
   $$\frac{x^2}{16} + \frac{y^2}{4} - \frac{1}{5}\cdot\frac{(x+3)^2}{4} = 1$$
   化简得：
   $$x^2 + 20y^2 - 24x - 116 = 0$$
   此为投影柱面方程（平行 z 轴）。
3. 联立 `z=0` 得投影曲线：
   $$\begin{cases} x^2 + 20y^2 - 24x - 116 = 0 \\ z = 0 \end{cases}$$

### 4.3 终极 Boss：空间区域在坐标面上的投影
**题目**：半球体 `x² + y² + z² ≤ a²`（`z ≥ 0`）与圆柱体 `x² + y² ≤ ax` 相交，求该立体在 xOy 面和 xOz 面上的投影区域。

**分析**：
- 立体是半球内部与圆柱内部的交集。
- **在 xOy 面投影**：直接就是圆柱的底面圆域（因为半球投影是更大的圆，但被圆柱限制）。
  $$\begin{cases} x^2 + y^2 \le ax \\ z = 0 \end{cases}$$
  可化为标准形 `(x - a/2)^2 + y^2 ≤ (a/2)^2`。
- **在 xOz 面投影**：需要综合考虑两个立体的轮廓。
  - 半球在 xOz 面的投影：半圆 `x² + z² ≤ a²`（`z ≥ 0`）
  - 圆柱在 xOz 面的投影：由 `x² + y² ≤ ax` 对 y 任意，消去 y 得 `x² ≤ ax`（即 `0 ≤ x ≤ a`），且无 z 限制，但结合半球后，z 受半球上界限制。
  - 最终投影为：`x ∈ [0, a]`，`0 ≤ z ≤ √(a² - x²)`，即半圆区域 `x² + z² ≤ a²` 且 `x ≥ 0`。

**结论**：
- xOy 投影：圆域 `(x - a/2)² + y² ≤ (a/2)²`
- xOz 投影：半圆域 `x² + z² ≤ a²` 且 `x ≥ 0, z ≥ 0`
