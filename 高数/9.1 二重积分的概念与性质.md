# 📐 二重积分的概念与性质 听课笔记

> 📌 **课程来源**：KR 老师高数下预习课
> 🎯 **核心目标**：理解二重积分的定义、几何意义，掌握其基本性质
> 💡 **重要性**：二重积分是重积分的基础，后续三重积分、曲面积分都要用到它的思想

---

## 一、二重积分的概念

### 1.1 引例：曲顶柱体的体积

- **问题**：设有一立体，底是 $xOy$ 面上的闭区域 $D$，顶是曲面 $z = f(x, y) \geq 0$，侧面是以 $D$ 的边界为准线、母线平行于 $z$ 轴的柱面。求这个**曲顶柱体**的体积。
- **思路**：和定积分求曲边梯形面积一样的"**分割 → 近似 → 求和 → 取极限**"四步法。

### 1.2 定义 ⭐核心

设 $f(x, y)$ 是有界闭区域 $D$ 上的有界函数。将 $D$ 任意分成 $n$ 个小闭区域 $\Delta\sigma_1, \Delta\sigma_2, \ldots, \Delta\sigma_n$，在每个小区域上任取一点 $(\xi_i, \eta_i)$，作积分和：
$$\sum_{i=1}^{n} f(\xi_i, \eta_i) \Delta\sigma_i$$

若当各小区域直径的最大值 $\lambda \to 0$ 时，此和的极限总存在且与分割方式及取点方式无关，则称此极限为 $f(x, y)$ 在 $D$ 上的**二重积分**：
$$\iint_D f(x, y) \,\mathrm{d}\sigma = \lim_{\lambda \to 0} \sum_{i=1}^{n} f(\xi_i, \eta_i) \Delta\sigma_i$$

- $f(x, y)$ 称为**被积函数**，$D$ 称为**积分区域**，$\mathrm{d}\sigma$ 称为**面积元素**。
- 在直角坐标系中，$\mathrm{d}\sigma = \mathrm{d}x\mathrm{d}y$。

### 1.3 几何意义

| $f(x,y)$ 的符号 | 二重积分的几何意义 |
| :--- | :--- |
| $f(x,y) \geq 0$ | 曲顶柱体的**体积** |
| $f(x,y) \leq 0$ | 曲顶柱体体积的**负值** |
| $f(x,y)$ 有正有负 | 各部分体积的**代数和**（上方面积减下方面积） |

### 1.4 存在性

> **定理**：若 $f(x, y)$ 在有界闭区域 $D$ 上**连续**，则二重积分 $\iint_D f(x,y)\,\mathrm{d}\sigma$ **一定存在**。

> 💡 **口诀**：连续必可积！和定积分一样。

## 二、二重积分的性质

### 2.1 线性性

$$\iint_D [af(x,y) + bg(x,y)] \,\mathrm{d}\sigma = a\iint_D f(x,y)\,\mathrm{d}\sigma + b\iint_D g(x,y)\,\mathrm{d}\sigma$$

### 2.2 区域可加性

若 $D = D_1 \cup D_2$，且 $D_1, D_2$ 无公共内点，则：
$$\iint_D f(x,y)\,\mathrm{d}\sigma = \iint_{D_1} f(x,y)\,\mathrm{d}\sigma + \iint_{D_2} f(x,y)\,\mathrm{d}\sigma$$

### 2.3 比较定理 ⭐

若在 $D$ 上 $f(x,y) \leq g(x,y)$，则：
$$\iint_D f(x,y)\,\mathrm{d}\sigma \leq \iint_D g(x,y)\,\mathrm{d}\sigma$$

**推论**：
$$\left|\iint_D f(x,y)\,\mathrm{d}\sigma\right| \leq \iint_D |f(x,y)|\,\mathrm{d}\sigma$$

### 2.4 估值定理 ⭐

设 $f(x,y)$ 在 $D$ 上的最大值为 $M$，最小值为 $m$，$\sigma$ 为 $D$ 的面积，则：
$$m\sigma \leq \iint_D f(x,y)\,\mathrm{d}\sigma \leq M\sigma$$

> 💡 **口诀**：最小值乘面积 ≤ 积分 ≤ 最大值乘面积。**夹住它！**

### 2.5 中值定理

若 $f(x,y)$ 在有界闭区域 $D$ 上连续，$\sigma$ 为 $D$ 的面积，则存在 $(\xi, \eta) \in D$，使得：
$$\iint_D f(x,y)\,\mathrm{d}\sigma = f(\xi, \eta) \cdot \sigma$$

> 💡 **理解**：一定存在一点，函数值恰好等于积分的"平均值"。

### 2.6 特殊情况

$$\iint_D 1 \,\mathrm{d}\sigma = \sigma$$

即被积函数为 1 时，二重积分等于区域 $D$ 的**面积**。

## 三、利用性质解题

> **例题 1**：估计 $I = \iint_D (x^2 + 4y^2 + 9)\,\mathrm{d}\sigma$ 的值，其中 $D = \{(x,y) \mid x^2 + y^2 \leq 4\}$。

- **Step 1**：求 $f(x,y) = x^2 + 4y^2 + 9$ 在 $D$ 上的最小值和最大值
  - 在圆 $x^2 + y^2 \leq 4$ 上：$x^2 + y^2 \leq 4$
  - $f = x^2 + 4y^2 + 9 = x^2 + y^2 + 3y^2 + 9 \leq 4 + 3 \cdot 4 + 9 = 25$
  - 最小值：当 $x = y = 0$ 时，$f = 9$
  - 最大值：在 $y^2 = 4$（即 $x = 0, y = \pm 2$）时，$f = 0 + 16 + 9 = 25$
- **Step 2**：$D$ 的面积 $\sigma = \pi \cdot 2^2 = 4\pi$
- **Step 3**：由估值定理
  - $9 \times 4\pi \leq I \leq 25 \times 4\pi$
  - $36\pi \leq I \leq 100\pi$ ✅

> **例题 2**：比较 $\iint_D \ln(x+y)\,\mathrm{d}\sigma$ 与 $\iint_D [\ln(x+y)]^2\,\mathrm{d}\sigma$ 的大小，其中 $D$ 是以 $(1,0), (0,1), (1,1)$ 为顶点的三角形区域。

- **Step 1**：确定 $D$ 上 $x+y$ 的范围
  - $D$ 的顶点处：$x + y = 1, 1, 2$
  - 在 $D$ 上：$1 \leq x + y \leq 2$
- **Step 2**：当 $1 \leq x + y \leq 2$ 时，$0 \leq \ln(x+y) \leq \ln 2 < 1$
  - 所以 $0 \leq \ln(x+y) < 1$
  - 故 $[\ln(x+y)]^2 \leq \ln(x+y)$（因为一个 $0\sim1$ 之间的数，平方后更小）
- **Step 3**：由比较定理
  - $\iint_D [\ln(x+y)]^2\,\mathrm{d}\sigma \leq \iint_D \ln(x+y)\,\mathrm{d}\sigma$ ✅

> **例题 3**：设 $D = \{(x,y) \mid x^2 + y^2 \leq 1\}$，证明 $\pi \leq \iint_D (1 + x^2 + y^2)\,\mathrm{d}\sigma \leq 2\pi$。

- **Step 1**：在 $D$ 上 $0 \leq x^2 + y^2 \leq 1$，所以 $1 \leq 1 + x^2 + y^2 \leq 2$
- **Step 2**：$D$ 的面积 $\sigma = \pi$
- **Step 3**：由估值定理，$1 \cdot \pi \leq I \leq 2 \cdot \pi$ ✅

## 📝 总结与记忆要点

| 性质 | 内容 | 用途 |
| :--- | :--- | :--- |
| **线性性** | 积分号可分配到加减 | 拆分复杂被积函数 |
| **区域可加性** | 分块积分再相加 | 处理复杂区域 |
| **比较定理** | $f \leq g \Rightarrow$ 积分也 $\leq$ | 比较积分大小 |
| **估值定理** | $m\sigma \leq I \leq M\sigma$ | 估计积分值范围 |
| **中值定理** | $I = f(\xi,\eta) \cdot \sigma$ | 理论证明 |
| **面积公式** | $\iint_D 1\,\mathrm{d}\sigma = \sigma$ | 求面积 |

- **口诀**：二重积分四步走，分割近似求和极限有。连续必可积，估值看最值乘面积！
- **易错点**：估值定理中的 $\sigma$ 是区域的**面积**，不是"体积"——别和三重积分搞混。
