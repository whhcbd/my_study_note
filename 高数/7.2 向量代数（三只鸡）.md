# 📐 向量代数与空间解析几何（第二关：三只鸡）听课笔记

## 🎯 核心内容概览
本期视频攻克三大运算：**数量积（点积）**、**向量积（叉积）**、**混合积**。掌握它们，你就能解决夹角、垂直、面积、体积、共面等问题。

---

## 一、第一只鸡：数量积（点积）—— 结果是一个数

### 1.1 定义与物理背景
- **物理模型**：恒力 `F` 使物体产生位移 `S`，所做的功 `W`。
  - `W = |F| * |S| * cosθ` （θ 是 F 与 S 的夹角）
  - 功是标量（数量），有正负（锐角正，钝角负）。
- **数学定义**：对于任意向量 `A` 和 `B`，数量积为：
  - `A · B = |A| * |B| * cosθ`

### 1.2 投影（重要概念）
- **定义**：向量 `B` 在向量 `A` 上的投影是一个**实数**（不是向量）。
  - `Prj_A B = |B| * cosθ` （“背头的A”做角标，被投的B写大）
- **几何意义**：过B的起点和终点向A作垂线，截在A上的长度（带方向，钝角为负）。
- **投影的运算性质**：
  - `Prj_C (A + B) = Prj_C A + Prj_C B`
  - `Prj_C (λA) = λ Prj_C A`
- **点积的投影理解**：`A · B = |A| * (B在A上的投影) = |B| * (A在B上的投影)`

### 1.3 坐标计算公式（核心！）
- **公式**：若 `A = (a₁, a₂, a₃)`，`B = (b₁, b₂, b₃)`，则：
  - `A · B = a₁b₁ + a₂b₂ + a₃b₃`
- **推导依赖的运算律**：
  - 交换律：`A·B = B·A`
  - 分配律：`(A+B)·C = A·C + B·C`
  - 数乘结合律：`(λA)·B = λ(A·B)`
  - 基向量性质：`i·i = j·j = k·k = 1`；`i·j = j·k = k·i = 0`

### 1.4 核心应用
1.  **求夹角**：
    - `cosθ = (A · B) / (|A| * |B|)`
    - 其中 `|A| = √(a₁² + a₂² + a₃²)`
2.  **判垂直**：
    - `A ⟂ B` ⇔ `A · B = 0`
3.  **求模长**：
    - `|A|² = A · A`

### 1.5 例题实战
> **题目**：已知 M(1,1,1), A(2,2,1), B(2,1,2)，求 ∠AMB。
- **Step 1：求向量**
  - `MA = (2-1, 2-1, 1-1) = (1, 1, 0)`
  - `MB = (2-1, 1-1, 2-1) = (1, 0, 1)`
- **Step 2：套公式**
  - `MA · MB = 1*1 + 1*0 + 0*1 = 1`
  - `|MA| = √(1²+1²+0²) = √2`，`|MB| = √2`
  - `cos∠AMB = 1 / (√2 * √2) = 1/2`
- **Step 3：得结论**
  - ∠AMB = 60° = π/3

## 二、第二只鸡：向量积（叉积）—— 结果是一个向量

### 2.1 物理背景：力矩
- **模型**：用扳手拧螺丝。力 `F` 作用在点 P（力臂 `OP`），产生转动效果——力矩 `M`。
  - **大小**：`|M| = |OP| * |F| * sinθ` （θ 是 OP 与 F 的夹角，取垂直于力臂的分力）
  - **方向**：**右手规则**。四指从 `OP` 转向 `F`，拇指指向即为力矩方向。力矩方向垂直于 `OP` 与 `F` 所在平面。
- **数学抽象**：`M = OP × F` （注意顺序！）

### 2.2 数学定义
- `C = A × B` 是一个向量。
  - **模长**：`|C| = |A| * |B| * sinθ` （θ 是 A 与 B 的夹角）
  - **方向**：垂直于 A 和 B 所在的平面，指向由右手规则决定（四指从 A 转向 B，拇指指向）。

### 2.3 几何意义（超级重要！）
- **垂直性**：`A × B` 同时垂直于 `A` 和 `B`。
- **面积**：`|A × B|` 等于以 `A` 和 `B` 为邻边的平行四边形的面积。
  - 三角形面积 = `1/2 * |A × B|`

### 2.4 运算性质
- **反交换律**：`A × B = - (B × A)` （顺序很重要！）
- **分配律**：`A × (B + C) = A×B + A×C`；`(A+B)×C = A×C + B×C`
- **数乘结合律**：`(λA) × B = λ (A×B)`
- **零向量**：`A × A = 0` （平行向量叉积也为 0）
- **基向量叉积（右手系循环）**：
  - `i × j = k`，`j × k = i`，`k × i = j`
  - `j × i = -k`，`k × j = -i`，`i × k = -j`

### 2.5 坐标计算公式（行列式法）
若 `A = (a₁, a₂, a₃)`，`B = (b₁, b₂, b₃)`，则：
`A × B = | i    j    k; a₁ a₂ a₃; b₁ b₂ b₃ |` （三阶行列式）

#### 🚀 记忆口诀：前上后下写两遍，掐头去尾取中间
1.  **写两遍**：将 `A` 的坐标写在上行，`B` 的坐标写下行，各写两遍。
    ```
    A:  a₁  a₂  a₃  a₁  a₂
    B:  b₁  b₂  b₃  b₁  b₂
    ```
2.  **取中间**：结果向量的三个分量分别是这三组二阶行列式的值：
    - 第一分量（i前）：`|a₂ a₃; b₂ b₃| = a₂b₃ - a₃b₂` （掐掉 a₁, b₁）
    - 第二分量（j中）：`|a₃ a₁; b₃ b₁| = a₃b₁ - a₁b₃` （注意！是**负的**，或直接记公式为 `-(a₁b₃ - a₃b₁)`，但口诀方便）
      - 口诀实操：第二分量看 `a₃ a₁` 和 `b₃ b₁`，直接交叉相乘再相减：`a₃*b₁ - a₁*b₃`
    - 第三分量（k尾）：`|a₁ a₂; b₁ b₂| = a₁b₂ - a₂b₁`

### 2.6 例题实战
> **题目**：已知 `A=(2,1,-1)`，`B=(1,-1,2)`，求与 A、B 都垂直的**单位向量**。

- **Step 1：求叉积 `C = A × B`**
  - 写两遍：`2, 1, -1, 2, 1` 和 `1, -1, 2, 1, -1`
  - 第一分量：`|1 -1; -1 2| = 1*2 - (-1)*(-1) = 2 - 1 = 1`
  - 第二分量：`|-1 2; 2 1| = (-1)*1 - 2*2 = -1 - 4 = -5`
  - 第三分量：`|2 1; 1 -1| = 2*(-1) - 1*1 = -2 - 1 = -3`
  - 所以 `C = A × B = (1, -5, -3)`
  - （验证：A·C=2*1+1*(-5)+(-1)*(-3)=2-5+3=0 ✔）

- **Step 2：求模长**
  - `|C| = √(1² + (-5)² + (-3)²) = √(1+25+9) = √35`

- **Step 3：求单位向量**
  - 与 A、B 垂直的向量有两个方向：`C` 和 `-C`。
  - `e = ± (1/√35, -5/√35, -3/√35)`
  - **⚠️ 注意**：答案必须带 ±，否则不完整！

> **题目**：求三角形 ABC 面积，A(1,2,3), B(3,4,5), C(2,3,4)
- **Step 1：求两边向量**
  - `AB = (2,2,2)`
  - `AC = (1,2,4)`
- **Step 2：求叉积**
  - `AB × AC = ?` （用行列式或口诀）
  - 结果：`(2*4 - 2*2, -(2*4 - 2*1), 2*2 - 2*1) = (8-4, -(8-2), 4-2) = (4, -6, 2)`
- **Step 3：求面积**
  - `S_△ = 1/2 * |AB × AC| = 1/2 * √(4²+(-6)²+2²) = 1/2 * √(16+36+4) = 1/2 * √56 = √14`

## 三、第三只鸡：混合积 —— 结果是一个数

### 3.1 定义
- `[A B C] = (A × B) · C`
- 运算顺序：先叉乘得到向量，再点乘得到数。

### 3.2 几何意义
- **绝对值**：`| [A B C] |` 等于以 `A, B, C` 为棱的平行六面体的**体积**。
  - 底面积：`|A × B|`
  - 高：`|C| * |cosθ|` （C 在垂直于底面的方向上的投影）
- **符号意义**：`[A B C] > 0` 表示 `A, B, C` 构成**右手系**； `< 0` 为左手系。

### 3.3 坐标公式
若 `A=(a₁,a₂,a₃)`，`B=(b₁,b₂,b₃)`，`C=(c₁,c₂,c₃)`，则：
`[A B C] = | a₁ a₂ a₃; b₁ b₂ b₃; c₁ c₂ c₃ |` （三阶行列式的值）

### 3.4 核心性质与应用
1.  **轮换对称性**：向量按顺序循环，值不变。
    - `[A B C] = [B C A] = [C A B]`
2.  **两向量交换**：混合积变号（行列式两行交换）。
    - `[A B C] = -[A C B]`
3.  **判共面（超级重要！）**：
    - 三个向量 `A, B, C` 共面 ⇔ 混合积 `[A B C] = 0`
    - （因为平行六面体没体积了，三个向量躺在同一个平面里）
4.  **判四点共面**：
    - 对于点 M, A, B, C，取向量 `MA, MB, MC`，计算混合积。若为 0，则四点共面。

### 3.5 例题实战
> **题目**：已知 `[A B C] = 2`，求 `[(A-B)×(B+C)] · (C+A)`
- **Step 1：展开叉乘**
  - `(A-B)×(B+C) = A×B + A×C - B×B - B×C`
  - `B×B = 0`，所以原式 = `A×B + A×C - B×C`
- **Step 2：点乘 `(C+A)`**
  - 原式 = `(A×B)·C + (A×B)·A + (A×C)·C + (A×C)·A - (B×C)·C - (B×C)·A`
- **Step 3：利用混合积性质化简**
  - `(A×B)·C = [A B C] = 2`
  - `(A×B)·A = [A B A] = 0` （有重复向量，共面 ⇒ 体积0）
  - `(A×C)·C = [A C C] = 0`
  - `(A×C)·A = [A C A] = 0`
  - `(B×C)·C = [B C C] = 0`
  - `(B×C)·A = [B C A]`，根据轮换对称，`[B C A] = [A B C] = 2`，但这里是 `-(B×C)·A = -2`
- **Step 4：求和**
  - 原式 = 2 + 0 + 0 + 0 - 0 - 2 = **0**

> **题目**：四点共面问题。A(1,2,3), B(2,3,4), C(4,5,5), M(x,y,z) 四点共面，求 x,y,z 关系。
- **Step 1：选三个向量（尽量含 M 少一点）**
  - `AB = (1,1,1)`
  - `AC = (3,3,2)`
  - `AM = (x-1, y-2, z-3)`
- **Step 2：混合积 = 0 （共面条件）**
  - `| x-1  y-2  z-3; 1 1 1; 3 3 2 | = 0`
- **Step 3：计算行列式**
  - `(x-1)*(1*2 - 1*3) - (y-2)*(1*2 - 1*3) + (z-3)*(1*3 - 1*3)`
  - 化简：`(x-1)*(2-3) - (y-2)*(2-3) + (z-3)*(3-3)`
  - `= (x-1)*(-1) - (y-2)*(-1) + 0`
  - `= -x + 1 + y - 2 = y - x - 1 = 0`
- **Step 4：得平面方程**
  - `y - x = 1` （这就是 A, B, C 所在平面的方程）

## 📝 总结与记忆要点

| 运算 | 结果类型 | 核心公式（坐标） | 几何意义 | 关键判定 |
| :--- | :--- | :--- | :--- | :--- |
| **数量积 (·)** | **数** | `a₁b₁ + a₂b₂ + a₃b₃` | 投影的乘积 | 垂直 ⇔ 点积=0 |
| **向量积 (×)** | **向量** | 三阶行列式 / 口诀 | 面积（模长）、垂直向量 | 平行 ⇔ 叉积=0 |
| **混合积 ([ ])** | **数** | 三阶行列式 | 体积（绝对值） | 共面 ⇔ 混合积=0 |

- **点积是数，叉积是向量，混合积是数。**
- **叉积口诀**：“前上后下写两遍，掐头去尾取中间，交叉相乘再相减。”
- **右手规则**：贯穿叉积和混合积的方向判定，必须熟练掌握。
