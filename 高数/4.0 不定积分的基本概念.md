# 4.0 不定积分的基本概念

> 📌 **本节核心**：理解原函数与不定积分的定义，熟记基本积分公式表

---

## 一、原函数的概念

### 1.1 定义

设 $f(x)$ 是定义在区间 $I$ 上的函数。若存在函数 $F(x)$，使得对一切 $x \in I$，都有：

$$F'(x) = f(x) \quad \text{或} \quad dF(x) = f(x)dx$$

则称 $F(x)$ 是 $f(x)$ 在 $I$ 上的一个**原函数**。

### 1.2 原函数的存在性

**定理**：若 $f(x)$ 在区间 $I$ 上**连续**，则 $f(x)$ 在 $I$ 上一定存在原函数。

### 1.3 原函数的性质

- 若 $F(x)$ 是 $f(x)$ 的一个原函数，则 $F(x) + C$（$C$ 为任意常数）也是 $f(x)$ 的原函数。
- $f(x)$ 的**任意两个**原函数之间只差一个常数。

---

## 二、不定积分的定义

### 2.1 定义

$f(x)$ 的**全体原函数**称为 $f(x)$ 的**不定积分**，记作：

$$\int f(x)dx = F(x) + C$$

其中：
- $\int$ 称为**积分号**
- $f(x)$ 称为**被积函数**
- $f(x)dx$ 称为**被积表达式**
- $x$ 称为**积分变量**
- $C$ 称为**积分常数**（任意常数）

### 2.2 不定积分与导数的关系

$$\left[\int f(x)dx\right]' = f(x) \quad \text{（先积后导，还原）}$$

$$\int f'(x)dx = f(x) + C \quad \text{（先导后积，多一个常数）}$$

> 💡 **核心结论**：不定积分与求导是**互逆运算**。

---

## 三、不定积分的基本性质

### 性质 1：线性性

$$\int [kf(x) + lg(x)]dx = k\int f(x)dx + l\int g(x)dx$$

### 性质 2：积分与微分互逆

$$d\left[\int f(x)dx\right] = f(x)dx$$

$$\int dF(x) = F(x) + C$$

---

## 四、基本积分公式表 ⭐⭐⭐ 必背

| 序号 | 积分公式 |
|---|---|
| 1 | $\int 0\,dx = C$ |
| 2 | $\int k\,dx = kx + C$（$k$ 为常数） |
| 3 | $\int x^n dx = \frac{x^{n+1}}{n+1} + C$（$n \neq -1$） |
| 4 | $\int \frac{1}{x}dx = \ln|x| + C$ |
| 5 | $\int a^x dx = \frac{a^x}{\ln a} + C$（$a > 0, a \neq 1$） |
| 6 | $\int e^x dx = e^x + C$ |
| 7 | $\int \sin x\,dx = -\cos x + C$ |
| 8 | $\int \cos x\,dx = \sin x + C$ |
| 9 | $\int \sec^2 x\,dx = \tan x + C$ |
| 10 | $\int \csc^2 x\,dx = -\cot x + C$ |
| 11 | $\int \sec x\tan x\,dx = \sec x + C$ |
| 12 | $\int \csc x\cot x\,dx = -\csc x + C$ |
| 13 | $\int \frac{1}{\sqrt{1-x^2}}dx = \arcsin x + C$ |
| 14 | $\int \frac{1}{1+x^2}dx = \arctan x + C$ |

## 六、直接积分法

利用基本积分公式和线性性直接计算。

> **例 1**：$\int (2x^3 - 3x^2 + 4x - 5)dx$
> $$= 2 \cdot \frac{x^4}{4} - 3 \cdot \frac{x^3}{3} + 4 \cdot \frac{x^2}{2} - 5x + C = \frac{x^4}{2} - x^3 + 2x^2 - 5x + C$$

> **例 2**：$\int \frac{x^4}{1+x^2}dx$
> - 代数变形：$\frac{x^4}{1+x^2} = \frac{x^4 - 1 + 1}{1+x^2} = \frac{(x^2-1)(x^2+1) + 1}{1+x^2} = x^2 - 1 + \frac{1}{1+x^2}$
> $$= \int \left(x^2 - 1 + \frac{1}{1+x^2}\right)dx = \frac{x^3}{3} - x + \arctan x + C$$

> **例 3**：$\int \frac{1}{\sin^2 x \cos^2 x}dx$
> - 利用 $\sin^2 x + \cos^2 x = 1$：
> $$= \int \frac{\sin^2 x + \cos^2 x}{\sin^2 x \cos^2 x}dx = \int \left(\frac{1}{\cos^2 x} + \frac{1}{\sin^2 x}\right)dx = \tan x - \cot x + C$$

> **例 4**：$\int \sin^2 \frac{x}{2}dx$
> - 降幂：$\sin^2 \frac{x}{2} = \frac{1 - \cos x}{2}$
> $$= \int \frac{1 - \cos x}{2}dx = \frac{x}{2} - \frac{\sin x}{2} + C$$

## 七、凑微分法（第一类换元法）初步

$$\int f[\varphi(x)]\varphi'(x)dx = \int f(u)du \quad (u = \varphi(x))$$

**常见凑微分模式**：

| 被积函数特征 | 凑微分 |
|---|---|
| 含 $f(x^n) \cdot x^{n-1}$ | $x^{n-1}dx = \frac{1}{n}d(x^n)$ |
| 含 $f(e^x) \cdot e^x$ | $e^x dx = d(e^x)$ |
| 含 $f(\sin x) \cdot \cos x$ | $\cos x\,dx = d(\sin x)$ |
| 含 $f(\ln x) \cdot \frac{1}{x}$ | $\frac{1}{x}dx = d(\ln x)$ |
| 含 $f(\arctan x) \cdot \frac{1}{1+x^2}$ | $\frac{1}{1+x^2}dx = d(\arctan x)$ |

> **例 5**：$\int 2x e^{x^2}dx = \int e^{x^2}d(x^2) = e^{x^2} + C$

> **例 6**：$\int \frac{\cos x}{1+\sin^2 x}dx = \int \frac{d(\sin x)}{1+\sin^2 x} = \arctan(\sin x) + C$

## 八、学习建议

1. **基本积分公式必须背熟**，这是所有积分计算的地基
2. 积分是导数的逆运算，做积分时要"反着想"
3. 遇到复杂被积函数，先尝试代数变形（拆项、配凑、三角恒等变换）
4. 不忘积分常数 $C$
5. 求出结果后可以**对结果求导**来验证正确性
