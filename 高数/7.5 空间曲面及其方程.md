# 高等数学听课笔记：空间曲面及其方程（7.5）

## 一、空间曲面概述

### 1. 基本概念

**定义**：空间曲面是空间中满足特定关系(x, y, z)的点的集合

**本节重点**：二次曲面方程（最高次数为二次的三元方程）

---

### 2. 学习目标

1. **识别曲面类型**：给定方程能说出对应的曲面名称
2. **掌握截痕法**：能看懂一切空间曲面
3. **理解旋转曲面**：会写旋转曲面方程
4. **运用伸缩变换**：掌握伸缩变换技巧

---

## 二、截痕法

### 1. 截痕法的定义

**截痕**：用垂直于坐标轴的平面截取空间曲面，产生的交线

**核心思想**：降维——通过观察截痕的形状来理解整体曲面

---

### 2. 截痕法的应用

#### 椭球面的截痕

**标准方程**：x²/a² + y²/b² + z²/c² = 1

**截痕分析**：

- **垂直于Z轴截取**（z = z₀）：x²/a² + y²/b² = 常数 → 椭圆
- **垂直于X轴截取**（x = x₀）：y²/b² + z²/c² = 常数 → 椭圆
- **垂直于Y轴截取**（y = y₀）：x²/a² + z²/c² = 常数 → 椭圆

**结论**：平行于各坐标面的截痕都是椭圆，故称椭球面

---

#### 圆锥面的截痕

**标准方程**：z² = x²/a² + y²/b²

**截痕分析**：

- **垂直于Z轴截取**（z = z₀）：x² + y² = 常数 → 圆（等高线）
- **垂直于X轴截取**（x = 0）：z = ±y/a → 直线

**特点**：Z有平方，截痕呈现直线状，体现圆锥特点

---

#### 旋转抛物面的截痕

**标准方程**：z = x²/a² + y²/b²

**截痕分析**：

- **垂直于Z轴截取**（z = z₀）：x² + y² = 常数 → 圆
- **垂直于X轴截取**（x = 0）：z = y²/a² → 抛物线
- **垂直于Y轴截取**（y = 0）：z = x²/b² → 抛物线

**特点**：Z无平方，截痕呈现抛物线形状

---

#### 单叶双曲面的截痕

**标准方程**：x²/a² + y²/b² - z²/c² = 1（两正一负）

**截痕分析**：

- **垂直于Z轴截取**（z = z₀）：x²/a² + y²/b² = 常数 → 椭圆
- **垂直于Y轴截取**（y = y₀）：x²/a² - z²/c² = 常数 → 双曲线
- **垂直于X轴截取**（x = x₀）：y²/b² - z²/c² = 常数 → 双曲线

**连通性**：曲面上任意两点都能通过曲面连接，一片叶子连在一起

**实例**：广州小蛮腰

---

#### 双叶双曲面的截痕

**标准方程**：x²/a² - y²/b² - z²/c² = 1（一正两负）

**截痕分析**：

- **垂直于Z轴截取**（z = z₀）：x²/a² - y²/b² = 常数 → 双曲线
- **垂直于Y轴截取**（y = y₀）：x²/a² - z²/c² = 常数 → 双曲线
- **垂直于X轴截取**（x = x₀）：
  - |x₀| < a：无解（切不到）
  - |x₀| > a：y²/b² + z²/c² = 常数 → 椭圆

**连通性**：曲面分成两部分，不连通

---

#### 双曲抛物面（马鞍面）的截痕

**标准方程**：z = x²/a² - y²/b²

**截痕分析**：

- **垂直于Z轴截取**（z = z₀）：x²/a² - y²/b² = 常数 → 双曲线
- **垂直于Y轴截取**（y = y₀）：z = x²/a² + 常数 → 抛物线（开口向上）
- **垂直于X轴截取**（x = x₀）：z = -y²/b² + 常数 → 抛物线（开口向下）

**特点**：同时展现双曲和抛物的两重性

**鞍点**：既不是极大值也不是极小值的点

- 某个方向看是极小值
- 另一个方向看是极大值

---

### 3. 截痕法总结

| 曲面类型   | 垂直于Z轴截痕 | 垂直于X轴截痕 | 垂直于Y轴截痕 |
| ---------- | ------------- | ------------- | ------------- |
| 椭球面     | 椭圆          | 椭圆          | 椭圆          |
| 圆锥面     | 圆            | 直线          | 直线          |
| 旋转抛物面 | 圆            | 抛物线        | 抛物线        |
| 单叶双曲面 | 椭圆          | 双曲线        | 双曲线        |
| 双叶双曲面 | 双曲线        | 椭圆/无解     | 双曲线        |
| 双曲抛物面 | 双曲线        | 抛物线        | 抛物线        |

---

## 三、九种重要二次曲面

### 1. 球面

**标准方程**：(x - x₀)² + (y - y₀)² + (z - z₀)² = R²

**特点**：

- 球面上任意点到球心的距离相等
- X²、Y²、Z²的系数必须完全相同
- 不会出现交叉项（xy、yz、zx）

**识别方法**：

1. 平方项系数相同
2. 无交叉项
3. 可配方化成标准形式

**示例**：x² + y² + z² + 2x - 3y + 8z = 0（配方后为球面）

---

### 2. 椭球面

**标准方程**：x²/a² + y²/b² + z²/c² = 1

**特点**：

- 各轴长度不同（a ≠ b ≠ c时）
- 所有截痕都是椭圆
- 中心在原点

**特殊形式**：x²/a² + y²/a² + z²/c² = 1（旋转椭球面）

---

### 3. 圆锥面

**标准方程**：z² = x²/a² + y²/b²

**特点**：

- 顶点在原点
- 等高线为圆（当a = b时）
- 垂直于Z轴的截痕为圆

**其他形式**：z = ±√(x² + y²)/a

---

### 4. 旋转抛物面

**标准方程**：z = x²/a² + y²/b²

**特点**：

- 开口向上（假设z²系数为正）
- 垂直于X轴、Y轴的截痕为抛物线
- 中心轴为Z轴

---

### 5. 单叶双曲面

**标准方程**：x²/a² + y²/b² - z²/c² = 1（两正一负）

**特点**：

- 连通的曲面（一片叶子）
- 垂直于Z轴的截痕为椭圆
- 垂直于X轴、Y轴的截痕为双曲线

**记忆口诀**：两正一负，单叶双曲

---

### 6. 双叶双曲面

**标准方程**：x²/a² - y²/b² - z²/c² = 1（一正两负）

**特点**：

- 不连通（两片叶子）
- 垂直于Z轴、Y轴的截痕为双曲线
- 垂直于X轴的截痕：|x| < a无解，|x| > a为椭圆

**记忆口诀**：一正两负，双叶双曲

---

### 7. 椭圆抛物面

**标准方程**：z = x²/a² + y²/b²

**特点**：

- 由旋转抛物面伸缩而来
- 系数不同时为椭圆抛物面
- 系数相同时为旋转抛物面

---

### 8. 双曲抛物面（马鞍面）

**标准方程**：z = x²/a² - y²/b²

**特点**：

- 鞍点（极大值点和极小值点的混合）
- 无限延展，无边界
- 可表示为z = xy（通过可逆线性变换）

**变换示例**：z = xy → 令u = (x + y)/2, v = (x - y)/2 → z = u² - v²

---

### 9. 特殊形式

#### 球面变形

- **半球面**：z = √(1 - x² - y²)（只取上半部分）

#### 圆锥面变形

- **椭圆锥面**：z² = x²/a² + y²/b²（a ≠ b）

#### 抛物面变形

- **椭圆抛物面**：z = x²/a² + y²/b²（a ≠ b）

---

## 四、旋转曲面

### 1. 旋转曲面的定义

**定义**：平面上一条曲线（母线）绕着该平面内的一条直线（轴）旋转一周形成的曲面

**基本概念**：

- **母线**：负责不断复制自己的曲线
- **轴**：旋转轴（定直线）

---

### 2. 旋转曲面的构造原理

#### 纬圆概念

**纬圆**：母线上的点M₁绕轴旋转一周形成的圆周

**特点**：

- 纬圆所在平面垂直于旋转轴
- 圆心在旋转轴上

---

### 3. 旋转曲面方程的求法

#### 基本原理

**给定条件**：曲线L在YOZ平面上，方程为F(y, z) = 0，且x = 0

**绕Z轴旋转**：

1. Z坐标不变：z = z₁
2. 到Z轴的距离不变：√(x² + y²) = |y₁|
3. 代入方程：F(±√(x² + y²), z) = 0

---

#### 通用公式

**绕哪个轴转，哪个变量不变**

| 旋转轴 | 不变量 | 变换规则                         |
| ------ | ------ | -------------------------------- |
| X轴    | x      | y → ±√(y² + z²)                  |
| Y轴    | y      | x → ±√(x² + z²)                  |
| Z轴    | z      | x → ±√(x² + y²), y → ±√(x² + y²) |

**简化规则**：

- 绕X轴转：y² → y² + z²
- 绕Y轴转：x² → x² + z²
- 绕Z轴转：x² → x² + y², y² → x² + y²

---

### 4. 例题解析

#### 例1：椭圆绕轴旋转

**题目**：x² + y²/a² = 1，分别绕X轴、Y轴旋转

**绕X轴旋转**：

- X不变
- y² → y² + z²
- 结果：x² + (y² + z²)/a² = 1（椭球面）

**绕Y轴旋转**：

- Y不变
- x² → x² + z²
- 结果：(x² + z²)/a² + y²/a² = 1（椭球面）

---

#### 例2：指数函数绕轴旋转

**题目**：z = e^y（0 ≤ y ≤ 2），绕Z轴旋转

**解题步骤**：

1. Z不变
2. y → ±√(x² + y²)
3. 结果：z = e^(±√(x² + y²))

**注意**：由于y ∈ [0, 2]，只取正号

---

### 5. 旋转曲面的识别

**识别方法**：

- 找出系数相同的两项
- 删除多余项，得到平面曲线
- 确定旋转轴（不变的变量）

**示例**：z = x²/4 + y²/4

- 删除y²项：z = x²/4（抛物线）
- 绕Z轴旋转：旋转抛物面

---

## 五、伸缩变换

### 1. 伸缩变换的基本原理

**平面曲线的伸缩**：

以y = sin x为例：

| 变换        | 方程         | 图形变化           |
| ----------- | ------------ | ------------------ |
| 横向压缩2倍 | y = sin 2x   | 频率变快，周期减半 |
| 横向拉伸2倍 | y = sin(x/2) | 频率变慢，周期加倍 |

**核心规律**：

- **伸长k倍**：变量乘以1/k
- **压缩到1/k**：变量乘以k

---

### 2. 空间曲面的伸缩

**伸缩倍数的确定**：

#### 例：椭球面的伸缩

**从旋转椭球面到一般椭球面**：

- 旋转椭球面：x²/a² + y²/a² + z²/c² = 1
- 一般椭球面：x²/a² + y²/b² + z²/c² = 1

**变换过程**：

- y/a → y/b
- y被变换为：(b/a)y
- **伸缩倍数**：a/b倍（沿Y轴方向）

---

#### 例：圆锥面的伸缩

**从圆锥面到椭圆锥面**：

- 圆锥面：z² = x²/a² + y²/a²
- 椭圆锥面：z² = x²/a² + y²/b²

**变换过程**：

- y/a → y/b
- y被变换为：(b/a)y
- **伸缩倍数**：a/b倍（沿Y轴方向）

---

#### 例：椭圆抛物面的伸缩

**题目**：画出z = x²/4 + y²/9的曲面

**解题步骤**：

1. **识别为椭圆抛物面**
2. **先看成旋转抛物面**：z = x²/4（令y = 0）
3. **绕Z轴旋转**：z = (x² + y²)/4
4. **沿Y轴伸缩**：
   - y/2 → y/3
   - y被变换为：(2/3)y
   - **伸缩倍数**：3/2倍（沿Y轴方向伸长）

**作图**：

- 先画旋转抛物面（截面为圆）
- 沿Y轴方向伸长1.5倍
- 得到椭圆抛物面（截面为椭圆）

---

### 3. 伸缩变换总结

**变换公式**：

- 沿某轴伸缩k倍：该变量乘以1/k
- 反之，变量乘以k：沿该轴伸缩1/k倍

**识别方法**：

- 比较旋转曲面和一般曲面
- 找出系数变化的关系
- 计算伸缩倍数

---

## 六、柱面

### 1. 柱面的定义

**定义**：一条直线（母线）沿着一条定曲线（准线）平行移动形成的曲面

**基本概念**：

- **母线**：负责复制自己的直线
- **准线**：决定移动轨迹的曲线

---

### 2. 柱面的特点

#### 缺少变量

**特征**：柱面方程中至少缺少一个变量

| 缺少的变量 | 母线方向  |
| ---------- | --------- |
| 缺z        | 平行于Z轴 |
| 缺x        | 平行于X轴 |
| 缺y        | 平行于Y轴 |

**理解方式**：

- 平面曲线沿垂直于其所在平面的方向平移
- 所有的等高线形状相同

---

### 3. 三种基本柱面

#### 椭圆柱面

**方程**：x²/a² + y²/b² = 1（缺z）

**特点**：

- 在XY平面为准线（椭圆）
- 沿Z轴方向延伸
- 平行于Z轴的截痕为椭圆

---

#### 双曲柱面

**方程**：x²/a² - y²/b² = 1（缺z）

**特点**：

- 在XY平面为准线（双曲线）
- 沿Z轴方向延伸
- 平行于Z轴的截痕为双曲线

---

#### 抛物柱面

**方程**：y² = 2px（缺z）

**特点**：

- 在XY平面为准线（抛物线）
- 沿Z轴方向延伸
- 平行于Z轴的截痕为抛物线

---

### 4. 柱面的求法

#### 例题：求柱面方程

**题目**：求以椭球面与旋转抛物面的交线为准线，母线平行于Z轴的柱面方程

**已知**：

- 椭球面：x²/a² + y²/b² + z²/c² = 1
- 旋转抛物面：z = x² + y²

**解题步骤**：

1. **消去z**（平行于Z轴，消z）
2. 代入z = x² + y²到椭球面方程
3. 化简得：3(x² + y²)² + 2(x² + y²) - 1 = 0
4. 因式分解：3z² + 2z - 1 = 0
5. 解得：z = 1/3
6. 回代：x² + y² = 1/3

**结果**：x² + y² = 1/3（圆柱面）

---

## 七、曲面识别综合练习

### 1. 双叶双曲面

**方程**：z²/4 - x²/15 - y²/15 = 1

**识别步骤**：

1. 平方项：两正一负？否，一正两负
2. 右边：等于1（正数）
3. 结论：双叶双曲面

**特点**：

- 中心轴：Z轴
- 喇叭口朝上
- 垂直于Z轴截取可能无解

---

### 2. 圆柱面

**方程**：(y - 10)² + z² = 100（缺x）

**识别步骤**：

1. 缺少变量：x
2. 平面曲线：圆（配方后）
3. 结论：圆柱面

**特点**：

- 圆心：(0, 10, 0)
- 半径：10
- 母线平行于X轴

---

### 3. 椭圆抛物面

**方程**：y = x²/a² + z²/b²

**识别步骤**：

1. 右边：平方和
2. 左边：一次变量y
3. 结论：椭圆抛物面

**特点**：

- 中心轴：Y轴
- 开口沿Y轴正方向

---

## 八、曲面积分应用

### 1. 圆锥面上的积分

**题目**：在圆锥面z = √(x² + y²)上计算曲面积分（0 ≤ z ≤ 2）

**解题思路**：

1. **识别曲面**：圆锥面（只取上半部分）
2. **绘制图形**：有盖子的圆锥
3. **确定投影**：投到XY平面为圆
4. **计算投影区域**：
   - z = 2时：x² + y² = 4
   - 圆心：(0, 0)
   - 半径：2

---

### 2. 球面上的积分

**题目**：在半球面z = √(1 - x² - y²)上计算曲面积分

**解题思路**：

1. **识别曲面**：半球面
2. **完整球面方程**：x² + y² + z² = 1
3. **球心和半径**：
   - 球心：(0, 0, 0)
   - 半径：1
4. **确定范围**：上半球面

---

### 3. 组合曲面上的三重积分

**解题步骤**：

1. **识别各曲面**：
   - 球面：x² + y² + z² = 1
   - 圆锥面：z² = x² + y²
2. **绘制交线**：确定积分区域
3. **选择坐标系**：球坐标或柱坐标
4. **确定积分限**：根据曲面交线

---

## 九、重要总结

### 1. 九种二次曲面对比

| 曲面类型   | 标准方程                  | 特点           | 截痕特征      |
| ---------- | ------------------------- | -------------- | ------------- |
| 球面       | x² + y² + z² = R²         | 到球心距离相等 | 圆            |
| 椭球面     | x²/a² + y²/b² + z²/c² = 1 | 三轴不同       | 椭圆          |
| 圆锥面     | z² = x²/a² + y²/b²        | 顶点在原点     | 圆/直线       |
| 旋转抛物面 | z = x²/a² + y²/b²         | 开口向上       | 圆/抛物线     |
| 单叶双曲面 | x²/a² + y²/b² - z²/c² = 1 | 两正一负       | 椭圆/双曲线   |
| 双叶双曲面 | x²/a² - y²/b² - z²/c² = 1 | 一正两负       | 双曲线/椭圆   |
| 椭圆抛物面 | z = x²/a² + y²/b²         | 平方和         | 椭圆/抛物线   |
| 双曲抛物面 | z = x²/a² - y²/b²         | 平方差         | 双曲线/抛物线 |

---

### 2. 单叶与双叶双曲面的判断

**判断方法**：

- **单叶双曲面**：两正一负
- **双叶双曲面**：一正两负

**记忆口诀**：

- 两正一负，单叶双曲
- 一正两负，双叶双曲

---

### 3. 旋转曲面记忆要点

**核心口诀**：绕哪个轴转，哪个变量不变

**变换规则**：

- 绕X轴：y² → y² + z²
- 绕Y轴：x² → x² + z²
- 绕Z轴：x² → x² + y²

---

### 4. 伸缩变换记忆要点

**变换公式**：

- 伸长k倍：变量乘以1/k
- 压缩到1/k：变量乘以k

**判断方法**：

- 比较旋转曲面和一般曲面
- 计算变量变换关系
- 确定伸缩倍数

---

### 5. 柱面识别要点

**识别特征**：

- 缺少变量
- 母线平行于缺失变量对应的轴

**常见柱面**：

- 椭圆柱面：x²/a² + y²/b² = 1
- 双曲柱面：x²/a² - y²/b² = 1
- 抛物柱面：y² = 2px
