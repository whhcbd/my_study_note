# 📐 全微分 听课笔记

> 📌 **课程来源**：KR 老师高数下预习课
> 🎯 **核心目标**：理解全微分的概念，搞清可微、可导、连续之间的关系
> 💡 **重要性**：全微分是多元函数微分学的核心概念，关系图是考试必考的判断题

---

## 一、全微分的概念

### 1.1 从偏导数到全微分
- **偏导数**只反映沿坐标轴方向的变化率，是"**单方向的**"信息。
- **全微分**反映函数在一点附近**所有方向**的整体变化情况，是"**全方位的**"近似。

### 1.2 全增量
设 $z = f(x, y)$ 在 $(x_0, y_0)$ 附近有定义，当 $x, y$ 分别有增量 $\Delta x, \Delta y$ 时：
$$\Delta z = f(x_0 + \Delta x, y_0 + \Delta y) - f(x_0, y_0)$$
称为**全增量**。

### 1.3 可微的定义 ⭐核心
若全增量可以表示为：
$$\Delta z = A\Delta x + B\Delta y + o(\rho)$$
其中 $A, B$ 不依赖于 $\Delta x, \Delta y$，$\rho = \sqrt{(\Delta x)^2 + (\Delta y)^2}$，$o(\rho)$ 是比 $\rho$ 高阶的无穷小，则称 $f$ 在 $(x_0, y_0)$ 处**可微**。

此时称 $A\Delta x + B\Delta y$ 为 $f$ 在 $(x_0, y_0)$ 处的**全微分**，记作：
$$\mathrm{d}z = A\Delta x + B\Delta y = f_x(x_0, y_0)\mathrm{d}x + f_y(x_0, y_0)\mathrm{d}y$$

> 💡 **直觉**：全微分就是用**切平面**来近似代替曲面——在一点附近，曲面几乎就是平的。

---

## 二、可微的必要条件与充分条件

### 2.1 定理一（必要条件）
若 $f$ 在 $(x_0, y_0)$ 可微，则：
1. $f$ 在 $(x_0, y_0)$ 处**连续**
2. $f_x(x_0, y_0)$ 和 $f_y(x_0, y_0)$ 都**存在**
3. $A = f_x(x_0, y_0)$，$B = f_y(x_0, y_0)$

### 2.2 定理二（充分条件）⭐
若 $f_x$ 和 $f_y$ 在 $(x_0, y_0)$ 的某邻域内**连续**，则 $f$ 在 $(x_0, y_0)$ 处**可微**。

> 💡 **口诀**：偏导连续 $\Rightarrow$ 可微。**连续是桥梁！**

---

## 三、可微、可导、连续的关系 ⭐⭐⭐ 考点！

### 3.1 关系图

```
偏导数连续 ──→ 可微 ──→ 连续
              │  ↑
              ↓  │
         偏导数存在
```

| 箭头方向 | 含义 |
| :--- | :--- |
| 偏导数连续 → 可微 | **充分条件** |
| 可微 → 连续 | **必要条件** |
| 可微 → 偏导数存在 | **必要条件** |
| 可微 ↔ 偏导数存在 | ❌ 不可逆！偏导数存在 ≠ 可微 |
| 连续 → 偏导数存在 | ❌ 连续 ≠ 偏导数存在 |
| 偏导数存在 → 连续 | ❌ 不对！参考上章例题2 |

### 3.2 经典反例

> **反例 1**：偏导数存在但不可微

$$f(x, y) = \begin{cases} \frac{xy}{\sqrt{x^2 + y^2}}, & (x,y) \neq (0,0) \\ 0, & (x,y) = (0,0) \end{cases}$$

- $f_x(0,0) = 0$，$f_y(0,0) = 0$（偏导数存在）
- 但 $\Delta z - (0 \cdot \Delta x + 0 \cdot \Delta y) = \frac{\Delta x \cdot \Delta y}{\sqrt{(\Delta x)^2 + (\Delta y)^2}} \neq o(\rho)$
- 沿 $\Delta x = \Delta y$ 时为 $\frac{(\Delta x)^2}{\sqrt{2}|\Delta x|} = \frac{|\Delta x|}{\sqrt{2}}$，不趋于 0
- **结论**：不可微 ✅

> **反例 2**：偏导数存在但不连续（上章例题2）
$$f(x, y) = \begin{cases} \frac{xy}{x^2 + y^2}, & (x,y) \neq (0,0) \\ 0, & (x,y) = (0,0) \end{cases}$$
- $f_x(0,0) = 0$，$f_y(0,0) = 0$
- 但 $(0,0)$ 处不连续

## 四、全微分的计算

### 4.1 计算公式
$$\mathrm{d}z = \frac{\partial z}{\partial x}\mathrm{d}x + \frac{\partial z}{\partial y}\mathrm{d}y$$

> **例题 1**：求 $z = e^{xy} \sin(x + y)$ 的全微分。

- **Step 1：求偏导**
  - $z_x = ye^{xy}\sin(x+y) + e^{xy}\cos(x+y) = e^{xy}[y\sin(x+y) + \cos(x+y)]$
  - $z_y = xe^{xy}\sin(x+y) + e^{xy}\cos(x+y) = e^{xy}[x\sin(x+y) + \cos(x+y)]$
- **Step 2：写全微分**
  - $\mathrm{d}z = e^{xy}[y\sin(x+y) + \cos(x+y)]\mathrm{d}x + e^{xy}[x\sin(x+y) + \cos(x+y)]\mathrm{d}y$

### 4.2 全微分在近似计算中的应用

$$\Delta z \approx \mathrm{d}z = f_x(x_0, y_0)\Delta x + f_y(x_0, y_0)\Delta y$$
$$f(x_0 + \Delta x, y_0 + \Delta y) \approx f(x_0, y_0) + f_x(x_0, y_0)\Delta x + f_y(x_0, y_0)\Delta y$$

> **例题 2**：计算 $(1.04)^{2.02}$ 的近似值。

- **Step 1**：设 $f(x, y) = x^y$，取 $(x_0, y_0) = (1, 2)$，$\Delta x = 0.04$，$\Delta y = 0.02$
- **Step 2**：求偏导
  - $f_x = yx^{y-1}$，$f_x(1, 2) = 2 \cdot 1^1 = 2$
  - $f_y = x^y \ln x$，$f_y(1, 2) = 1^2 \cdot \ln 1 = 0$
- **Step 3**：代入近似公式
  - $(1.04)^{2.02} \approx f(1, 2) + f_x \cdot 0.04 + f_y \cdot 0.02 = 1 + 2 \times 0.04 + 0 = 1.08$

## 五、判断可微的步骤 ⭐考点

> **例题 3**：设 $f(x,y) = \begin{cases} (x^2 + y^2)\sin\frac{1}{\sqrt{x^2+y^2}}, & (x,y) \neq (0,0) \\ 0, & (x,y) = (0,0) \end{cases}$，判断 $f$ 在 $(0,0)$ 是否可微。

- **Step 1：求 $f_x(0,0)$（用定义）**
  - $f_x(0,0) = \lim_{\Delta x \to 0} \frac{f(\Delta x, 0) - f(0,0)}{\Delta x} = \lim_{\Delta x \to 0} \frac{(\Delta x)^2 \sin\frac{1}{|\Delta x|}}{\Delta x} = \lim_{\Delta x \to 0} \Delta x \sin\frac{1}{|\Delta x|} = 0$
- **Step 2：同理 $f_y(0,0) = 0$**
- **Step 3：检验 $\Delta z - [f_x(0,0)\Delta x + f_y(0,0)\Delta y]$ 是否为 $o(\rho)$**
  - $\Delta z = f(\Delta x, \Delta y) - f(0,0) = ((\Delta x)^2+(\Delta y)^2)\sin\frac{1}{\sqrt{(\Delta x)^2+(\Delta y)^2}}$
  - $\Delta z - 0 = \rho^2 \sin\frac{1}{\rho}$
  - $\frac{\Delta z - 0}{\rho} = \rho \sin\frac{1}{\rho} \to 0$（当 $\rho \to 0$）
- **结论**：$\Delta z - 0 = o(\rho)$，所以 $f$ 在 $(0,0)$ **可微** ✅

## 📝 总结与记忆要点

| 概念 | 核心结论 |
| :--- | :--- |
| **全微分公式** | $\mathrm{d}z = f_x\mathrm{d}x + f_y\mathrm{d}y$ |
| **偏导数连续 → 可微** | 充分条件（最常用的判断方法） |
| **可微 → 连续** | 必要条件 |
| **可微 → 偏导数存在** | 必要条件 |
| **偏导数存在 → 连续** | ❌ 错！ |
| **连续 → 偏导数存在** | ❌ 错！ |

- **口诀**：可微最强啥都推，偏导连续能到可微。其余箭头不可逆，反例记住就OK！
- **判断可微三步走**：① 求偏导（用定义）→ ② 算 $\Delta z - f_x\Delta x - f_y\Delta y$ → ③ 除以 $\rho$ 看是否趋于 0
