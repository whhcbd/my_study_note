# 定积分的应用

本文档系统总结了定积分在几何和物理方面的应用，包括平面图形面积、旋转体体积、弧长、变力做功等。

---

## 一、平面图形的面积

### 1. 直角坐标系下的面积

#### 函数图像与x轴围成的面积

若函数$f(x)$在$[a, b]$上可积，则曲线$y = f(x)$、直线$x = a$、$x = b$与x轴围成的面积为：

$$S = \int_{a}^{b} |f(x)| dx$$

**特别情况**：
- 当$f(x) \geq 0$时：$S = \int_{a}^{b} f(x) dx$
- 当$f(x) \leq 0$时：$S = -\int_{a}^{b} f(x) dx$

**例**：求抛物线$y = x^2$与直线$x = 0$、$x = 1$、y = 0围成的面积

解：由于在$[0, 1]$上$y = x^2 \geq 0$
$$S = \int_{0}^{1} x^2 dx = \left[\frac{x^3}{3}\right]_{0}^{1} = \frac{1}{3}$$

#### 两函数图像之间的面积

设$f(x) \geq g(x)$在$[a, b]$上，则两曲线之间的面积为：

$$S = \int_{a}^{b} [f(x) - g(x)] dx$$

**例**：求抛物线$y = x^2$与直线$y = x$围成的面积

解：求交点：$x^2 = x \Rightarrow x = 0$或$x = 1$

在$[0, 1]$上，$x \geq x^2$，故：
$$S = \int_{0}^{1} (x - x^2) dx = \left[\frac{x^2}{2} - \frac{x^3}{3}\right]_{0}^{1} = \frac{1}{2} - \frac{1}{3} = \frac{1}{6}$$

#### 以y为变量的面积

若以y为自变量，则面积为：

$$S = \int_{c}^{d} [\varphi(y) - \psi(y)] dy$$

其中$x = \varphi(y)$是右边曲线，$x = \psi(y)$是左边曲线

**例**：求抛物线$y = x^2$与直线$y = 4$围成的面积

解：从$y$的角度考虑，$x = \sqrt{y}$和$x = -\sqrt{y}$

$$S = \int_{0}^{4} [\sqrt{y} - (-\sqrt{y})] dy = 2\int_{0}^{4} \sqrt{y} dy = 2 \cdot \left[\frac{2}{3}y^{\frac{3}{2}}\right]_{0}^{4} = \frac{4}{3} \cdot 8 = \frac{32}{3}$$

### 2. 参数方程下的面积

设曲线的参数方程为$x = \varphi(t)$、$y = \psi(t)$，$t \in [\alpha, \beta]$，则面积为：

$$S = \int_{\alpha}^{\beta} |\psi(t) \cdot \varphi'(t)| dt$$

**例**：求椭圆$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$的面积

解：椭圆的参数方程为$x = a\cos t$，$y = b\sin t$，$t \in [0, 2\pi]$

$$S = 4 \int_{0}^{\frac{\pi}{2}} b\sin t \cdot (-a\sin t) dt = 4ab \int_{0}^{\frac{\pi}{2}} \sin^2 t dt$$

$$= 4ab \cdot \frac{\pi}{4} = \pi ab$$

### 3. 极坐标系下的面积

设曲线的极坐标方程为$r = r(\theta)$，$\theta \in [\alpha, \beta]$，则扇形面积为：

$$S = \frac{1}{2} \int_{\alpha}^{\beta} r^2(\theta) d\theta$$

**例**：求圆$r = 2a\cos\theta$的面积

解：$\theta$的变化范围是$[-\frac{\pi}{2}, \frac{\pi}{2}]$

$$S = \frac{1}{2} \int_{-\frac{\pi}{2}}^{\frac{\pi}{2}} (2a\cos\theta)^2 d\theta = 2a^2 \int_{-\frac{\pi}{2}}^{\frac{\pi}{2}} \cos^2\theta d\theta$$

$$= 2a^2 \cdot \frac{\pi}{2} = \pi a^2$$

## 二、旋转体的体积

### 1. 绕x轴旋转的体积

设曲线$y = f(x)$在$[a, b]$上非负，则绕x轴旋转所得旋转体的体积为：

$$V = \int_{a}^{b} \pi f^2(x) dx$$

**几何理解**：每个小薄片的体积为$\pi r^2 dx$，其中$r = f(x)$

**例**：求$y = x^2$、$x = 1$、$y = 0$围成的图形绕x轴旋转的体积

解：$$V = \pi \int_{0}^{1} (x^2)^2 dx = \pi \int_{0}^{1} x^4 dx = \pi \left[\frac{x^5}{5}\right]_{0}^{1} = \frac{\pi}{5}$$

### 2. 绕y轴旋转的体积

#### 以x为变量
$$V = 2\pi \int_{a}^{b} x \cdot f(x) dx$$

#### 以y为变量
设$x = g(y)$，$y \in [c, d]$，则：
$$V = \int_{c}^{d} \pi g^2(y) dy$$

**例**：求$y = x^2$、$x = 1$、$y = 0$围成的图形绕y轴旋转的体积

解：$x = \sqrt{y}$，$y \in [0, 1]$

$$V = \pi \int_{0}^{1} (\sqrt{y})^2 dy = \pi \int_{0}^{1} y dy = \pi \left[\frac{y^2}{2}\right]_{0}^{1} = \frac{\pi}{2}$$

### 3. 绕其他直线旋转的体积

设曲线$y = f(x)$绕直线$y = k$旋转的体积为：

$$V = \int_{a}^{b} \pi [f(x) - k]^2 dx$$

**例**：求$y = x^2$、$x = 1$、$y = 0$围成的图形绕直线$y = -1$旋转的体积

解：每个截面圆的半径为$r = x^2 - (-1) = x^2 + 1$

$$V = \pi \int_{0}^{1} (x^2 + 1)^2 dx = \pi \int_{0}^{1} (x^4 + 2x^2 + 1) dx$$

$$= \pi \left[\frac{x^5}{5} + \frac{2x^3}{3} + x\right]_{0}^{1} = \pi\left(\frac{1}{5} + \frac{2}{3} + 1\right) = \frac{28\pi}{15}$$

### 4. 旋转体体积的其他方法（圆柱壳法）

以$y$轴为旋转轴时，也可用圆柱壳法：

$$V = 2\pi \int_{a}^{b} x \cdot f(x) dx$$

**例**：用圆柱壳法求$y = x^2$、$x = 1$、$y = 0$绕y轴旋转的体积

解：$$V = 2\pi \int_{0}^{1} x \cdot x^2 dx = 2\pi \int_{0}^{1} x^3 dx = 2\pi \left[\frac{x^4}{4}\right]_{0}^{1} = \frac{\pi}{2}$$

## 三、平面曲线的弧长

### 1. 直角坐标系下的弧长公式

设曲线方程为$y = f(x)$，$x \in [a, b]$，则弧长为：

$$L = \int_{a}^{b} \sqrt{1 + [f'(x)]^2} dx$$

**推导**：由微积分基本公式，$ds = \sqrt{dx^2 + dy^2} = \sqrt{1 + (y')^2} dx$

**例**：求抛物线$y = x^2$在$x = 0$到$x = 1$之间的弧长

解：$y' = 2x$

$$L = \int_{0}^{1} \sqrt{1 + (2x)^2} dx = \int_{0}^{1} \sqrt{1 + 4x^2} dx$$

令$2x = \tan t$，则$dx = \frac{1}{2}\sec^2 t dt$

$$L = \frac{1}{2} \int_{0}^{\arctan 2} \sec^3 t dt$$

## 四、旋转面的面积

### 1. 绕x轴旋转的表面积

设曲线$y = f(x)$在$[a, b]$上绕x轴旋转，则表面积为：

$$A = \int_{a}^{b} 2\pi y \cdot ds = \int_{a}^{b} 2\pi f(x) \cdot \sqrt{1 + [f'(x)]^2} dx$$

**几何理解**：每个小条带的表面积为$2\pi y \cdot ds$

**例**：求$y = x$在$[0, 1]$上绕x轴旋转的表面积

解：$f(x) = x$，$f'(x) = 1$

$$A = \int_{0}^{1} 2\pi x \cdot \sqrt{2} dx = 2\sqrt{2}\pi \left[\frac{x^2}{2}\right]_{0}^{1} = \sqrt{2}\pi$$

### 2. 绕y轴旋转的表面积

$$A = \int_{a}^{b} 2\pi x \cdot \sqrt{1 + [f'(x)]^2} dx$$

## 五、物理应用

### 1. 变力做功

若力$F(x)$随位置x变化，则从$a$到$b$所做的功为：

$$W = \int_{a}^{b} F(x) dx$$

**例**：弹簧被拉长x米时，拉力$F = kx$（k为常数），求弹簧从原长拉长a米所做的功

解：$$W = \int_{0}^{a} kx dx = k\left[\frac{x^2}{2}\right]_{0}^{a} = \frac{ka^2}{2}$$

### 2. 液体压力

若液体密度为$\rho$，深度为h处的压强为$p = \rho gh$（g为重力加速度），则液体对垂直平板的压力为：

$$P = \int_{a}^{b} \rho g h \cdot \text{面积微元}$$

**例**：求水深为$H$的垂直矩形板$[a, b] \times [c, d]$受到的压力

解：设在深度h处的水平条带高度为$dh$，宽度为$d - c$

$$P = \int_{c}^{d} \rho g h \cdot (d - c) dh$$

### 3. 引力计算

根据万有引力定律，两质点间的引力为$F = G\frac{m_1 m_2}{r^2}$

对于连续分布的物体，需要用积分计算引力。

## 六、定积分的应用步骤

### 1. 几何应用的一般步骤

1. **画图分析**：明确图形的边界和形状
2. **确定积分变量**：选择x或y作为积分变量
3. **建立积分表达式**：
   - 面积：确定上函数和下函数
   - 体积：确定半径或高度函数
   - 弧长：计算$y'$或$x'$
4. **确定积分限**：找到积分变量的取值范围
5. **计算积分**：运用适当的积分方法计算

### 2. 物理应用的一般步骤

1. **分析物理模型**：理解问题的物理意义
2. **建立微元**：将问题分解为微小的物理量
3. **确定函数关系**：建立物理量与位置的关系
4. **确定积分限**：根据物理条件确定积分范围
5. **计算积分**：求解得到结果

## 七、典型例题详解

### 1. 求平面图形面积

**例**：求$y = \sin x$、$y = \cos x$在$[0, \frac{\pi}{2}]$上围成的面积

解：求交点：$\sin x = \cos x \Rightarrow x = \frac{\pi}{4}$

在$[0, \frac{\pi}{4}]$上，$\sin x \leq \cos x$
在$[\frac{\pi}{4}, \frac{\pi}{2}]$上，$\sin x \geq \cos x$

$$S = \int_{0}^{\frac{\pi}{4}} (\cos x - \sin x) dx + \int_{\frac{\pi}{4}}^{\frac{\pi}{2}} (\sin x - \cos x) dx$$

$$= [\sin x + \cos x]_{0}^{\frac{\pi}{4}} + [-\cos x - \sin x]_{\frac{\pi}{4}}^{\frac{\pi}{2}}$$

$$= (\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2} - 1) + (-1 + \frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2})$$

$$= (\sqrt{2} - 1) + (\sqrt{2} - 1) = 2\sqrt{2} - 2$$

### 2. 求旋转体体积

**例**：求$y = \sqrt{1 - x^2}$、$x = 0$、$y = 0$围成的图形绕x轴旋转的体积

解：这是单位圆在第一象限的部分

$$V = \pi \int_{0}^{1} (\sqrt{1 - x^2})^2 dx = \pi \int_{0}^{1} (1 - x^2) dx$$

$$= \pi \left[x - \frac{x^3}{3}\right]_{0}^{1} = \pi\left(1 - \frac{1}{3}\right) = \frac{2\pi}{3}$$

### 3. 求变力做功

**例**：要把水从底面半径为r、高为h的圆锥形容器中抽出，求需要做的功

解：设在深度x处的截面半径为$R(x)$，由相似三角形：$\frac{R(x)}{r} = \frac{h - x}{h}$

故$R(x) = \frac{r(h - x)}{h}$

设在深度x处厚度为$dx$的水层质量为$dm = \pi R^2(x) \rho dx$

将这层水移出水面需要做的功：$dW = \rho g \pi R^2(x) \cdot x \cdot dx$

$$W = \rho g \pi \int_{0}^{h} \frac{r^2(h - x)^2}{h^2} \cdot x dx$$

令$t = h - x$，$dx = -dt$

$$W = \rho g \pi \frac{r^2}{h^2} \int_{h}^{0} t^2 \cdot (h - t)(-dt) = \rho g \pi \frac{r^2}{h^2} \int_{0}^{h} (ht^2 - t^3) dt$$

$$= \rho g \pi \frac{r^2}{h^2} \left[\frac{ht^3}{3} - \frac{t^4}{4}\right]_{0}^{h} = \rho g \pi \frac{r^2}{h^2}\left(\frac{h^4}{3} - \frac{h^4}{4}\right)$$

$$= \rho g \pi r^2 h \cdot \frac{1}{12} = \frac{1}{12}\rho g \pi r^2 h$$

## 八、常见易错点

### 1. 面积计算时忽略绝对值

❌ 认为：$\int_{a}^{b} f(x) dx$就是面积
✅ 正确：面积是$\int_{a}^{b} |f(x)| dx$，特别是当$f(x)$有正有负时

### 2. 旋转体积中半径确定错误

❌ 认为：绕y轴旋转时，半径总是x
✅ 正确：绕直线$y = k$旋转时，半径是$f(x) - k$，不是$f(x) - 0$

### 3. 弧长公式混淆

❌ 认为：弧长公式是$\int \sqrt{(dx)^2 + (dy)^2}$
✅ 正确：弧长公式是$\int \sqrt{1 + (y')^2} dx$或$\int \sqrt{(x')^2 + 1} dy$

### 4. 物理应用中单位问题

❌ 认为：力的单位、位移单位可以不一致
✅ 正确：注意单位统一，特别是重力加速度$g$的单位

## 九、解题技巧总结

### 1. 几何应用技巧
- **选择合适的积分变量**：有时以y为变量更简单
- **利用对称性**：对称图形可以计算一部分再乘倍数
- **分解复杂图形**：将复杂图形分解为简单图形的组合

### 2. 旋转体体积技巧
- **根据旋转轴选择方法**：绕坐标轴旋转用圆盘法，其他轴可能用圆环法
- **注意函数的正负**：半径必须为正，取绝对值
- **验证结果**：体积应该为正数

### 3. 物理应用技巧
- **理解物理原理**：先理解物理意义再建立数学模型
- **选择合适的微元**：微元选择要合理，便于计算
- **检查物理意义**：结果要符合物理直觉

## 十、记忆要点

- **表面积**：$A = \int_{a}^{b} 2\pi f(x) \cdot \sqrt{1 + [f'(x)]^2} dx$
- **变力做功**：$W = \int_{a}^{b} F(x) dx$
- **极坐标面积**：$S = \frac{1}{2} \int_{\alpha}^{\beta} r^2(\theta) d\theta$
