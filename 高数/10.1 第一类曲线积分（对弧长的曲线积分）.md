# 10.1 第一类曲线积分（对弧长的曲线积分）

> 📌 **本节核心**：理解第一类曲线积分的定义、物理意义，掌握其计算方法

---

## 一、第一类曲线积分的引入

### 1. 实际问题：曲线形构件的质量

设有一根细曲线 $L$，线密度为 $\rho(x, y)$（单位长度的质量），求曲线的总质量。

💡 **解决思路**：
- **分割**：将曲线分成 $n$ 小段，每段长度 $\Delta s_i$
- **近似**：每小段质量 $\approx \rho(\xi_i, \eta_i) \Delta s_i$
- **求和**：总质量 $\approx \sum_{i=1}^{n} \rho(\xi_i, \eta_i) \Delta s_i$
- **取极限**：$\lim_{\lambda \to 0} \sum_{i=1}^{n} \rho(\xi_i, \eta_i) \Delta s_i$

---

## 二、第一类曲线积分的定义

设 $L$ 是平面上一条光滑曲线，$f(x, y)$ 在 $L$ 上有界。

若极限

$$\lim_{\lambda \to 0} \sum_{i=1}^{n} f(\xi_i, \eta_i) \Delta s_i$$

存在且与分割和取点方式无关，则称此极限为 $f(x, y)$ 在曲线 $L$ 上**对弧长的曲线积分**（或**第一类曲线积分**），记作：

$$\int_L f(x, y) \, ds$$

其中：
- $f(x, y)$：**被积函数**
- $L$：**积分曲线**
- $ds$：**弧长元素**

### 空间曲线的第一类曲线积分

类似地，对空间曲线 $\Gamma$：

$$\int_\Gamma f(x, y, z) \, ds$$

---

## 三、第一类曲线积分的性质

### 性质1：线性性

$$\int_L [k_1 f(x, y) + k_2 g(x, y)] \, ds = k_1 \int_L f(x, y) \, ds + k_2 \int_L g(x, y) \, ds$$

### 性质2：路径可加性

若曲线 $L = L_1 + L_2$，则：

$$\int_L f(x, y) \, ds = \int_{L_1} f(x, y) \, ds + \int_{L_2} f(x, y) \, ds$$

### 性质3：与路径方向无关

$$\int_L f(x, y) \, ds = \int_{L^-} f(x, y) \, ds$$

其中 $L^-$ 表示 $L$ 的反向曲线。

💡 **重要特征**：第一类曲线积分与积分路径的方向无关！

## 四、第一类曲线积分的计算

### 1. 平面曲线（参数方程）

设曲线 $L$ 的参数方程为：

$$\begin{cases}
x = \varphi(t) \\
y = \psi(t)
\end{cases}, \quad \alpha \leq t \leq \beta$$

则：

$$\int_L f(x, y) \, ds = \int_\alpha^\beta f(\varphi(t), \psi(t)) \sqrt{[\varphi'(t)]^2 + [\psi'(t)]^2} \, dt$$

🎯 **计算口诀**：
- **Step 1**：代入参数方程 $x = \varphi(t), y = \psi(t)$
- **Step 2**：计算弧长元素 $ds = \sqrt{(x')^2 + (y')^2} \, dt$
- **Step 3**：化为定积分

### 2. 平面曲线（直角坐标方程）

若曲线 $L$ 的方程为 $y = y(x)$，$a \leq x \leq b$，则：

$$\int_L f(x, y) \, ds = \int_a^b f(x, y(x)) \sqrt{1 + [y'(x)]^2} \, dx$$

### 3. 空间曲线（参数方程）

设空间曲线 $\Gamma$ 的参数方程为：

$$\begin{cases}
x = x(t) \\
y = y(t) \\
z = z(t)
\end{cases}, \quad \alpha \leq t \leq \beta$$

则：

$$\int_\Gamma f(x, y, z) \, ds = \int_\alpha^\beta f(x(t), y(t), z(t)) \sqrt{[x'(t)]^2 + [y'(t)]^2 + [z'(t)]^2} \, dt$$

## 五、例题实战

### 📝 例1：直线段上的积分

计算 $\int_L (x + y) \, ds$，其中 $L$ 是从点 $(0, 0)$ 到点 $(1, 1)$ 的直线段。

**解**：

**Step 1**：参数化曲线

直线方程：$y = x$，$0 \leq x \leq 1$

或参数方程：$x = t, y = t$，$0 \leq t \leq 1$

**Step 2**：计算弧长元素

$$ds = \sqrt{1 + (y')^2} \, dx = \sqrt{1 + 1} \, dx = \sqrt{2} \, dx$$

或

$$ds = \sqrt{(x')^2 + (y')^2} \, dt = \sqrt{1 + 1} \, dt = \sqrt{2} \, dt$$

**Step 3**：化为定积分

$$\int_L (x + y) \, ds = \int_0^1 (x + x) \sqrt{2} \, dx = \sqrt{2} \int_0^1 2x \, dx$$

$$= \sqrt{2} \cdot x^2\bigg|_0^1 = \sqrt{2}$$

✅ **答案**：$\sqrt{2}$

---

### 📝 例2：圆弧上的积分

计算 $\int_L xy \, ds$，其中 $L$ 是圆 $x^2 + y^2 = a^2$ 在第一象限的部分。

**解**：

**Step 1**：参数化曲线

$$x = a\cos t, \quad y = a\sin t, \quad 0 \leq t \leq \frac{\pi}{2}$$

**Step 2**：计算弧长元素

$$x' = -a\sin t, \quad y' = a\cos t$$

$$ds = \sqrt{(x')^2 + (y')^2} \, dt = \sqrt{a^2\sin^2 t + a^2\cos^2 t} \, dt = a \, dt$$

**Step 3**：化为定积分

$$\int_L xy \, ds = \int_0^{\pi/2} (a\cos t)(a\sin t) \cdot a \, dt = a^3 \int_0^{\pi/2} \sin t \cos t \, dt$$

$$= a^3 \int_0^{\pi/2} \frac{1}{2}\sin 2t \, dt = \frac{a^3}{2} \cdot \left(-\frac{1}{2}\cos 2t\right)\bigg|_0^{\pi/2}$$

$$= \frac{a^3}{4} \left[-\cos\pi + \cos 0\right] = \frac{a^3}{4} \cdot 2 = \frac{a^3}{2}$$

✅ **答案**：$\frac{a^3}{2}$

---

### 📝 例3：空间曲线积分

计算 $\int_\Gamma (x^2 + y^2 + z^2) \, ds$，其中 $\Gamma$ 是螺旋线 $x = a\cos t, y = a\sin t, z = bt$，$0 \leq t \leq 2\pi$。

**解**：

**Step 1**：计算弧长元素

$$x' = -a\sin t, \quad y' = a\cos t, \quad z' = b$$

$$ds = \sqrt{a^2\sin^2 t + a^2\cos^2 t + b^2} \, dt = \sqrt{a^2 + b^2} \, dt$$

**Step 2**：代入被积函数

$$x^2 + y^2 + z^2 = a^2\cos^2 t + a^2\sin^2 t + b^2t^2 = a^2 + b^2t^2$$

**Step 3**：化为定积分

$$\int_\Gamma (x^2 + y^2 + z^2) \, ds = \int_0^{2\pi} (a^2 + b^2t^2) \sqrt{a^2 + b^2} \, dt$$

$$= \sqrt{a^2 + b^2} \left[a^2 t + \frac{b^2 t^3}{3}\right]_0^{2\pi}$$

$$= \sqrt{a^2 + b^2} \left(2\pi a^2 + \frac{8\pi^3 b^2}{3}\right)$$

✅ **答案**：$\sqrt{a^2 + b^2} \left(2\pi a^2 + \frac{8\pi^3 b^2}{3}\right)$

---

### 📝 例4：利用对称性

计算 $\int_L y \, ds$，其中 $L$ 是圆 $x^2 + y^2 = R^2$。

**解**：

圆关于 $x$ 轴对称，被积函数 $f(x, y) = y$ 关于 $y$ 是奇函数。

根据对称性：

$$\int_L y \, ds = 0$$

✅ **答案**：$0$

💡 **对称性技巧**：
- 若积分曲线关于某轴对称，被积函数关于该变量是奇函数，则积分为0
- 若被积函数是偶函数，可利用对称性简化计算

## 六、核心公式速查表

| 曲线类型 | 参数方程 | 计算公式 |
|---|---|---|
| **平面曲线（参数）** | $x = \varphi(t), y = \psi(t)$ | $\int_L f(x, y) \, ds = \int_\alpha^\beta f(\varphi(t), \psi(t)) \sqrt{(\varphi')^2 + (\psi')^2} \, dt$ |
| **平面曲线（直角）** | $y = y(x), a \leq x \leq b$ | $\int_L f(x, y) \, ds = \int_a^b f(x, y(x)) \sqrt{1 + (y')^2} \, dx$ |
| **空间曲线（参数）** | $x = x(t), y = y(t), z = z(t)$ | $\int_\Gamma f(x, y, z) \, ds = \int_\alpha^\beta f(x(t), y(t), z(t)) \sqrt{(x')^2 + (y')^2 + (z')^2} \, dt$ |

## 七、常见题型总结

| 题型 | 方法 | 注意事项 |
|---|---|---|
| 参数方程曲线 | 代入参数，计算 $ds$ | 弧长元素公式 |
| 直角坐标曲线 | $ds = \sqrt{1 + (y')^2} \, dx$ | 注意积分限 |
| 对称性问题 | 利用奇偶性 | 判断对称轴和函数性质 |
| 分段曲线 | 分段计算后相加 | 路径可加性 |
