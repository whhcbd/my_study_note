# 导数应用专题

本文档系统总结了导数在函数性态分析、极值问题、实际应用等方面的应用。

---

## 一、函数的单调性

### 1. 单调性的判定定理

**定理**：设函数$f(x)$在区间$I$上可导

1. 若$f'(x) > 0$，$\forall x \in I$，则$f(x)$在$I$上严格递增
2. 若$f'(x) < 0$，$\forall x \in I$，则$f(x)$在$I$上严格递减
3. 若$f'(x) \geq 0$，$\forall x \in I$，则$f(x)$在$I$上单调递增（非严格）
4. 若$f'(x) \leq 0$，$\forall x \in I$，则$f(x)$在$I$上单调递减（非严格）

### 2. 应用步骤

1. 求导$f'(x)$
2. 解$f'(x) > 0$得到递增区间
3. 解$f'(x) < 0$得到递减区间
4. 临界点$f'(x) = 0$可能是极值点

### 3. 典型例题

**例**：讨论函数$f(x) = x^3 - 3x + 2$的单调性

解：

1. 求导：$f'(x) = 3x^2 - 3 = 3(x^2 - 1)$

2. 解不等式：
   - $f'(x) > 0 \Rightarrow 3(x^2 - 1) > 0 \Rightarrow x^2 > 1 \Rightarrow x < -1$或$x > 1$
   - $f'(x) < 0 \Rightarrow 3(x^2 - 1) < 0 \Rightarrow x^2 < 1 \Rightarrow -1 < x < 1$

3. 结论：
   - $f(x)$在$(-\infty, -1)$上递增
   - $f(x)$在$(-1, 1)$上递减
   - $f(x)$在$(1, +\infty)$上递增

## 二、函数的极值

### 1. 极值的概念

**极值点**：函数在某点附近取得最大值或最小值

- **极大值**：存在$\delta > 0$，使得$\forall x \in (x_0 - \delta, x_0 + \delta)$，有$f(x) \leq f(x_0)$
- **极小值**：存在$\delta > 0$，使得$\forall x \in (x_0 - \delta, x_0 + \delta)$，有$f(x) \geq f(x_0)$

### 2. 极值的必要条件（费马定理）

**定理**：若$f(x)$在$x_0$处可导且取得极值，则$f'(x_0) = 0$

**说明**：$f'(x_0) = 0$的点称为**驻点**，但驻点不一定是极值点

### 3. 极值的充分条件

#### 第一充分条件（一阶导数判别法）

设$f(x)$在$x_0$的某邻域内可导（可允许$f'(x_0) = 0$）：

1. 若$f'(x)$在$x_0$左右两侧由正变负，则$x_0$是极大值点
2. 若$f'(x)$在$x_0$左右两侧由负变正，则$x_0$是极小值点
3. 若$f'(x)$在$x_0$左右两侧不变号，则$x_0$不是极值点

#### 第二充分条件（二阶导数判别法）

设$f(x)$在$x_0$处二阶可导，且$f'(x_0) = 0$：

1. 若$f''(x_0) > 0$，则$x_0$是极小值点
2. 若$f''(x_0) < 0$，则$x_0$是极大值点
3. 若$f''(x_0) = 0$，需用第一充分条件或更高阶导数判断

### 4. 求极值的步骤

1. 求$f'(x)$
2. 解$f'(x) = 0$，得到驻点
3. 用一阶或二阶导数判别法判断每个驻点是否为极值点
4. 计算极值$f(x_0)$

### 5. 典型例题

**例**：求函数$f(x) = x^3 - 6x^2 + 9x + 1$的极值

解：

1. 求导：$f'(x) = 3x^2 - 12x + 9 = 3(x^2 - 4x + 3)$

2. 解$f'(x) = 0$：
   $$3(x^2 - 4x + 3) = 0 \Rightarrow x = 1, x = 3$$

3. 求二阶导数：$f''(x) = 6x - 12$

4. 判断：
   - $f''(1) = 6 - 12 = -6 < 0$，故$x = 1$是极大值点
   - $f''(3) = 18 - 12 = 6 > 0$，故$x = 3$是极小值点

5. 计算极值：
   - 极大值：$f(1) = 1 - 6 + 9 + 1 = 5$
   - 极小值：$f(3) = 27 - 54 + 27 + 1 = 1$

## 三、函数的凹凸性与拐点

### 1. 凹凸性的定义

**凹函数**：对于任意$x_1, x_2 \in I$和$\lambda \in (0, 1)$，有
$$f(\lambda x_1 + (1-\lambda)x_2) \leq \lambda f(x_1) + (1-\lambda)f(x_2)$$

几何意义：曲线上任意两点的连线位于曲线上方

**凸函数**：对于任意$x_1, x_2 \in I$和$\lambda \in (0, 1)$，有
$$f(\lambda x_1 + (1-\lambda)x_2) \geq \lambda f(x_1) + (1-\lambda)f(x_2)$$

几何意义：曲线上任意两点的连线位于曲线下方

### 2. 凹凸性的判定

**定理**：设$f(x)$在区间$I$上二阶可导

1. 若$f''(x) > 0$，$\forall x \in I$，则$f(x)$在$I$上是凹函数
2. 若$f''(x) < 0$，$\forall x \in I$，则$f(x)$在$I$上是凸函数

### 3. 拐点的概念

**拐点**：函数凹凸性改变的点，即$f''(x)$变号的点

**拐点的必要条件**：若$(x_0, f(x_0))$是拐点，且$f''(x_0)$存在，则$f''(x_0) = 0$

### 4. 求拐点的步骤

1. 求$f''(x)$
2. 解$f''(x) = 0$得到可能拐点的x坐标
3. 判断$f''(x)$在这些点左右两侧是否变号
4. 若变号，则对应点为拐点

### 5. 典型例题

**例**：求函数$f(x) = x^4 - 4x^3 + 6x^2$的凹凸区间和拐点

解：

1. 求导：$f'(x) = 4x^3 - 12x^2 + 12x$

2. 求二阶导数：$f''(x) = 12x^2 - 24x + 12 = 12(x^2 - 2x + 1) = 12(x-1)^2$

3. 解$f''(x) = 0$：
   $$12(x-1)^2 = 0 \Rightarrow x = 1$$

4. 分析：
   - $f''(x) = 12(x-1)^2 \geq 0$，函数始终是凹的
   - 在$x = 1$处，$f''(x)$不变号，故不是拐点

5. 结论：
   - $f(x)$在$(-\infty, +\infty)$上都是凹的
   - 无拐点

## 四、函数的最大最小值问题

### 1. 闭区间上的最值

**定理**：连续函数在闭区间$[a, b]$上必有最大值和最小值

### 2. 求最值的方法

1. 找出$f(x)$在$(a, b)$内的所有驻点（$f'(x) = 0$）
2. 计算函数在端点和驻点的函数值
3. 比较所有函数值，最大者为最大值，最小者为最小值

### 3. 开区间上的最值

**方法**：

- 分析函数的单调性
- 通过极限判断最值是否存在

**例**：求$f(x) = x + \frac{1}{x}$在$(0, +\infty)$上的最小值

解：

1. 求导：$f'(x) = 1 - \frac{1}{x^2}$

2. 解$f'(x) = 0$：
   $$1 - \frac{1}{x^2} = 0 \Rightarrow x = 1$$

3. 二阶导数：$f''(x) = \frac{2}{x^3} > 0$（当$x > 0$）

4. 结论：$x = 1$是极小值点，最小值为$f(1) = 2$

## 五、函数作图

### 1. 函数作图的步骤

1. **确定定义域**：找出函数有意义的x的范围
2. **判断奇偶性和周期性**：
   - 奇函数：$f(-x) = -f(x)$，图像关于原点对称
   - 偶函数：$f(-x) = f(x)$，图像关于y轴对称
   - 周期函数：$f(x + T) = f(x)$
3. **求一阶导数**：确定单调性和极值
4. **求二阶导数**：确定凹凸性和拐点
5. **求渐近线**：水平、垂直、斜渐近线
6. **找特殊点**：与坐标轴的交点、极值点、拐点等
7. **绘制草图**

### 2. 渐近线的求法

#### 水平渐近线

若$\lim_{x \to \infty} f(x) = A$（或$\lim_{x \to -\infty} f(x) = A$），则$y = A$是水平渐近线

**例**：$f(x) = \frac{1}{x}$
$$\lim_{x \to \infty} \frac{1}{x} = 0$$
故$y = 0$是水平渐近线

#### 垂直渐近线

若$\lim_{x \to x_0} f(x) = \pm \infty$，则$x = x_0$是垂直渐近线

**例**：$f(x) = \frac{1}{x - 1}$
$$\lim_{x \to 1} \frac{1}{x - 1} = \infty$$
故$x = 1$是垂直渐近线

#### 斜渐近线

若$\lim_{x \to \infty} \frac{f(x)}{x} = k$（非零常数），且$\lim_{x \to \infty} [f(x) - kx] = b$，则$y = kx + b$是斜渐近线

**例**：$f(x) = \frac{x^2 + 1}{x} = x + \frac{1}{x}$
$$k = \lim_{x \to \infty} \frac{x^2 + 1}{x^2} = 1$$
$$b = \lim_{x \to \infty} \left(\frac{x^2 + 1}{x} - x\right) = \lim_{x \to \infty} \frac{1}{x} = 0$$
故$y = x$是斜渐近线

### 3. 函数作图例题

**例**：作函数$f(x) = x^3 - 3x^2 + 2$的图像

解：

1. 定义域：$(-\infty, +\infty)$

2. 奇偶性：$f(-x) = -x^3 - 3x^2 + 2 \neq f(x)$，既非奇函数也非偶函数

3. 一阶导数：$f'(x) = 3x^2 - 6x = 3x(x - 2)$
   - 驻点：$x = 0, x = 2$
   - 单调性：
     - $x < 0$时，$f'(x) > 0$，递增
     - $0 < x < 2$时，$f'(x) < 0$，递减
     - $x > 2$时，$f'(x) > 0$，递增

4. 二阶导数：$f''(x) = 6x - 6 = 6(x - 1)$
   - 拐点：$x = 1$
   - 凹凸性：
     - $x < 1$时，$f''(x) < 0$，凸的
     - $x > 1$时，$f''(x) > 0$，凹的

5. 极值：
   - $x = 0$：$f(0) = 2$，由增变减，极大值
   - $x = 2$：$f(2) = -2$，由减变增，极小值

6. 拐点：$(1, 0)$

7. 特殊点：$f(0) = 2$，$f(1) = 0$，$f(3) = 2$

## 六、洛必达法则

### 1. 基本形式

#### $\frac{0}{0}$型

若$\lim_{x \to x_0} \frac{f(x)}{g(x)}$为$\frac{0}{0}$型，且满足洛必达条件，则：
$$\lim_{x \to x_0} \frac{f(x)}{g(x)} = \lim_{x \to x_0} \frac{f'(x)}{g'(x)}$$

#### $\frac{\infty}{\infty}$型

若$\lim_{x \to x_0} \frac{f(x)}{g(x)}$为$\frac{\infty}{\infty}$型，则：
$$\lim_{x \to x_0} \frac{f(x)}{g(x)} = \lim_{x \to x_0} \frac{f'(x)}{g'(x)}$$

### 2. 其他不定型的处理

#### $0 \cdot \infty$型

$$f(x) \cdot g(x) = \frac{f(x)}{\frac{1}{g(x)}} \quad \text{或} \quad \frac{g(x)}{\frac{1}{f(x)}}$$

#### $\infty - \infty$型

通分化为分式形式

#### $1^\infty, 0^0, \infty^0$型

取对数后化为$0 \cdot \infty$型

### 3. 典型例题

**例**：求$\lim_{x \to 0} \frac{e^x - 1 - x}{x^2}$

解：这是$\frac{0}{0}$型

$$\lim_{x \to 0} \frac{e^x - 1 - x}{x^2} = \lim_{x \to 0} \frac{e^x - 1}{2x} = \lim_{x \to 0} \frac{e^x}{2} = \frac{1}{2}$$

## 七、实际应用问题

### 1. 几何应用

#### 面积最大化问题

**例**：用长为L的篱笆围一个矩形菜园，问长宽各为多少时面积最大？

解：设长为$x$，宽为$y$，则：
$$2x + 2y = L \Rightarrow y = \frac{L - 2x}{2}$$

面积：$S = x \cdot y = x \cdot \frac{L - 2x}{2} = \frac{Lx - 2x^2}{2}$

求导：$S' = \frac{L - 4x}{2}$

解$S' = 0$：$x = \frac{L}{4}$

此时$y = \frac{L - 2 \cdot \frac{L}{4}}{2} = \frac{L}{4}$

结论：当长宽相等时（正方形），面积最大。

#### 距离最小化问题

**例**：求点$P(1, 1)$到抛物线$y = x^2$的最短距离

解：设抛物线上点为$(x, x^2)$

距离的平方：$D^2 = (x - 1)^2 + (x^2 - 1)^2$

求导：$\frac{dD^2}{dx} = 2(x - 1) + 2(x^2 - 1) \cdot 2x = 2(x - 1) + 4x(x^2 - 1)$

令导数为零：$2(x - 1) + 4x(x + 1)(x - 1) = 0$

提取公因式：$2(x - 1)(2x^2 + 2x + 1) = 0$

因$2x^2 + 2x + 1$的判别式$\Delta = 4 - 8 < 0$，故无实根，因此唯一实根为$x = 1$

最短距离：$D = \sqrt{(1 - 1)^2 + (1 - 1)^2} = 0$

验证：点$(1, 1)$本身就在抛物线$y = x^2$上，故最短距离为$0$

### 2. 物理应用

#### 速度与加速度

**例**：物体运动位移为$s(t) = t^3 - 6t^2 + 9t$，求物体在何时速度为零？

解：
速度：$v(t) = s'(t) = 3t^2 - 12t + 9$

令$v(t) = 0$：$3t^2 - 12t + 9 = 0$

解得：$t = 1$或$t = 3$

### 3. 经济应用

#### 利润最大化

**例**：某商品的需求函数为$Q = 20 - P$（Q为需求量，P为价格），成本函数为$C = 2Q + 10$，求利润最大化时的价格和最大利润。

解：
利润：$L = R - C = P \cdot Q - C = (20 - Q)Q - (2Q + 10)$

$L(Q) = 20Q - Q^2 - 2Q - 10 = 18Q - Q^2 - 10$

求导：$L'(Q) = 18 - 2Q$

令$L'(Q) = 0$：$Q = 9$

此时价格：$P = 20 - 9 = 11$

最大利润：$L = 18 \times 9 - 9^2 - 10 = 162 - 81 - 10 = 71$

## 八、常见易错点

### 1. 极值与最值混淆

❌ 认为：极大值一定比极小值大
✅ 正确：极值是局部概念，最值是全局概念

### 2. 驻点与极值点混淆

❌ 认为：$f'(x_0) = 0$则$x_0$是极值点
✅ 正确：$f'(x_0) = 0$的点是驻点，驻点不一定是极值点

### 3. 凹凸性判断错误

❌ 认为：$f''(x) > 0$时函数是凸的
✅ 正确：$f''(x) > 0$时函数是凹的，$f''(x) < 0$时函数是凸的

### 4. 忽略定义域

❌ 认为：求极值时只需解$f'(x) = 0$
✅ 正确：还需考虑定义域边界和不可导点

## 九、解题技巧总结

### 1. 函数性态分析流程

1. 求一阶导数，确定单调区间和极值点
2. 求二阶导数，确定凹凸区间和拐点
3. 找特殊点（与坐标轴交点、极值点、拐点）
4. 求渐近线（水平、垂直、斜）
5. 绘制函数图像

### 2. 最值问题解题步骤

1. 建立数学模型（列函数关系式）
2. 求导找驻点
3. 判断极值性质
4. 结合实际意义确定答案

### 3. 洛必达法则应用技巧

1. 先确认极限类型（$\frac{0}{0}$或$\frac{\infty}{\infty}$）
2. 可结合等价无穷小替换简化计算
3. 注意洛必达法则的适用条件
4. 失效时考虑其他方法

## 十、记忆要点

- **单调性**：$f'(x) > 0$递增，$f'(x) < 0$递减
- **极值判别**：一阶导数变号或二阶导数判断
- **凹凸性**：$f''(x) > 0$凹，$f''(x) < 0$凸
- **拐点**：凹凸性改变的点
- **最值**：闭区间比较端点和驻点函数值
