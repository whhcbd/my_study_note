# 分部积分法（分步积分法）完整详解

分部积分法是微积分中核心的积分计算方法，专门用于求解**两个不同类型初等函数相乘**的积分问题，本质是函数乘积求导法则的逆运算，核心逻辑是把难算的积分转化为易算的积分。

---

## 一、核心公式与严格推导

### 1. 基础公式

设函数 $u=u(x)$、$v=v(x)$ 具有连续导数，分部积分公式有两种等价形式：

- **微分形式（最常用）**：
  $$\boldsymbol{\int u\,dv = uv - \int v\,du}$$

- **导数形式**：
  $$\int u(x)v'(x)dx = u(x)v(x) - \int u'(x)v(x)dx$$

- **定积分形式**（带上下限）：
  $$\boldsymbol{\int_{a}^{b} u\,dv = uv\bigg|_{a}^{b} - \int_{a}^{b} v\,du}$$

### 2. 公式推导

从乘积的求导法则出发，对可导函数 $u(x)$、$v(x)$，有：

$$(uv)' = u'v + uv'$$

将等式移项，得到：

$$uv' = (uv)' - u'v$$

对等式两边同时求不定积分：

$$\int uv'dx = \int (uv)'dx - \int u'vdx$$

根据微积分基本定理，$\int (uv)'dx = uv + C$，省略积分常数（最终结果统一加），整理后即得分部积分公式。

---

## 二、标准分步解题步骤

使用分部积分法，严格遵循以下4步即可完成计算，核心难点在第1步的$u$和$dv$选择：

1. **拆分被积函数，选定$u$和$dv$**

   将被积表达式拆分为 $u$ 和 $dv$ 两部分：$u$ 是一个函数，$dv$ 是剩余部分与$dx$的组合。

2. **计算辅助量$du$和$v$**

   对$u$求导得 $du=u'(x)dx$，对$dv$积分得 $v=\int dv$（积分常数可省略）。

3. **代入公式，转化积分**

   将$u、dv、du、v$代入核心公式，把原积分转化为 $uv - \int v\,du$ 的形式。

4. **计算新积分，得到最终结果**

   求解转化后的$\int v\,du$，补充积分常数$C$，完成原积分计算。

---

## 三、核心关键：$u$与$dv$的选择规则

$u$和$dv$的选择直接决定积分能否顺利求解，选错会导致积分更复杂，甚至无法计算。

### 1. 通用选择口诀

最经典、最易用的优先级口诀：**反对幂指三**

口诀对应5类基本初等函数，优先级从高到低，**优先级高的优先选作$u$，剩余部分全部凑成$dv$**：

| 口诀顺序 | 函数类型 | 典型例子 | 选择逻辑 |
|----------|----------|----------|----------|
| 反 | 反三角函数 | $\arcsin x、\arctan x、\arccos x$ | 求导后变为代数函数，复杂度大幅降低，无法凑成$dv$（积分无初等表达式） |
| 对 | 对数函数 | $\ln x、\log_a x$ | 求导后变为幂函数，复杂度降低，同样无法直接凑$dv$ |
| 幂 | 幂函数（代数函数） | $x、x^2、x^n$ | 求导后次数降低，多次求导可变为常数，简化积分 |
| 指 | 指数函数 | $e^x、a^x$ | 积分/求导形式不变，适合凑$dv$ |
| 三 | 三角函数 | $\sin x、\cos x$ | 积分/求导循环，适合凑$dv$ |

补充：也有"反对幂三指"的口诀，指数函数和三角函数优先级接近，二者组合时可任选其一为$u$，最终都需要循环积分求解。

### 2. 选择的核心原则

- 选作$u$的函数：求导后形式更简单，复杂度降低；
- 选作$dv$的部分：积分后容易求出$v$，且$\int v\,du$ 比原积分 $\int u\,dv$ 更易计算。

## 四、分类型典型例题（分步拆解）

### 类型1：幂函数 × 指数/三角函数（最基础）

核心规则：幂函数选作$u$，指数/三角函数凑$dv$，多次分部可将幂函数降为常数。

**例1：计算 $\int x e^x dx$**

1. 选$u=x$，$dv=e^x dx$（符合"幂>指"优先级）；
2. 求导积分：$du=dx$，$v=\int e^x dx=e^x$；
3. 代入公式：$\int x e^x dx = x e^x - \int e^x dx$；
4. 计算新积分：$x e^x - e^x + C$。

**例2：计算 $\int x \cos x dx$**

1. 选$u=x$，$dv=\cos x dx$（幂>三）；
2. $du=dx$，$v=\int \cos x dx = \sin x$；
3. 代入公式：$\int x \cos x dx = x \sin x - \int \sin x dx$；
4. 计算新积分：$x \sin x + \cos x + C$。

### 类型2：反三角/对数函数 × 幂函数

核心规则：反三角函数/对数函数选作$u$，幂函数凑$dv$。

**例3：计算 $\int \ln x dx$**

1. 选$u=\ln x$，$dv=dx$（对>幂）；
2. $du=\frac{1}{x}dx$，$v=x$；
3. 代入公式：$\int \ln x dx = x\ln x - \int x\cdot\frac{1}{x}dx$；
4. 计算新积分：$x\ln x - \int 1\,dx = x\ln x - x + C$。

**例4：计算 $\int x \arctan x dx$**

1. 选$u=\arctan x$，$dv=x dx$（反>幂）；
2. $du=\frac{1}{1+x^2}dx$，$v=\frac{x^2}{2}$；
3. 代入公式：$\int x \arctan x dx = \frac{x^2}{2}\arctan x - \int \frac{x^2}{2}\cdot\frac{1}{1+x^2}dx$；
4. 化简新积分：$\frac{x^2}{2}\arctan x - \frac{1}{2}\int \frac{x^2}{1+x^2}dx = \frac{x^2}{2}\arctan x - \frac{1}{2}\int (1-\frac{1}{1+x^2})dx$；
5. 计算结果：$\frac{x^2}{2}\arctan x - \frac{x}{2} + \frac{1}{2}\arctan x + C = \frac{x^2+1}{2}\arctan x - \frac{x}{2} + C$。

### 类型3：指数函数 × 三角函数（循环积分）

核心规则：任选其一为$u$，经过2次分部积分后会出现原积分，解方程即可。

**例5：计算 $\int e^x \cos x dx$**

1. 选$u=e^x$，$dv=\cos x dx$；
2. $du=e^x dx$，$v=\sin x$；
3. 代入公式：$\int e^x \cos x dx = e^x \sin x - \int e^x \sin x dx$；
4. 对$\int e^x \sin x dx$再分部一次：
   - 选$u=e^x$，$dv=\sin x dx$；
   - $du=e^x dx$，$v=-\cos x$；
   - $\int e^x \sin x dx = -e^x \cos x + \int e^x \cos x dx$；
5. 代入原式：$\int e^x \cos x dx = e^x \sin x - (-e^x \cos x + \int e^x \cos x dx)$；
6. 整理得：$\int e^x \cos x dx = e^x \sin x + e^x \cos x - \int e^x \cos x dx$；
7. 移项得：$2\int e^x \cos x dx = e^x (\sin x + \cos x)$；
8. 最终结果：$\int e^x \cos x dx = \frac{e^x}{2}(\sin x + \cos x) + C$。

### 类型4：高次幂函数 × 指数/三角函数（多次分部）

核心规则：需要多次应用分部积分，直到幂函数降为0次。

**例6：计算 $\int x^2 e^x dx$**

1. 选$u=x^2$，$dv=e^x dx$；
2. $du=2x dx$，$v=e^x$；
3. 代入公式：$\int x^2 e^x dx = x^2 e^x - \int 2x e^x dx$；
4. 对$\int x e^x dx$再分部一次：
   - 选$u=x$，$dv=e^x dx$；
   - $du=dx$，$v=e^x$；
   - $\int x e^x dx = x e^x - \int e^x dx = x e^x - e^x$；
5. 代入原式：$\int x^2 e^x dx = x^2 e^x - 2(x e^x - e^x) + C$；
6. 最终结果：$x^2 e^x - 2x e^x + 2e^x + C = e^x(x^2 - 2x + 2) + C$。

## 五、常见陷阱与注意事项

### 1. 忘记积分常数

分部积分的中间步骤可以省略常数，但最终结果必须加上$C$。

### 2. u和dv选择错误

选择错误的$u$和$dv$会导致积分越来越复杂。例如计算$\int x \ln x dx$时：
- 正确：$u=\ln x$，$dv=x dx$ → 求导后$\frac{1}{x}$，积分后$\frac{x^2}{2}$
- 错误：$u=x$，$dv=\ln x dx$ → $\ln x$无法积分

### 3. 循环积分时忘记解方程

对于指数×三角函数类型，会出现原积分，需要解方程而不是继续分部。

### 4. 定积分的边界项处理

使用定积分形式时，边界项$uv\big|_{a}^{b}$不要遗漏。

### 5. 特殊技巧：降阶法

对于$\int e^{ax}\sin(bx)dx$和$\int e^{ax}\cos(bx)dx$，可用降阶法简化计算。

---

## 六、快速解题检查清单

在应用分部积分法时，快速检查以下要点：

- [ ] 确认被积函数是两个不同类型函数的乘积
- [ ] 根据"反对幂指三"口诀正确选择$u$和$dv$
- [ ] 正确计算$du$和$v$（注意$v$的积分常数可省略）
- [ ] 代入公式时符号不要出错
- [ ] 若出现循环积分，记得解方程
- [ ] 最终结果加上积分常数$C$
- [ ] 定积分记得计算边界项$uv\big|_{a}^{b}$

---

## 七、典型应用场景

分部积分法适用于以下场景：

1. **乘积型积分**：两个不同类型初等函数相乘
2. **对数/反三角函数积分**：单独的$\ln x$、$\arcsin x$等
3. **高阶多项式与指数/三角函数**：需要多次分部降次
4. **某些定积分的对称性应用**：利用分部积分简化计算

---

## 八、总结

分部积分法的核心要点：

- **公式**：$\int u\,dv = uv - \int v\,du$
- **口诀**：反对幂指三（$u$的选择优先级）
- **关键**：正确选择$u$和$dv$，使转化后的积分更简单
- **技巧**：循环积分类型需要解方程，高次幂需要多次分部
- **验证**：对结果求导可检验正确性

掌握分部积分法需要大量练习，建议从基础的幂×指数/三角函数开始，逐步过渡到反三角/对数函数和循环积分类型。
