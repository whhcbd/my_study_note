# 积分技巧详解

本文档系统总结了各种积分方法和技巧，包括换元积分法、分部积分法、特殊函数积分等。

---

## 一、换元积分法

### 1. 第一类换元法（凑微分法）

#### 基本原理
若$\int f(x)dx = F(x) + C$，则：
$$\int f[\varphi(x)]\varphi'(x)dx = F[\varphi(x)] + C$$

#### 常见的凑微分形式

#### 幂函数凑微分
$$\int x^{n-1} f(x^n)dx = \frac{1}{n}\int f(u)du \quad (\text{其中} u = x^n)$$

**例**：$\int x \cdot e^{x^2} dx$

解：令$u = x^2$，$du = 2x dx$

$$\int x \cdot e^{x^2} dx = \frac{1}{2}\int e^{u} du = \frac{1}{2}e^{x^2} + C$$

#### 指数函数凑微分
$$\int e^x f(e^x)dx = \int f(u)du \quad (\text{其中} u = e^x)$$

**例**：$\int \frac{e^x}{1 + e^{2x}} dx$

解：令$u = e^x$，$du = e^x dx$

$$\int \frac{e^x}{1 + e^{2x}} dx = \int \frac{1}{1 + u^2} du = \arctan e^x + C$$

#### 三角函数凑微分
$$\int \cos x \cdot f(\sin x)dx = \int f(u)du \quad (\text{其中} u = \sin x)$$
$$\int \sin x \cdot f(\cos x)dx = -\int f(u)du \quad (\text{其中} u = \cos x)$$

**例**：$\int \frac{\sin x}{\cos^2 x} dx$

解：令$u = \cos x$，$du = -\sin x dx$

$$\int \frac{\sin x}{\cos^2 x} dx = -\int \frac{1}{u^2} du = \frac{1}{\cos x} + C$$

#### 根式凑微分
$$\int \frac{1}{\sqrt{x}} f(\sqrt{x})dx = 2\int f(u)du \quad (\text{其中} u = \sqrt{x})$$

**例**：$\int \frac{\ln x}{x} dx$

解：令$u = \ln x$，$du = \frac{1}{x} dx$

$$\int \frac{\ln x}{x} dx = \int u du = \frac{1}{2}\ln^2 x + C$$

### 2. 第二类换元法

#### 基本原理
若$x = \varphi(t)$，$\varphi'(t) \neq 0$，则：
$$\int f(x)dx = \int f[\varphi(t)]\varphi'(t)dt$$

#### 三角代换

#### 含$\sqrt{a^2 - x^2}$的积分
令$x = a\sin t$，则$\sqrt{a^2 - x^2} = a\cos t$

**例**：$\int \sqrt{a^2 - x^2} dx$

解：令$x = a\sin t$，$dx = a\cos t dt$

$$\int \sqrt{a^2 - x^2} dx = \int a\cos t \cdot a\cos t dt = a^2\int \cos^2 t dt$$

$$= a^2 \int \frac{1 + \cos 2t}{2} dt = a^2\left(\frac{t}{2} + \frac{\sin 2t}{4}\right) + C$$

$$= \frac{a^2}{2}\arcsin\frac{x}{a} + \frac{x}{2}\sqrt{a^2 - x^2} + C$$

#### 含$\sqrt{a^2 + x^2}$的积分
令$x = a\tan t$，则$\sqrt{a^2 + x^2} = a\sec t$

**例**：$\int \frac{1}{\sqrt{a^2 + x^2}} dx$

解：令$x = a\tan t$，$dx = a\sec^2 t dt$

$$\int \frac{1}{\sqrt{a^2 + x^2}} dx = \int \frac{a\sec^2 t}{a\sec t} dt = \int \sec t dt$$

$$= \ln|\sec t + \tan t| + C = \ln|x + \sqrt{a^2 + x^2}| + C$$

#### 含$\sqrt{x^2 - a^2}$的积分
令$x = a\sec t$，则$\sqrt{x^2 - a^2} = a\tan t$

**例**：$\int \frac{1}{\sqrt{x^2 - a^2}} dx$

解：令$x = a\sec t$，$dx = a\sec t \tan t dt$

$$\int \frac{1}{\sqrt{x^2 - a^2}} dx = \int \frac{a\sec t \tan t}{a\tan t} dt = \int \sec t dt$$

$$= \ln|\sec t + \tan t| + C = \ln|x + \sqrt{x^2 - a^2}| + C$$

#### 根式代换

#### 含$\sqrt[n]{ax + b}$的积分
令$t = \sqrt[n]{ax + b}$，则$x = \frac{t^n - b}{a}$，$dx = \frac{nt^{n-1}}{a} dt$

**例**：$\int \frac{1}{1 + \sqrt{x}} dx$

解：令$t = \sqrt{x}$，$x = t^2$，$dx = 2t dt$

$$\int \frac{1}{1 + \sqrt{x}} dx = \int \frac{2t}{1 + t} dt = 2\int \frac{t}{1 + t} dt$$

$$= 2\int \left(1 - \frac{1}{1 + t}\right) dt = 2(t - \ln|1 + t|) + C$$

$$= 2(\sqrt{x} - \ln|1 + \sqrt{x}|) + C$$

## 二、分部积分法

### 1. 基本原理

$$\int u \, dv = uv - \int v \, du$$

### 2. 应用法则

#### 反对幂三指
按"反对幂三指"顺序选择u：
- **反**：反三角函数
- **对**：对数函数  
- **幂**：幂函数
- **三**：三角函数
- **指**：指数函数

前面的函数选为u，后面的凑为dv

### 3. 典型应用

#### 对数函数与幂函数的乘积
**例**：$\int x \ln x dx$

解：$u = \ln x$，$dv = x dx$

$$\int x \ln x dx = \frac{x^2}{2}\ln x - \int \frac{x^2}{2} \cdot \frac{1}{x} dx$$

$$= \frac{x^2}{2}\ln x - \frac{1}{2}\int x dx = \frac{x^2}{2}\ln x - \frac{x^2}{4} + C$$

#### 反三角函数与幂函数的乘积
**例**：$\int x \arctan x dx$

解：$u = \arctan x$，$dv = x dx$

$$\int x \arctan x dx = \frac{x^2}{2}\arctan x - \int \frac{x^2}{2} \cdot \frac{1}{1 + x^2} dx$$

$$= \frac{x^2}{2}\arctan x - \frac{1}{2}\int \left(1 - \frac{1}{1 + x^2}\right) dx$$

$$= \frac{x^2}{2}\arctan x - \frac{x}{2} + \frac{1}{2}\arctan x + C$$

$$= \frac{x^2 + 1}{2}\arctan x - \frac{x}{2} + C$$

#### 三角函数与指数函数的乘积
**例**：$\int e^x \sin x dx$

解：使用两次分部积分

设$I = \int e^x \sin x dx$

第一次：$u = \sin x$，$dv = e^x dx$

$$I = e^x \sin x - \int e^x \cos x dx$$

第二次：$u = \cos x$，$dv = e^x dx$

$$I = e^x \sin x - \left(e^x \cos x - \int e^x (-\sin x) dx\right)$$

$$I = e^x \sin x - e^x \cos x - I$$

$$2I = e^x (\sin x - \cos x)$$

$$I = \frac{e^x}{2}(\sin x - \cos x) + C$$

#### 幂函数与指数函数的乘积
**例**：$\int x e^{ax} dx$（$a \neq 0$）

解：$u = x$，$dv = e^{ax} dx$，$du = dx$，$v = \frac{e^{ax}}{a}$

$$\int x e^{ax} dx = x \cdot \frac{e^{ax}}{a} - \int \frac{e^{ax}}{a} dx$$

$$= \frac{x e^{ax}}{a} - \frac{1}{a} \cdot \frac{e^{ax}}{a} + C$$

$$= \frac{e^{ax}}{a}\left(x - \frac{1}{a}\right) + C$$

## 三、有理函数积分

### 1. 部分分式分解

有理函数$\frac{P(x)}{Q(x)}$可分解为部分分式之和：

#### 真分式（$\deg P < \deg Q$）
1. 分解分母为不可约多项式的乘积
2. 每个一次因子$(x - a)^k$对应$k$项：$\frac{A_1}{x - a} + \frac{A_2}{(x - a)^2} + \cdots + \frac{A_k}{(x - a)^k}$
3. 每个二次因子$(x^2 + px + q)^k$对应$k$项：$\frac{B_1 x + C_1}{x^2 + px + q} + \frac{B_2 x + C_2}{(x^2 + px + q)^2} + \cdots$

#### 假分式（$\deg P \geq \deg Q$）
先用多项式除法化为多项式与真分式之和

### 2. 典型例题

#### 简单有理函数
**例**：$\int \frac{x + 1}{x^2 - 5x + 6} dx$

解：分母分解：$x^2 - 5x + 6 = (x - 2)(x - 3)$

设$\frac{x + 1}{(x - 2)(x - 3)} = \frac{A}{x - 2} + \frac{B}{x - 3}$

$$x + 1 = A(x - 3) + B(x - 2)$$

取$x = 2$：$3 = -A$，故$A = -3$
取$x = 3$：$4 = B$，故$B = 4$

$$\int \frac{x + 1}{x^2 - 5x + 6} dx = \int \left(\frac{-3}{x - 2} + \frac{4}{x - 3}\right) dx$$

$$= -3\ln|x - 2| + 4\ln|x - 3| + C$$

#### 含重因子的有理函数
**例**：$\int \frac{1}{x(x - 1)^2} dx$

解：设$\frac{1}{x(x - 1)^2} = \frac{A}{x} + \frac{B}{x - 1} + \frac{C}{(x - 1)^2}$

$$1 = A(x - 1)^2 + Bx(x - 1) + Cx$$

取$x = 0$：$1 = A$
取$x = 1$：$1 = C$
取$x = 2$：$1 = A + 2B + 2C = 1 + 2B + 2 = 3 + 2B$，故$B = -1$

$$\int \frac{1}{x(x - 1)^2} dx = \int \left(\frac{1}{x} - \frac{1}{x - 1} + \frac{1}{(x - 1)^2}\right) dx$$

$$= \ln|x| - \ln|x - 1| - \frac{1}{x - 1} + C$$

## 四、三角有理函数积分

### 1. 万能代换

设$t = \tan \frac{x}{2}$，则：
$$\sin x = \frac{2t}{1 + t^2}, \quad \cos x = \frac{1 - t^2}{1 + t^2}, \quad dx = \frac{2}{1 + t^2} dt$$

### 2. 典型例题

**例**：$\int \frac{1}{1 + \sin x + \cos x} dx$

解：令$t = \tan \frac{x}{2}$

$$\int \frac{1}{1 + \sin x + \cos x} dx = \int \frac{1}{1 + \frac{2t}{1 + t^2} + \frac{1 - t^2}{1 + t^2}} \cdot \frac{2}{1 + t^2} dt$$

$$= \int \frac{2}{(1 + t^2) + 2t + (1 - t^2)} dt = \int \frac{2}{2 + 2t} dt = \int \frac{1}{1 + t} dt$$

$$= \ln|1 + t| + C = \ln\left|1 + \tan\frac{x}{2}\right| + C$$

### 3. 特殊技巧

#### 利用三角恒等式
**例**：$\int \frac{\sin x}{1 + \sin x} dx$

解：$$\int \frac{\sin x}{1 + \sin x} dx = \int \frac{\sin x(1 - \sin x)}{(1 + \sin x)(1 - \sin x)} dx$$

$$= \int \frac{\sin x - \sin^2 x}{\cos^2 x} dx = \int \frac{\sin x}{\cos^2 x} dx - \int \tan^2 x dx$$

$$= \frac{1}{\cos x} - \int (\sec^2 x - 1) dx = \sec x - \tan x + x + C$$

## 五、简单无理函数积分

### 1. 含$\sqrt{ax^2 + bx + c}$的积分

#### 方法：配方+三角代换

**例**：$\int \frac{1}{\sqrt{x^2 + 4x + 5}} dx$

解：配方：$x^2 + 4x + 5 = (x + 2)^2 + 1$

令$u = x + 2$，则：

$$\int \frac{1}{\sqrt{x^2 + 4x + 5}} dx = \int \frac{1}{\sqrt{u^2 + 1}} du = \ln|u + \sqrt{u^2 + 1}| + C$$

$$= \ln|x + 2 + \sqrt{x^2 + 4x + 5}| + C$$

### 2. 欧拉代换

#### 第一种欧拉代换（当$a > 0$时）
设$\sqrt{ax^2 + bx + c} = t \pm \sqrt{a}x$

#### 第二种欧拉代换（当$c > 0$时）
设$\sqrt{ax^2 + bx + c} = xt \pm \sqrt{c}$

#### 第三种欧拉代换（当二次式有实根时）
设$\sqrt{ax^2 + bx + c} = t(x - \alpha)$，其中$\alpha$是二次式的一个实根

**例**：$\int \frac{1}{x\sqrt{x^2 - 1}} dx$（$x > 1$）

解：令$\sqrt{x^2 - 1} = t(x - 1)$，$x^2 - 1 = t^2(x - 1)^2$

$x + 1 = t^2(x - 1)$，解得$x = \frac{t^2 + 1}{t^2 - 1}$

$$dx = \frac{-4t}{(t^2 - 1)^2} dt$$

$$\sqrt{x^2 - 1} = t(x - 1) = t \cdot \frac{2}{t^2 - 1} = \frac{2t}{t^2 - 1}$$

$$\int \frac{1}{x\sqrt{x^2 - 1}} dx = \int \frac{t^2 - 1}{t^2 + 1} \cdot \frac{t^2 - 1}{2t} \cdot \frac{-4t}{(t^2 - 1)^2} dt = \int \frac{-2}{t^2 + 1} dt$$

$$= -2\arctan t + C$$

回代 $t = \frac{\sqrt{x^2 - 1}}{x - 1}$，令 $x = \sec\theta$（$\theta \in (0, \pi/2)$），则：

$t = \frac{\tan\theta}{\sec\theta - 1} = \frac{\sin\theta}{1 - \cos\theta} = \cot\frac{\theta}{2}$

$-2\arctan(\cot\frac{\theta}{2}) = -2(\frac{\pi}{2} - \frac{\theta}{2}) = -\pi + \theta$

最终结果为 $\theta + C = \text{arcsec}\, x + C$

> 注：此题用三角代换 $x = \sec\theta$ 更简洁，直接得 $\int d\theta = \theta + C = \text{arcsec}\, x + C$

## 六、积分技巧总结

### 1. 方法选择策略

#### 观察被积函数的形式
- **含$x^n f(x^{n+1})$**：用凑微分法
- **含$\sqrt{a^2 \pm x^2}$**：用三角代换
- **含对数函数或反三角函数**：用分部积分法
- **有理函数**：用部分分式分解
- **三角有理函数**：用万能代换或三角恒等式

#### 尝试多种方法
有些积分可以用多种方法求解，选择最简便的方法。

### 2. 常用技巧

#### 加项减项
**例**：$\int \frac{x^2}{x^2 + 1} dx$

$$\int \frac{x^2}{x^2 + 1} dx = \int \left(1 - \frac{1}{x^2 + 1}\right) dx = x - \arctan x + C$$

#### 乘以巧妙因子
**例**：$\int \frac{1}{\sin x} dx$

$$\int \frac{1}{\sin x} dx = \int \frac{\sin x}{\sin^2 x} dx = \int \frac{\sin x}{1 - \cos^2 x} dx$$

$$= \int \frac{-d(\cos x)}{1 - \cos^2 x} = -\frac{1}{2}\ln\left|\frac{1 + \cos x}{1 - \cos x}\right| + C$$

#### 反复使用同一种技巧
有些积分需要反复使用同一种技巧，如分部积分法的循环使用。

## 七、典型例题详解

### 1. 综合应用多种方法

**例**：$\int \frac{x + 1}{\sqrt{x^2 + 2x - 3}} dx$

解：配方：$x^2 + 2x - 3 = (x + 1)^2 - 4$

令$u = x + 1$，则：

$$\int \frac{x + 1}{\sqrt{x^2 + 2x - 3}} dx = \int \frac{u}{\sqrt{u^2 - 4}} du$$

$$= \sqrt{u^2 - 4} + C = \sqrt{x^2 + 2x - 3} + C$$

### 2. 复杂的分部积分

**例**：$\int x^3 e^{x^2} dx$

解：先设$t = x^2$

$$\int x^3 e^{x^2} dx = \frac{1}{2}\int t e^t dt = \frac{1}{2}(t e^t - \int e^t dt)$$

$$= \frac{1}{2}(t e^t - e^t) + C = \frac{1}{2}e^{x^2}(x^2 - 1) + C$$

### 3. 三角积分的技巧

**例**：$\int \sin^3 x \cos^2 x dx$

解：$\sin^3 x = \sin^2 x \cdot \sin x = (1 - \cos^2 x)\sin x$

$$\int \sin^3 x \cos^2 x dx = \int (1 - \cos^2 x)\cos^2 x \sin x dx$$

令$u = \cos x$，$du = -\sin x dx$

$$= -\int (1 - u^2)u^2 du = -\int (u^2 - u^4) du = -\left(\frac{u^3}{3} - \frac{u^5}{5}\right) + C$$

$$= \frac{\cos^5 x}{5} - \frac{\cos^3 x}{3} + C$$

## 八、常见易错点

### 1. 换元时忘记还原变量

❌ $\int \frac{x}{\sqrt{1 - x^2}} dx = \int \frac{du}{\sqrt{u}} = 2\sqrt{u} + C$
✅ $\int \frac{x}{\sqrt{1 - x^2}} dx = \int \frac{du}{\sqrt{u}} = 2\sqrt{u} + C = 2\sqrt{1 - x^2} + C$

### 2. 分部积分时选择错误

❌ 对$\int x e^x dx$选择$u = e^x$
✅ 应选择$u = x$，$dv = e^x dx$

### 3. 有理函数分解时漏项

❌ $\frac{1}{x(x - 1)^2} = \frac{A}{x} + \frac{B}{x - 1}$
✅ 应包含重因子项：$\frac{1}{x(x - 1)^2} = \frac{A}{x} + \frac{B}{x - 1} + \frac{C}{(x - 1)^2}$

### 4. 三角代换时忘记还原

❌ $\int \frac{1}{\sqrt{1 - x^2}} dx = \int \frac{a\cos t}{a\cos t} dt = t + C$
✅ 应还原为：$\int \frac{1}{\sqrt{1 - x^2}} dx = t + C = \arcsin\frac{x}{a} + C$

## 九、积分技巧记忆口诀

- **凑微分**：看结构，找关系，凑出du来替dx
- **三角代换**：$a^2 - x^2$用正弦，$a^2 + x^2$用正切，$x^2 - a^2$用正割
- **分部积分**：反对幂三指，选u有规律
- **有理函数**：先分后合，部分分式，逐项积分
- **三角有理**：万能代换，恒等变换，灵活运用
