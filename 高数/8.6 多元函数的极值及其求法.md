# 📐 多元函数的极值及其求法 听课笔记

> 📌 **课程来源**：KR 老师高数下预习课
> 🎯 **核心目标**：掌握无条件极值的判别法、条件极值的拉格朗日乘数法、方向导数与梯度
> 💡 **重要性**：极值问题是应用题的核心，拉格朗日乘数法是考试必考大题

---

## 一、无条件极值（二元函数极值）

### 1.1 定义
设 $z = f(x, y)$ 在 $(x_0, y_0)$ 的某邻域内有定义。若对该邻域内所有 $(x, y) \neq (x_0, y_0)$，都有：
- $f(x, y) < f(x_0, y_0)$ → $(x_0, y_0)$ 是**极大值点**
- $f(x, y) > f(x_0, y_0)$ → $(x_0, y_0)$ 是**极小值点**

### 1.2 极值的必要条件
若 $f$ 在 $(x_0, y_0)$ 处有极值，且偏导数存在，则：
$$f_x(x_0, y_0) = 0, \quad f_y(x_0, y_0) = 0$$

满足此条件的点称为**驻点**。

> ⚠️ **注意**：驻点 $\neq$ 极值点！如 $z = xy$ 在 $(0,0)$ 是驻点但不是极值点（**鞍点**）。

### 1.3 极值的充分条件 ⭐⭐⭐

设 $f$ 在 $(x_0, y_0)$ 的某邻域内有连续二阶偏导数，且 $(x_0, y_0)$ 是驻点（$f_x = f_y = 0$）。记：
$$A = f_{xx}(x_0, y_0), \quad B = f_{xy}(x_0, y_0), \quad C = f_{yy}(x_0, y_0)$$

判别式 $\Delta = AC - B^2$：

| $\Delta$ 的值 | $A$ 的符号 | 结论 |
| :--- | :--- | :--- |
| $\Delta > 0$ | $A < 0$ | **极大值** |
| $\Delta > 0$ | $A > 0$ | **极小值** |
| $\Delta < 0$ | 任意 | **不是极值**（鞍点） |
| $\Delta = 0$ | 任意 | **无法判断**，需另法 |

> 💡 **口诀**：**$\Delta > 0$ 看 $A$ 号，$A$ 负极大正极小。$\Delta < 0$ 是鞍点，$\Delta = 0$ 没法搞。**

> **例题 1**：求 $f(x, y) = x^3 + y^3 - 3xy$ 的极值。

- **Step 1：求驻点**
  - $f_x = 3x^2 - 3y = 0 \Rightarrow y = x^2$
  - $f_y = 3y^2 - 3x = 0 \Rightarrow x = y^2$
  - 代入：$x = (x^2)^2 = x^4 \Rightarrow x^4 - x = 0 \Rightarrow x(x^3 - 1) = 0$
  - 驻点：$(0, 0)$ 和 $(1, 1)$
- **Step 2：求二阶偏导**
  - $A = f_{xx} = 6x$，$B = f_{xy} = -3$，$C = f_{yy} = 6y$
- **Step 3：在 $(0, 0)$ 处**：$A = 0$，$B = -3$，$C = 0$
  - $\Delta = 0 \times 0 - (-3)^2 = -9 < 0$ → **鞍点**，不是极值
- **Step 4：在 $(1, 1)$ 处**：$A = 6$，$B = -3$，$C = 6$
  - $\Delta = 6 \times 6 - (-3)^2 = 36 - 9 = 27 > 0$，且 $A = 6 > 0$
  - → **极小值**，$f(1, 1) = 1 + 1 - 3 = -1$ ✅

## 二、条件极值与拉格朗日乘数法 ⭐⭐⭐

### 2.1 问题形式
求 $f(x, y)$ 在约束条件 $\varphi(x, y) = 0$ 下的极值。

### 2.2 拉格朗日乘数法步骤

1. **构造拉格朗日函数**：
   $$L(x, y, \lambda) = f(x, y) + \lambda \varphi(x, y)$$

2. **求偏导并令为 0**：
   $$\begin{cases} L_x = f_x + \lambda \varphi_x = 0 \\ L_y = f_y + \lambda \varphi_y = 0 \\ L_\lambda = \varphi(x, y) = 0 \end{cases}$$

3. **解方程组**，得到候选点 $(x_0, y_0)$

4. **判断**是极大还是极小（一般由实际问题背景确定）

> 💡 **口诀**：拉格朗日三步走——构造函数、求偏导、解方程！

### 2.3 多约束条件

两个约束条件 $\varphi(x, y, z) = 0$，$\psi(x, y, z) = 0$ 的情况：
$$L(x, y, z, \lambda, \mu) = f(x, y, z) + \lambda\varphi(x, y, z) + \mu\psi(x, y, z)$$
对 $x, y, z, \lambda, \mu$ 分别求偏导令为 0。

> **例题 2**：求 $f(x, y) = xy$ 在约束 $x + y = 1$ 下的极值。

- **Step 1**：设 $L(x, y, \lambda) = xy + \lambda(1 - x - y)$
- **Step 2**：
  - $L_x = y - \lambda = 0 \Rightarrow \lambda = y$
  - $L_y = x - \lambda = 0 \Rightarrow \lambda = x$
  - $L_\lambda = 1 - x - y = 0$
- **Step 3**：由前两式得 $x = y$，代入第三式 $2x = 1 \Rightarrow x = y = \frac{1}{2}$
- **Step 4**：$f\left(\frac{1}{2}, \frac{1}{2}\right) = \frac{1}{4}$（由实际问题知这是最大值）✅

## 三、方向导数与梯度 ⭐

### 3.1 方向导数
设 $f(x, y)$ 在 $P_0(x_0, y_0)$ 处可微，$\vec{l}$ 是一个单位向量 $(\cos\alpha, \cos\beta)$，则 $f$ 在 $P_0$ 处沿 $\vec{l}$ 方向的**方向导数**为：
$$\frac{\partial f}{\partial l}\bigg|_{P_0} = f_x(x_0, y_0)\cos\alpha + f_y(x_0, y_0)\cos\beta$$

> 💡 **理解**：方向导数 = 函数在某个指定方向上的变化率。偏导数是沿坐标轴的特殊方向导数。

### 3.2 梯度
$$\text{grad}\, f = \nabla f = (f_x, f_y) = f_x\vec{i} + f_y\vec{j}$$

### 3.3 梯度与方向导数的关系 ⭐

$$\frac{\partial f}{\partial l} = \nabla f \cdot \vec{l} = |\nabla f|\cos\theta$$

其中 $\theta$ 是梯度与方向 $\vec{l}$ 的夹角。

| $\theta$ 的值 | 方向导数 | 含义 |
| :--- | :--- | :--- |
| $\theta = 0$（沿梯度方向） | 最大值 $= \|\nabla f\|$ | 函数增长最快 |
| $\theta = \pi$（逆梯度方向） | 最小值 $= -\|\nabla f\|$ | 函数下降最快 |
| $\theta = \frac{\pi}{2}$ | $= 0$ | 函数不变化 |

> 💡 **口诀**：**梯度指路增长快，逆着它走下降快，垂直方向不变换！**

### 3.4 三维推广
$$\nabla f = (f_x, f_y, f_z)$$
$$\frac{\partial f}{\partial l} = f_x\cos\alpha + f_y\cos\beta + f_z\cos\gamma$$

> **例题 3**：求 $f(x, y) = x^2 + y^2$ 在点 $(1, 1)$ 处沿方向 $\vec{l} = \left(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}\right)$ 的方向导数，以及梯度。

- **Step 1**：求偏导：$f_x = 2x$，$f_y = 2y$
- **Step 2**：在 $(1,1)$ 处：$\nabla f = (2, 2)$
- **Step 3**：方向导数 = $\nabla f \cdot \vec{l} = 2 \cdot \frac{1}{\sqrt{2}} + 2 \cdot \frac{1}{\sqrt{2}} = \frac{4}{\sqrt{2}} = 2\sqrt{2}$
- **Step 4**：梯度方向是增长最快方向，$|\nabla f| = \sqrt{4+4} = 2\sqrt{2}$，方向为 $\left(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}\right)$

## 📝 总结与记忆要点

| 概念 | 核心公式/结论 |
| :--- | :--- |
| **驻点** | $f_x = 0$ 且 $f_y = 0$ |
| **极值判别法** | $\Delta = AC - B^2$；$\Delta > 0$ 看 $A$ |
| **拉格朗日乘数法** | $L = f + \lambda\varphi$，解方程组 |
| **方向导数** | $\frac{\partial f}{\partial l} = \nabla f \cdot \vec{l}$ |
| **梯度** | $\nabla f = (f_x, f_y)$，增长最快方向 |
| **梯度模** | 最大方向导数值 |

- **极值口诀**：$\Delta > 0$ 看 $A$ 号，$A$ 负极大正极小。$\Delta < 0$ 是鞍点，$\Delta = 0$ 没法搞。
- **梯度口诀**：梯度指路增长快，逆着它走下降快，垂直方向不变换！
- **易错点**：拉格朗日乘数法解出候选点后，极值的判定一般由**实际背景**决定，不用二阶判别法。
